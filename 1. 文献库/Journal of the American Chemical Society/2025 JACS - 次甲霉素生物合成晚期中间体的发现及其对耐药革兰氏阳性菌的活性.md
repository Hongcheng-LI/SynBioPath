**二：核心速览**

**研究背景**

次甲霉素类 (Methylenomycins) 是一类由天蓝色链霉菌 A3(2) (*Streptomyces coelicolor* A3(2)) 产生的、具有环戊酮结构的抗生素。尽管其生物合成基因簇 (*mmy*) 早已被发现，并且基于生物信息学分析和同位素标记实验提出了一个假说性的生物合成路径，但该途径中各个酶，特别是后期修饰酶的具体生化功能，一直缺乏直接的实验证据支持。因此，对这些酶的功能进行系统性的实验确证，对于完整理解次甲霉素的生物合成逻辑至关重要。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20次甲霉素生物合成晚期中间体的发现及其对耐药革兰氏阳性菌的活性.002.png)

**前期研究**

前期的研究主要通过基因序列比对，推测了 *mmy* 基因簇中各个基因的潜在功能。例如，推测 MmyD 催化早期缩合反应，MmyE 负责引入外切亚甲基，而 MmyO 和 MmyF 则协同催化最后的环氧化步骤，将次甲霉素 C (methylenomycin C, **2**) 转化为次甲霉素 A (methylenomycin A, **1**)。然而，这些推测均未经过遗传学或生物化学方法的直接验证。

**本文突破点**

本文的核心突破在于通过系统的基因敲除和代谢物组学分析，首次实验性地阐明了次甲霉素生物合成途径中四个关键后期酶 (MmyD, MmyE, MmyO, MmyF) 的确切功能。更重要的突破是，在 *mmyE* 基因敲除突变株中发现了两个全新的、此前未报道的生物合成中间体：前次甲霉素 C (premethylenomycin C, **6**) 及其内酯形式 **5**。出人意料的是，生物活性测试显示，这两个新发现的中间体对多种革兰氏阳性菌，特别是耐甲氧西林金黄色葡萄球菌 (MRSA) 和屎肠球菌 (*Enterococcus faecium*) 等耐药菌株的抑制活性，比最终产物次甲霉素 A 和 C 高出一到两个数量级。

**研究难点**

本研究的主要难点在于对基因敲除突变株中积累的微量、不稳定的新代谢物进行分离和结构鉴定。特别是前次甲霉素 C (**6**) 作为一个γ-羟基酸，易于自发环化形成其内酯形式 **5**，这为分离纯化和结构确证带来了挑战。此外，在异源宿主中功能性地表达黄素依赖性单加氧酶 MmyO/MmyF 并验证其催化活性也具有一定的技术难度。

**三：研究方法**

- **基因工程与菌株构建**: 利用包含完整 *mmy* 基因簇的黏粒 (cosmid) C73-787，通过 PCR-targeting 的方法在其中构建目标基因 (*mmyD*, *mmyE*, *mmyO*, *mmyF*) 的框内缺失。同时敲除阻遏蛋白基因 *mmyR* 以提高产量。随后将改造后的黏粒通过接合转移整合到不含该基因簇的宿主天蓝色链霉菌 M145 的染色体中。
- **代谢物分析**: 采用液相色谱-质谱联用 (LC-MS) 技术对不同突变株的发酵提取物进行代谢轮廓分析，通过比较野生型和突变株的代谢产物差异，鉴定基因功能。
- **化合物分离与结构鉴定**: 通过制备型高效液相色谱 (preparative HPLC) 从 *mmyE* 突变株的大规模发酵液中分离新化合物。利用核磁共振波谱 (NMR)，包括 ¹H, ¹³C, COSY, HSQC, HMBC 和 ROESY，以及高分辨质谱 (HRMS) 和圆二色谱 (CD)，确定新化合物的平面结构和绝对构型。
- **异源表达与生化验证**: 在链霉菌 M145 中构建表达载体，共表达 *mmyO* 和 *mmyF* 基因以及抗性基因 *mmr*。通过向该工程菌株饲喂底物次甲霉素 C (**2**)，并在 ¹⁸O₂ 环境下进行培养，验证 MmyO/MmyF 的环氧化功能及氧原子的来源。
- **抗菌活性测试**: 采用微量肉汤稀释法测定新发现的中间体、最终产物及副产物对一系列革兰氏阳性菌、革兰氏阴性菌和酵母菌的最低抑菌浓度 (Minimum Inhibitory Concentration, MIC) 和最低杀菌浓度 (Minimum Bactericidal Concentration, MBC)。

- **W89 (Δ*mmyR*)**: 作为阳性对照，该菌株能够正常产生次甲霉素 C (**2**) 和次甲霉素 A (**1**)。

**研究部分二：MmyE 功能的解析与新生物活性中间体的发现**

在验证了其他酶的功能后，本部分集中研究 MmyE 的作用。通过构建 *mmyE* 敲除突变株，旨在阐明其功能，并期望能够捕获其催化的反应底物。

- **(图 3, W86)** *mmyE* 敲除突变株 (W86) 不再产生 **1** 和 **2**，而是积累了两个新的、分子量与 **2** 相关的化合物，分别命名为 **5** 和 **6**。这强烈暗示 MmyE 负责催化从 **6** 到 **2** 的转化，而 **5** 和 **6** 是该步骤的上游中间体。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20次甲霉素生物合成晚期中间体的发现及其对耐药革兰氏阳性菌的活性.004.png)

- **(图 4)** 通过对化合物 **5** 和 **6** 的大规模制备和详细的 NMR 波谱分析，确定了它们的结构。化合物 **6** 被鉴定为前次甲霉素 C (premethylenomycin C)，其 C5 位是一个羟甲基，而非外切亚甲基。化合物 **5** 是 **6** 的γ-羟基酸结构发生分子内酯化形成的丁内酯，被命名为前次甲霉素 C 内酯 (premethylenomycin C lactone)。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20次甲霉素生物合成晚期中间体的发现及其对耐药革兰氏阳性菌的活性.006.png)

- **(Scheme 1)** 基于 **6** 到 **2** 的转化是脱去一分子水，且 MmyE 具有黄素依赖性氧化还原酶的同源性但该反应无净氧化还原变化，作者提出了一个由 MmyE 催化的 E1cB 消除反应机理，即通过酶活性位点的酸碱残基协同作用，促进 C5 位羟基的消除，从而形成外切亚甲基。

**研究部分三：MmyO/MmyF 环氧化功能的生化确证**

为了直接证明 MmyO 和 MmyF 能够催化从 **2** 到 **1** 的环氧化反应，作者设计了一个异源表达和底物饲喂实验。

- **(图 6)** 作者在对次甲霉素不敏感的宿主 (*S. coelicolor* M145 表达 *mmr* 抗性基因) 中共表达了 *mmyO* 和 *mmyF* (菌株 W301)。当向该菌株饲喂底物 **2** 时，LC-MS 检测到了最终产物 **1** 的生成。而对照菌株 W302 (仅表达 *mmr*) 则不能进行该转化。这直接在生化层面证实了 MmyO 和 MmyF 共同负责催化这一环氧化步骤。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20次甲霉素生物合成晚期中间体的发现及其对耐药革兰氏阳性菌的活性.008.png)

- **(图 S31)** 在 ¹⁸O₂ 环境下进行饲喂实验，发现产物 **1** 中整合了一个 ¹⁸O 原子，而底物 **2** 中没有。这证实了环氧化反应中的氧原子来源于分子氧 O₂，与 MmyO 作为单加氧酶的功能相符。
- **(Scheme 2)** 基于上述结果，作者提出了一个合理的催化机理：MmyF 作为黄素还原酶，利用 NADPH 还原 FAD 为 FADH₂；MmyO 作为单加氧酶，利用 FADH₂ 和 O₂ 生成 C4a-过氧黄素活性中间体，该中间体对 **2** 的双键进行亲核加成，最终形成环氧化产物 **1**。

**研究部分四：生物合成中间体与产物的抗菌活性评价**

本研究的一个意外发现是在 Δ*mmyE* 突变株中积累了新化合物 **5** 和 **6**。本部分旨在系统地比较这些新发现的中间体、最终产物 (**1**, **2**) 以及旁路产物 (**7**, **8**) 的抗菌活性，以评估它们的药用潜力。

**(Table 1)** 这是本研究最具影响力的发现之一。抗菌活性测试结果显示：

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20次甲霉素生物合成晚期中间体的发现及其对耐药革兰氏阳性菌的活性.010.png)

- 最终产物 **1** 和 **2** 对革兰氏阳性菌的活性中等 (MICs 主要在 64-256 µg/mL)。
- 旁路产物 **7** 和 **8** 完全没有活性，表明外切亚甲基是 **1** 和 **2** 活性的关键药效团。
- **惊人的发现**：新中间体前次甲霉素 C (**6**) 的活性比最终产物高出约一个数量级 (MICs 在 8-32 µg/mL)。
- **更显著的发现**：其内酯形式，前次甲霉素 C 内酯 (**5**)，活性最高，对多种革兰氏阳性菌的 MIC 值低至 0.5-2 µg/mL，比最终产物 **1** 和 **2** 的活性**高出两个数量级（约 100-500 倍）**。

综合以上所有遗传学和生物化学证据，本部分旨在提出一个经过实验验证的、更新版的次甲霉素生物合成途径。

**(Scheme 4)** 该图总结了本研究的所有发现，描绘了次甲霉素生物合成后期的完整路径：

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20次甲霉素生物合成晚期中间体的发现及其对耐药革兰氏阳性菌的活性.005.png)

1. MmyD 催化早期前体缩合，经过 MmyG/K/Q/Y/X 等一系列酶的催化，形成环戊酮核心和内酯结构 **5**。
2. 硫酯酶 MmyT 水解内酯环，生成开环的γ-羟基酸 **6**。
3. MmyE 催化 **6** 脱水，形成外切亚甲基，生成次甲霉素 C (**2**)。
4. MmyO 和 MmyF 协同催化 **2** 的环氧化，生成最终产物次甲霉素 A (**1**)。
5. 同时，途径中也存在旁路反应，如 **2** 可被非特异性还原为无活性的 **7** 和 **8**。

**五：总体结论**

本研究通过系统的基因敲除、代谢物分析和生化验证，首次实验性地阐明了次甲霉素生物合成途径中后期关键酶 MmyD, MmyE, MmyO 和 MmyF 的确切功能。研究证实了 MmyD 在早期缩合、MmyE 在外切亚甲基形成以及 MmyO/F 在最终环氧化步骤中的作用，与之前的生物信息学预测基本一致。更为重要的是，本研究发现并鉴定出两个全新的生物合成中间体，前次甲霉素 C (**6**) 及其内酯 **5**。出人意料的生物活性测试结果表明，这两个中间体，特别是前次甲霉素 C 内酯 (**5**)，对包括 MRSA 在内的多种耐药革兰氏阳性菌表现出比最终产物高出两个数量级的强效抗菌活性。这一发现不仅完整地描绘了次甲霉素的生物合成蓝图，更揭示了生物合成途径中的中间体可能具有比终产物更优越的生物活性，为通过“途径工程”和“中间体挖掘”策略发现新型抗生素先导化合物提供了一个极具启发性的范例。

**六：论文评价**

**优点与创新**

1. **研究的系统性与完整性**：该研究采用经典的“基因敲除-表型分析-化学鉴定-生化验证”的逻辑，对一个完整的生物合成途径进行了系统性的功能解析，证据链完整，逻辑严密。
2. **“中间体活性更高”的意外发现**：本研究最核心的创新和价值在于发现了中间体 **5** 和 **6** 的超强活性。这一“反常”现象挑战了“最终产物活性最高”的传统认知，为天然产物药物发现提供了一个全新的思路：即生物合成途径本身就是一个多样化的化合物库，其中间体可能蕴含着比终产物更优的药理活性。
3. **对耐药菌的潜在治疗价值**：化合物 **5** 对 MRSA 和耐药肠球菌的低 MIC 值 (1-2 µg/mL) 及其不易产生耐药性的特点，使其成为一个极具潜力的抗生素先导化合物，具有重要的临床应用前景。
4. **对酶功能的深入解析**：研究不仅确定了酶的功能，还对其催化机理进行了初步探讨（如 MmyE 的 E1cB 机理，MmyO/F 的环氧化机理），为酶学研究提供了新的案例。

**未来研究方向**

1. **前次甲霉素 C 内酯 (5) 的作用机制研究**：阐明化合物 **5** 强效抗菌活性的分子靶点和作用机制。由于其结构与最终产物不同（缺少外切亚甲基），其作用靶点可能也与次甲霉素 A/C 不同。
2. **优化 5 的生物合成**：既然中间体 **5** 的活性更高，未来的研究重点可以转向通过代谢工程手段，在 Δ*mmyE* 突变株的基础上进一步优化培养基和发酵条件，或改造上游基因，以实现 **5** 的高效、专一性生物合成。
3. **化学合成与结构优化**：对先导化合物 **5** 进行化学全合成或半合成，一方面可以验证其结构，另一方面可以进行结构修饰和构效关系研究，以开发活性更高、成药性更好的衍生物。
4. **早期途径酶的功能解析**：本研究聚焦于后期酶，而途径中 MmyG, MmyK, MmyQ, MmyX, MmyY 等早期酶的功能仍是推测。对这些酶进行功能研究，将能最终完整地揭示整个次甲霉素的合成全貌。

**七：关键问题及回答**

**问题一：研究者是如何确定 MmyE 的功能是催化外切亚甲基的形成？**

**回答：** 该结论是基于对 *mmyE* 基因敲除突变株 (W86) 的表型分析得出的：

1. **产物变化**：野生型菌株产生含有外切亚甲基的次甲霉素 C (**2**)。而 Δ*mmyE* 突变株不再产生 **2**。
2. **中间体积累**：Δ*mmyE* 突变株积累了两个新化合物 **5** 和 **6**。
3. **结构对比**：通过 NMR 鉴定，确定了 **6** (前次甲霉素 C) 的结构。将其与产物 **2** 对比，唯一的结构差异在于 C5 位：**6** 是一个 -CH₂OH（羟甲基），而 **2** 是一个 =CH₂（外切亚甲基）。从 **6** 到 **2** 的转化在化学上是一个脱水反应。
4. **逻辑推断**：由于敲除 *mmyE* 导致了底物 **6** 的积累和产物 **2** 的消失，因此可以得出结论，MmyE 的功能就是催化从 **6** 到 **2** 的转化，即催化羟甲基脱水形成外切亚甲基。

**问题二：本研究最重要的科学发现是什么？为什么它具有重要意义？**

**回答：** 本研究最重要的科学发现是**生物合成中间体前次甲霉素 C 内酯 (5) 的抗菌活性比最终产物次甲霉素 A/C 高出两个数量级**。
这一发现具有极其重要的意义，因为它为**抗生素发现提供了一种全新的策略**：

1. **颠覆传统认知**：传统上，天然产物研究主要关注生物合成途径的最终产物，认为它们是经过进化优化后活性最佳的分子。本研究表明，途径中的中间体可能因为结构或物理化学性质的不同，而具有更强的生物活性。
2. **开辟“中间体挖掘”新策略**：该发现提示我们，可以通过在已知的生物合成途径中，通过基因工程手段（如基因敲除）阻断某些步骤，来“挖掘”并积累各种生物合成中间体，并对这些中间体进行系统的活性筛选。这相当于将一条生物合成途径从“单一产物生产线”变成了一个“多样性化合物库”。
3. **应对抗生素耐药性**：化合物 **5** 对 MRSA 等临床重要的耐药菌表现出强效活性，且不易诱导耐药，使其成为一个非常有价值的新型抗生素先导化合物。这证明了“中间体挖掘”策略在寻找能克服现有耐药机制的新型药物方面具有巨大潜力。

**问题三：研究者是如何证明 MmyO 和 MmyF 共同催化了从次甲霉素 C (2) 到 A (1) 的环氧化反应？**

**回答：** 该结论由一组环环相扣的实验共同证明：

1. **遗传学证据**：单独敲除 *mmyO* (W100) 或 *mmyF* (W108) 均导致产物 **1** 消失，但保留了底物 **2** 的产生。这表明 **1** 的合成需要 MmyO 和 MmyF 两者同时存在，且它们的作用位于 **2** 之后。
2. **生化验证**：研究者构建了一个异源表达菌株 W301，该菌株同时表达 *mmyO* 和 *mmyF*。当向该菌株的培养物中直接饲喂纯化的底物 **2** 时，能够检测到产物 **1** 的生成。而没有表达这两个基因的对照菌株 W302 则不能进行此转化。这直接在生化层面证明了 MmyO 和 MmyF 能够催化从 **2** 到 **1** 的转化。
3. **机理证据**：通过在 ¹⁸O₂ 环境下进行饲喂实验，发现产物 **1** 的环氧基团中的氧原子来源于分子氧 O₂。这与 MmyO 作为单加氧酶、MmyF 作为其伴侣黄素还原酶的催化机制完全吻合。

综合这三方面的证据，研究者确证了 MmyO 和 MmyF 协同催化了次甲霉素生物合成的最后一步环氧化反应。
