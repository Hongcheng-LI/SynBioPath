**二：核心速览**

**研究背景**

本研究的核心科学问题是阐明真菌天然产物 Parnafungin 家族复杂的生物合成途径。

Parnafungin 具有一个独特的、由六个环稠合而成的骨架，其核心是一个罕见的 5-异噁唑烷酮 (5-isoxazolidinone, 5ILD) 杂环。这个 5ILD 结构是其抗真菌活性的关键药效团，其作用机制是抑制真核生物中对 mRNA 加工至关重要的聚腺苷酸聚合酶 (poly(A)-polymerase, PAP)。由于 5ILD 杂环在天然产物中极为罕见，其生物合成机制尚属未知。因此，解析 Parnafungin 的完整生物合成路径，特别是揭示这个关键 5ILD 杂环的形成机制，对于理解自然界如何构建新颖的含氮杂环、发现新的生物催化工具以及为药物开发提供新的合成策略具有重要的理论价值和潜在应用价值。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.002.png)

**前期研究**

在本文发表前，该领域的研究现状如下：

1. **Parnafungin 的发现与活性**：Merck 公司的研究人员已经分离并鉴定了 Parnafungin A-D (**1-4**)，并阐明了它们通过抑制 PAP 发挥抗真菌活性的作用。同时，他们也发现 Parnafungin 在生理条件下不稳定，会发生差向异构化、B 环异构化和 5ILD 环开环等非酶促反应，这表明 5ILD 环的形成可能是生物合成的后期步骤。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.003.png)

2. **5ILD 结构**：5ILD 结构在天然产物中非常稀有。已知的相关生物合成研究仅限于 3-异噁唑烷酮 (3ILD) 环，如抗生素 D-环丝氨酸的生物合成。其路径涉及 L-精氨酸的 N-羟基化和后续由 ATP-grasp 酶催化的环化。这为 5ILD 的生物合成提供了一个可能的参考模型，即 γ-N-羟基羧酸的环化。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.004.png)

3. **前体化合物**：通过逆合成分析，研究者推测 Parnafungin 的 A/B/C 环部分来源于一个已知的 xanthone 类天然产物 blennolide C (**9**)。Blennolide C 的生物合成途径及其在二聚体 neosartorin 合成中的作用已有部分研究基础。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.005.png)

**本文突破点**

1. **完整途径解析**：首次完整解析了一个包含 26 个基因的复杂天然产物 Parnafungin 的生物合成基因簇 (*par* BGC)，并阐明了从初级代谢物到最终产物 Parnafungin A-D 的全部酶促步骤。
2. **新颖的生物芳基偶联机制**：发现了一个前所未有的、由三个酶 (CoA 连接酶 ParJ、P450 酶 ParO 和 DUF829 家族蛋白 ParD) 协同催化的“隐蔽式”硫酯化-氧化偶联-水解级联反应，实现了两个结构上无关联的分子 blennolide C 和邻氨基苯甲酸的高效偶联。
3. **5ILD 杂环的形成机制**：揭示了 5-异噁唑烷酮 (5ILD) 环的全新生物合成逻辑。该过程由一个黄素依赖性单加氧酶 (FMO) ParB 催化的 N-羟基化反应和一个二烯内酯水解酶 (DLH) 家族的同源蛋白 ParF 催化的非 ATP 依赖性直接内酰胺化反应串联完成。
4. **非典型酶的功能表征**：鉴定并表征了一个不含经典催化三联体的非典型短链脱氢酶/还原酶 (SDR) ParT，证明其能够高效还原一个高度共轭的 phenanthridine 环系中的亚胺键。

**研究难点**

1. **大型基因簇的异源表达**：*par* BGC 包含超过 20 个酶，完整地在异源宿主中进行重构非常困难。研究人员采用了分步重构和体外酶学分析相结合的策略来解决这个问题。
2. **关键中间体的不稳定性**：许多 Parnafungin 的生物合成中间体和终产物在生理条件下不稳定，容易发生非酶促的异构化和开环反应，这给分离鉴定和实验验证带来了巨大挑战。
3. **未知功能的酶**：基因簇中包含多个功能未知的酶，特别是 DUF829 家族的 ParD 和非典型的 SDR ParT。确定它们在复杂途径中的确切功能是本研究的一大难点。
4. **低效的酶促反应**：在初步探索芳基偶联反应时，仅使用 P450 酶 ParO 的催化效率极低 (转化率 < 5%)，这提示了存在其他辅助酶或正确的底物形式，是研究初期的一个关键瓶颈。

**三：研究方法**

**基因组挖掘**：通过对 Parnafungin 的两个已知产生菌 *Cladobotryum varium* 和 *Microcera larvarum* 进行全基因组测序，并基于生物合成逻辑（需要包含合成前体 blennolide C 的同源基因）进行比对分析，从而定位到一个包含 26 个保守基因的 *par* 生物合成基因簇 (BGC)。

**菌株遗传操作**：主要采用在酿酒酵母 *Saccharomyces cerevisiae* RC01 中进行异源重构 (heterologous reconstitution) 的策略。通过构建包含不同 *par* 基因组合的表达质粒，转化到酵母中，并通过饲喂前体或中间体的方式，分步解析整个生物合成路径。

**酶功能及催化机制**：

- **体外生化分析**：将关键酶（如 ParJ, ParT, ParB, ParF）在大肠杆菌中进行异源表达和纯化。通过在体外体系中加入底物、辅酶（如 ATP, CoA, NADPH）和纯化的酶，利用 HPLC-MS 监测产物生成，从而精确验证酶的功能。例如，验证了 ParJ 的邻氨基苯甲酰-CoA 合成酶活性，以及 ParT 对亚胺底物 **5** 的还原活性。
- **定点突变**：为了探究酶的催化机制，对推测的关键催化残基进行突变。例如，将 ParD 的催化三联体丝氨酸残基 S110 突变为丙氨酸 (S110A)，以及将 ParF 的催化三联体半胱氨酸残基 C138 突变为丙氨酸 (C138A)，观察到酶活性的完全丧失，从而证实了这些残基的关键作用。
- **结构生物学预测**：使用 Alphafold3 和 Dali server 对功能未知的 ParD 蛋白进行结构预测和同源性搜索，发现其具有 α/β 水解酶折叠和类似丝氨酸蛋白酶的催化三联体，为其功能注释提供了重要线索。

**化合物分离与结构鉴定**：通过高效液相色谱 (HPLC) 分离酵母发酵产物和体外酶反应产物，利用高分辨率质谱 (HRMS) 确定分子量和元素组成，并通过核磁共振谱 (NMR) 对关键产物（如 **12**, **15**, **17**）进行全谱解析，从而明确其化学结构。

**四：实验设计及结果分析**

**研究部分一：*par* 基因簇的鉴定与前体 Blennolide C 的异源重构**

**实验目的与设计**

本部分的核心目的是：1. 通过生物信息学方法，在 Parnafungin 产生菌中鉴定出负责其生物合成的基因簇 (*par* BGC)。2. 实验验证该基因簇中部分基因的功能，即在异源宿主酵母中重构出推测的生物合成前体 blennolide C (**9**)。

设计步骤如下：

1. 对两种 Parnafungin 产生菌进行基因组测序。
2. 基于逆合成分析，推测 Parnafungin 的合成需要 blennolide C (**9**)，因此 BGC 中应包含与已知 neosartorin 基因簇 (*nsr*) 中合成 **9** 的同源基因。
3. 以此为线索，在两个基因组中筛选并鉴定出一个包含 26 个基因的保守 BGC，即 *par* 簇。
4. 将 *par* 簇中负责合成 **9** 的 6 个基因 (*ParR, ParS, ParH, ParE, ParG, ParI*) 克隆至酵母表达载体，转化至 *S. cerevisiae* RC01，并饲喂上游前体 chrysophanol (**10**)，通过 HPLC-MS 检测 **9** 的生成。

**实验结果与分析**

**深入解读**：作者成功地通过基因组挖掘策略，在两个产生菌中鉴定到了一个高度保守的 *par* 基因簇。该基因簇不仅包含了合成 blennolide C (**9**) 所需的全套酶的同源基因，还包含了一个 poly(A)-polymerase (PAP) 的同源基因 ParW。由于 PAP 是 Parnafungin 的作用靶点，ParW 的存在极有可能是作为自身的抗性基因 (self-resistance enzyme)，这为 *par* 簇与 Parnafungin 生物合成的关联性提供了强有力的遗传学证据。

**Figure 4**：该图展示了 Parnafungin (*par*) 和 Neosartorin (*nsr*) 两个基因簇的共线性对比。可以看出，两个产生菌 *C. varium* 和 *M. larvarum* 中的 *par* 簇基因排布和序列具有极高的同源性 (70-95% identity)。与 *nsr* 簇相比，*par* 簇中负责合成 blennolide C 的基因 (图中蓝色) 和负责芳基偶联的基因 (图中红色) 均有同源物，但 *par* 簇额外多出了一系列用于后期修饰（还原胺化、5ILD 环形成等）的基因（图中绿色等），清晰地展示了两个 BGC 在进化上的关联与功能上的分化。

**研究部分二：一个由三酶级联催化的新颖芳基偶联反应**

**实验目的与设计**

本部分的核心目的是阐明连接 blennolide C (9) 和邻氨基苯甲酸 (anthranilate) 的芳基-芳基偶联反应的酶学机制。

设计步骤如下：

1. 初步假设：基于 *nsr* 簇中 NsrP 的功能，推测 *par* 簇中的 P450 同源酶 ParO 负责催化 **9** 和邻氨基苯甲酸的偶联。在表达 ParEGI 的酵母中进一步表达 ParO，并饲喂 **11** 和邻氨基苯甲酸，检测偶联产物 **12** 的生成。
2. 优化探索：由于仅表达 ParO 反应效率极低，推测需要其他辅助酶。通过组合表达 *par* 簇中功能未知的酶，筛选能显著提高偶联效率的酶组合。
3. 机制验证：对筛选出的关键酶 ParJ (CoA 连接酶) 和 ParD (DUF829) 进行功能验证。通过体外酶学实验验证 ParJ 是否能活化邻氨基苯甲酸。通过饲喂化学合成的底物类似物 (AA-SNAC) 和对 ParD 进行定点突变，来验证 ParJ-ParO-ParD 三酶级联催化的“硫酯化-氧化偶联-水解”机制。

**实验结果与分析**

研究发现，单独的 P450 酶 ParO 催化 **9** 和邻氨基苯甲酸的偶联反应效率极低 (转化率 < 5%)。然而，当与 CoA 连接酶 ParJ 和功能未知的 DUF829 蛋白 ParD 共表达时，反应效率显著提高，底物 **9** 几乎被完全消耗。这一结果揭示了一个前所未有的三酶级联催化机制。作者提出，ParJ 首先利用 ATP 将邻氨基苯甲酸转化为高活性的邻氨基苯甲酰-CoA (AA-CoA)。随后，ParO 以 AA-CoA 为底物，与 **9** 进行氧化偶联，形成一个硫酯连接的中间体 **13**。最后，ParD 作为水解酶，切断硫酯键，释放出最终的偶联产物 **12** 并再生 CoA。这个“隐蔽式”的硫酯化策略，可能是为了提高底物识别特异性或将邻氨基苯甲酸从初级代谢中有效“隔离”出来用于次级代谢。

- **Figure 6A**：清晰地展示了所提出的三酶级联反应机理，从邻氨基苯甲酸出发，经 ParJ 活化为 AA-CoA，再由 ParO 偶联到 **9** 上形成 **13**，最后由 ParD 水解得到产物 **12**。

**研究部分三：通过还原胺化完成 D 环的构建**

**实验目的与设计**

本部分的核心目的是阐明在偶联产物 **12** 的基础上，如何通过关环反应构建 Parnafungin 的 D 环，形成 phenanthridine 核心骨架。

设计步骤如下：

1. 提出假设：D 环的形成需要一个 C-N 键的构建，这可能通过一个氧化-环化-还原的序列完成。即，**12** 的 C4 位苄基醇被氧化成醛，随后与芳香胺发生分子内环化形成亚胺 **5** (即 parnafungin A')，最后亚胺被还原成胺 **15**。
2. 酶的筛选：在 *par* 簇中寻找可能的氧化酶和还原酶。ParP (α-KG 依赖性加氧酶) 和 ParU (P450) 是氧化酶候选者；ParC 和 ParT (SDR) 是还原酶候选者。
3. 体内重构：在能够产生 **12** 的酵母体系 (ParEGIOJD) 中，分别表达 ParP 或 ParU，检测亚胺中间体 **5** 的生成。
4. 体外验证：由于 ParT 是一个不含经典催化三联体的“非典型”SDR，对其功能进行体外验证。将纯化的 ParT 与底物 **5** 和辅酶 NADPH 共同孵育，检测还原产物 **15** 的生成。

**实验结果与分析**

实验证明，D 环的构建确实分为两步。首先，α-酮戊二酸依赖性加氧酶 ParP 催化了 **12** 的 C4 位氧化和分子内脱水环化，一步形成了高度共轭的亚胺中间体 **5** (parnafungin A')。这一步是构建 phenanthridine 环系的关键。随后，一个非典型的短链还原酶 ParT 催化了对亚胺 **5** 的还原。ParT 的特殊之处在于它缺少保守的催化三联体 (S-Y-K)，但依然能高效地还原化学性质非常稳定的共轭亚胺体系，这在 SDR 家族中是罕见的功能，拓展了我们对这类酶催化能力的认知。

- **Figure 7A**：展示了 D 环构建的 proposed 两步反应：由 ParP 催化 **12** 生成亚胺 **5**，再由 ParT 催化 **5** 还原为胺 **15**。

- **Figure 7B**：酵母体内重构实验的 HPLC 结果。在 ParEGIOJD 体系中加入 ParP 后 (ParEGIOJDP)，检测到了与 parnafungin A' (**5**) 和 B' (**6**) 标准品保留时间、UV 和质谱信号 (m/z 452 [M+H]⁺) 完全一致的新产物峰。这证实了 ParP 负责催化亚胺的形成。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.013.png)

- **Figure 7C**：ParT 的体外酶活测定。将底物 **5** (m/z 452) 与纯化的 ParT 和 NADPH 孵育后，HPLC-MS 检测到 **5** 的峰高显著下降，同时出现了两个新的产物峰 **15** 和 **15-B**，其质谱信号为 m/z 454 [M+H]⁺，正好是 **5** 还原（增加 2 个氢原子）后的分子量。在没有 ParT 的对照组中，则没有还原产物的生成。这个直接的生化证据确凿地证明了 ParT 是负责催化亚胺还原的还原酶。

**研究部分四：由 ParB 和 ParF 串联催化 5ILD 环的形成**

**实验目的与设计**

本部分是研究的核心，旨在阐明关键药效团 5-异噁唑烷酮 (5ILD) 环的生物合成机制。

设计步骤如下：

1. 提出假设：基于 D-环丝氨酸的生物合成逻辑，推测 5ILD 环的形成源于胺 **15** 的 N-羟基化，生成不稳定的羟胺中间体 **16**，随后 **16** 的羧基被活化并发生分子内环化（内酰胺化）。
2. 酶的筛选：在 *par* 簇中寻找 N-加氧酶和环化酶。ParB (FMO) 和 ParU (P450) 是 N-加氧酶的候选者。由于没有发现典型的 ATP 依赖性连接酶或合成酶，作者将目光转向了一个不寻常的候选者——ParF，一个二烯内酯水解酶 (DLH) 的同源物。
3. 体内重构：在能够产生 **15** 的酵母体系 (ParEGIOJDP T) 中，组合表达 ParB/ParU 和 ParF，检测最终产物 parnafungin A (**1**) 和 B (**2**) 的生成。并通过基因敲除和定点突变实验验证每个酶的必要性。
4. 体外重构：为了进一步确认反应顺序和机制，使用纯化的 ParT, ParB, ParF 进行多酶体外级联反应。从底物 **5** 开始，逐步加入酶，观察中间产物和最终产物的生成情况。

**实验结果与分析** 

本部分的研究揭示了一个全新的 5ILD 环合成机制。实验证明，黄素依赖性单加氧酶 (FMO) ParB 负责将胺 **15** 的氮原子氧化为羟胺 **16**。这个中间体 **16** 非常不稳定，在没有后续酶的情况下会迅速脱水变回亚胺 **5**。关键的环化步骤由 ParF 催化，它是一个 dienelactone hydrolase (DLH) 家族的酶。经典的 DLH 催化内酯的水解，而 ParF 在这里却催化了逆向反应——内酰胺化，将羟胺 **16** 的游离羧基和 N-羟基直接环化形成 5ILD 环，生成 parnafungin A (**1**)。这一发现颠覆了传统的认知，即环化通常需要 ATP 供能来活化羧基。ParF 利用其半胱氨酸蛋白酶样的催化三联体，实现了非 ATP 依赖的直接内酰胺化，是一种新颖高效的生物催化策略。

**Figure 8A**：展示了 5ILD 环形成的两步机制：ParB (FMO) 催化 N-羟基化生成 **16**，ParF (hydrolase) 催化内酰胺化生成 **1**。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.014.png)

**Figure 8B**：酵母体内重构实验。只有当 ParB 和 ParF 同时在 ParEGIOJDPT 体系中表达时 (ParEGIOJDPTBF)，才能检测到最终产物 **1** 和 **2** (m/z 452)。缺少 ParB 或 ParF，或将 ParF 的催化残基 C138 突变为 C138A，都完全无法生成产物。这证明了 ParB 和 ParF 对于 5ILD 环的形成都是必需的。

![...](..\Journal%20of%20the%20American%20Chemical%20Society\images\2025%20JACS%20-%20含%205-异噁唑烷酮的六环结构%20Parnafungin%20的生物合成.015.png)

**Figure 8C**：多酶体外级联反应。

- 将底物 **5** 与 ParT, ParB, ParF 三个纯化的酶以及 NADPH 一起孵育 (ParTBF+5)，成功检测到了产物 **1** 和 **2**，完整重现了体内过程。
- 在缺少 ParF 的情况下 (ParTB+5)，反应体系中没有生成 **1** 和 **2**，而是出现了一个新的副产物 **17** (m/z 468)。通过结构鉴定，**17** 是羟胺 **16** 被过度氧化形成的硝酮 (N-oxide)。这表明中间体 **16** 确实生成了，但由于缺少 ParF 的及时环化而走向了副反应路径。
- 在 ParB+5 的反应中，没有生成 **17**，说明 **17** 不是由 **5** 直接氧化而来，而是必须经过 **16**。这些精巧的体外实验设计，不仅证实了 ParB 和 ParF 的功能，还成功捕获了不稳定中间体 **16** 存在的间接证据。

**研究部分五：Parnafungin C 和 D 的后期生物合成**

**实验目的与设计**

本部分旨在阐明 parnafungin A (1) 如何被进一步修饰，生成 Parnafungin C (3) 和 D (4)。

设计步骤如下：

1. 提出假设：对比 **1**, **3**, **4** 的结构，推测 **3** 是由 **1** 的 C7-OH 甲基化而来，**4** 是由 **3** 的 C13-C14 双键环氧化而来。
2. 酶的筛选：在 *par* 簇中寻找甲基转移酶和 P450 氧化酶。ParX 被注释为甲基转移酶，ParU 是另一个 P450 酶。
3. 体内重构：在能够原位生成 **1** 的酵母体系 (ParTBF+5) 中，分别表达 ParX 和 ParU，检测 **3** 和 **4** 的生成。由于 **1** 的水溶性差，实验采用饲喂前体 **5**，在酵母中原位生成 **1** 作为后续反应的底物。

**实验结果与分析**

研究确认了 Parnafungin 家族的后期多样化步骤。甲基转移酶 ParX 负责将 parnafungin A (**1**) 的 C7-OH 进行 O-甲基化，生成 parnafungin C (**3**)。随后，P450 酶 ParU 催化一个连续的去饱和-环氧化反应，将 **3** 的 C13-C14 单键转化为环氧乙烷，生成最终产物 parnafungin D (**4**)。这种由单个 P450 酶催化的去饱和-环氧化串联反应在天然产物生物合成中也偶有报道，体现了 P450 酶催化功能的多样性。

- **Figure 9A**：展示了后期修饰的路径：**1** 经 ParX 甲基化为 **3**，**3** 经 ParU 环氧化为 **4**。

- **Figure 9B**：酵母体内重构 parnafungin C (**3**)。在能够生成 **1** 的体系中 (ParTBF+5)，额外表达 ParX (ParTBFX+5)，HPLC-MS 检测到了与 **3** 标准品一致的新产物峰 (m/z 466 [M+H]⁺)，同时还检测到其开环产物 **7**。这证实了 ParX 的甲基转移酶功能。

**五：总体结论**

1. **完整途径的阐明**：本研究成功解析了具有抗真菌活性的复杂天然产物 Parnafungin 的完整生物合成途径，揭示了自然界如何从一个已知的天然产物 blennolide C 出发，通过三组独立的酶促级联反应（芳基偶联、D 环构建、5ILD 环形成）和后期修饰，逐步构建出一个结构新颖、活性独特的六环骨架分子。
2. **新颖酶学机制的发现**：研究发现了多个新颖的酶学机制，包括：(1) 由 ParJ (CoA 连接酶)、ParO (P450) 和 ParD (DUF829 水解酶) 协同催化的芳基偶联新策略；(2) 由非典型 SDR ParT 催化的高度共轭亚胺的高效还原；(3) 由 ParB (FMO) 和 ParF (DLH 同源物) 串联催化的、非 ATP 依赖的 5-异噁唑烷酮 (5ILD) 环的全新合成逻辑。
3. **理论与应用贡献**：这些发现极大地丰富了我们对天然产物生物合成逻辑和酶催化多样性的理解。所揭示的新酶（如 ParOJD, ParT, ParF）具有作为生物催化剂在合成化学中构建 C-C, C-N, N-O, C-O 键的巨大潜力，为未来通过基因组挖掘发现新结构天然产物和设计新的生物合成途径提供了理论指导和工具酶库。

**六：论文评价**

**优点与创新**

1. **逻辑严谨的系统性研究**：论文的研究思路清晰，从基因簇鉴定到分步异源重构，再到关键酶的体外验证和定点突变，环环相扣，逻辑严谨，是天然产物生物合成研究的典范。
2. **发现全新的三酶芳基偶联系统**：揭示的 ParOJD 级联反应是一种前所未有的生物芳基偶联模式，特别是“隐蔽式”硫酯化策略和 DUF829 家族蛋白功能的首次阐明，是本文最重大的创新之一。
3. **阐明 5ILD 环的全新生物合成机制**：首次揭示了 5-异噁唑烷酮这一罕见药效团的形成机制，特别是 ParF 作为一个 DLH 同源物催化非 ATP 依赖的内酰胺化反应，颠覆了传统认知，极具新颖性。
4. **对非典型酶功能的深入挖掘**：成功表征了非典型 SDR ParT 的功能，证明其能还原化学惰性的共轭亚胺，拓展了对 SDR 酶催化潜力的认识。
5. **巧妙的实验设计**：面对不稳定的中间体，作者通过体外多酶级联反应和捕获副产物 (**17**) 的方式，巧妙地验证了反应机理，体现了高超的实验设计能力。

**未来研究方向**

1. **结构生物学研究**：解析关键酶 ParO, ParD, ParT, ParB, ParF 与其底物或辅酶的复合物晶体结构，从原子层面揭示它们独特催化机制的结构基础，例如 ParO 如何识别两个完全不同的底物，ParF 如何催化内酰胺化而非水解。
2. **酶的定向改造与合成生物学应用**：对 ParOJD 芳基偶联系统进行蛋白质工程改造，拓展其底物范围，使其能够偶联更多非天然的芳香分子，开发其在生物催化领域的应用价值。同样，ParF 作为一个非 ATP 依赖的内酰胺化酶，也具有巨大的合成应用潜力。
3. **剩余基因功能探索**：基因簇中仍有 ParC (SDR) 和 ParV (细胞色素 b5) 的功能尚不完全清楚。ParV 可能作为电子传递体辅助 P450 酶，而 ParC 的具体功能仍是未解之谜。进一步研究这些基因的功能，将有助于我们更全面地理解 Parnafungin 的生物合成调控网络。

**七：关键问题及回答**

**问题一：作者是如何证明芳基偶联反应是由 ParJ, ParO, ParD 三个酶协同催化，而不仅仅是 P450 酶 ParO？**

**回答**： 作者通过一系列严谨的体内和体外实验证明了这一点：

1. **体内组合表达效率对比**：在酵母中，单独表达 ParO (以及合成前体 **9** 的酶) 时，芳基偶联产物 **12** 的产量极低，底物 **9** 的转化率低于 5%。然而，当同时表达 ParJ 和 ParD 后，**9** 几乎被完全消耗，**12** 的产量显著提高 (Figure 6B)。这表明 ParJ 和 ParD 对于高效偶联是必需的。
2. **ParJ 功能的体外验证**：作者将 ParJ 蛋白纯化，在体外与底物邻氨基苯甲酸、ATP 和 CoA 孵育，通过质谱成功检测到产物邻氨基苯甲酰-CoA (AA-CoA) 的生成 (Figure 6C)。这直接证明了 ParJ 是一个 CoA 连接酶，负责活化邻氨基苯甲酸。
3. **硫酯底物模拟物饲喂实验**：作者合成了 AA-CoA 的模拟物 AA-SNAC，并将其饲喂给表达 ParO 和 ParD 的酵母。结果发现，AA-SNAC 能够高效地被转化为偶联产物，证明了硫酯是 ParO 的有效底物。更重要的是，在有 ParD 的体系中，硫酯偶联产物 **14** 减少，而水解产物 **12** 增加 (Figure 6D)，这证实了 ParD 扮演着水解硫酯的角色。
4. **ParD 的定点突变**：作者将 ParD 预测的催化残基 S110 突变为丙氨酸，发现其完全丧失了水解活性 (Figure S5)，进一步确认了其水解酶功能。

综上所述，通过证明 ParJ 负责底物活化、ParO 负责以活化底物进行偶联、ParD 负责产物释放，作者确立了这个三酶协同催化的新颖机制。

**问题二：在 5ILD 环的形成过程中，作者是如何确定 ParB 是 N-加氧酶，而 ParF 是内酰胺化酶的？**

**回答**： 作者通过体内基因组合、体外酶学重构和副产物捕获等多种手段确定了二者的功能：

1. **体内基因组合实验**：在能够产生前体 **15** 的酵母体系中，只有当 ParB 和 ParF 同时存在时，才能检测到最终产物 parnafungin A/B (**1/2**)。缺少其中任何一个酶，或将 ParF 的催化半胱氨酸 C138 突变，都会导致产物完全消失 (Figure 8B)。这证明了二者都是 5ILD 环形成所必需的。
2. **体外多酶级联反应**：作者用纯化的 ParT, ParB, ParF 进行体外反应。当三个酶同时存在时，可以从底物 **5** 成功合成出 **1** 和 **2**。而当去掉 ParF 后，反应不仅不能生成 **1** 和 **2**，反而积累了一个新的副产物 **17** (Figure 8C)。
3. **不稳定中间体的间接证据**：通过 NMR 和 HRMS，作者将副产物 **17** 鉴定为 N-羟基化中间体 **16** 被过度氧化形成的硝酮。**17** 的出现是 **16** 曾经存在的有力证据。因为 **17** 只有在 ParB 存在时才生成，说明 ParB 催化生成了 **16**；而 **17** 的积累又是因为缺少 ParF 进行及时的环化。
4. **酶家族功能类比**：ParB 属于 FMO 家族，该家族酶催化 N-加氧反应是已知的。ParF 属于 DLH 家族，虽然通常催化水解，但其催化三联体具备催化逆向反应（酰胺化）的化学可能性。

通过这些实验，作者清晰地描绘了反应路径：ParB 生成不稳定的羟胺 **16**，随后 ParF 必须立即将其环化为稳定的 5ILD 环，否则 **16** 就会走向脱水或过度氧化的副反应路径。

**问题三：ParT 被描述为“非典型”短链还原酶 (SDR)，其“非典型”之处体现在哪里？作者如何确认它就是负责还原亚胺中间体 5 的酶？**

**回答**：

1. **“非典型”之处**：典型的 SDR 酶含有一个高度保守的催化三联体，通常由丝氨酸 (Ser)、酪氨酸 (Tyr) 和赖氨酸 (Lys) 组成 (S-Y-K)，在催化过程中起关键作用。然而，通过序列比对分析，作者发现 ParT 缺乏这个经典的催化三联体。这种结构上的“非典型”性使得它是否具备还原酶活性以及其催化机制变得不确定，是研究中的一个难点。
2. **功能确认**：尽管结构非典型，作者通过直接的体外生化实验确凿地证明了其功能。他们将 ParT 蛋白在大肠杆菌中表达纯化，然后将其与底物——亚胺中间体 **5** (parnafungin A') 和还原性辅酶 NADPH 一起孵育。通过 HPLC-MS 分析，他们观察到底物 **5** (m/z 452) 被显著消耗，同时生成了分子量为 454 的新产物 **15** (m/z 454 = 452 + 2H)，这与亚胺被还原为胺的化学过程完全吻合 (Figure 7C)。在缺少 ParT 或 NADPH 的对照实验中，则没有还原反应发生。这个体外实验结果直接、有力地证明了 ParT 就是负责催化这一关键还原步骤的酶。
