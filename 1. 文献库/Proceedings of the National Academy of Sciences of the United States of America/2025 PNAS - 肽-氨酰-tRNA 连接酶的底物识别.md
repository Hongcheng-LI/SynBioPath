**期刊名称**：Proceedings of the National Academy of Sciences of the United States of America (PNAS)

**二：核心速览**

**研究背景**

肽键的形成是生物合成中至关重要的过程，主要通过核糖体或非核糖体肽合成酶 (NRPS) 完成。近年来，一类新型酶——肽-氨酰-tRNA 连接酶 (peptide-aminoacyl-tRNA ligases, PEARLs) 被发现，它们能够以氨酰-tRNA 依赖的方式，将氨基酸连接到肽底物的 C 端。这种非核糖体依赖的肽延伸机制为制备天然和非天然肽提供了新的途径，尤其是在非规范氨基酸 (noncanonical amino acids, ncAAs) 掺入蛋白质以进行生物偶联和功能研究方面具有巨大潜力。然而，尽管已发现多种 PEARLs 能够连接 Cys, Ala, Trp, Gly, Leu, Asn, Thr 等氨基酸，但其对肽底物和氨酰-tRNA 的识别机制尚不清楚，这限制了对这些酶的工程改造和应用。

**前期研究**

1. **蛋白质标记技术的重要性**：在生物分子研究中，蛋白质标记技术具有广泛应用，尤其通过引入 ncAAs 提供生物正交手柄，便于与外源分子偶联。
2. **传统 ncAA 掺入方法**：主要依赖正交翻译系统，通过工程改造正交 tRNA 和氨酰-tRNA 合成酶对，将 ncAAs 掺入特定密码子。但该方法存在与核糖体不兼容、错配和掺入效率低等问题。
3. **PEARLs 的发现**：最近发现的 PEARLs 是一类新型酶，它们独立于核糖体，以 ATP 依赖和模板非依赖的方式，利用氨酰-tRNA 作为氨基酸供体，催化氨基酸与肽底物 C 端的连接。
4. **PEARLs 的进化关系**：PEARLs 在进化上与 I 类 lanthipeptide 脱水酶 (LanBs) 相关，后者通过谷氨酰-tRNA 依赖机制使底物肽中的 Ser 和 Thr 残基侧链谷氨酰化。但 PEARLs 通常只包含类似 LanB 谷氨酰化结构域的部分，且需要 ATP，而 LanBs 则不需要。
5. **底物识别机制的空白**：目前对于 PEARLs 识别氨酰-tRNA 的结合位点以及酰胺键形成活性位点的信息知之甚少，这阻碍了对其底物特异性的全面理解。
6. **无细胞表达 (Cell-free expression, CFE) 技术的兴起**：CFE 已成为一种快速评估 RiPP 生物合成酶底物特异性的强大工具，具有速度快、无核酸酶/蛋白酶干扰、可表达毒性或难表达蛋白等优点。

**本文突破点**

1. **全面评估了代表性 PEARL (BhaB^Trp) 的肽底物特异性**：利用 CFE 体系，首次系统性地研究了 BhaB^Trp 对其肽底物 BhaA-Ala C 端序列的识别机制，揭示了该酶对 C 端氨基酸的广泛耐受性。
2. **鉴定了 Trp 连接所需的最小肽序列**：通过截短、丙氨酸扫描和单点突变实验，确定了 BhaB^Trp 催化 Trp 连接所需的最小 14-mer 肽序列 (LEEIEDKVAPLALA)。
3. **利用 AlphaFold3 提供了结构机制洞察**：首次利用 AlphaFold3 模型预测了 BhaB^Trp 与 tRNA^Trp、肽底物 BhaA-Ala 和 ATP 的四元复合物结构，揭示了关键的结合相互作用，并为 tRNA 识别机制提供了线索。
4. **通过定点突变验证了 tRNA 识别的关键残基**：基于 AlphaFold3 预测，设计并验证了 BhaB^Trp 中与 tRNA 3'-CCA 序列相互作用的关键残基（如 Arg208, Tyr205, Arg429），证实了这些残基对酶活性的重要性。
5. **成功将最小序列应用于蛋白质 C 端标记**：将鉴定的最小 14-mer 序列作为标签，成功实现了 eGFP, 溶菌酶 (lysozyme) 和 MBP 等模型蛋白的 C 端 Trp 和 5-Br-Trp 标记，展示了 PEARLs 在生物分子工程中的应用潜力。

**研究难点**

1. **PEARLs 底物识别机制的复杂性**：PEARLs 对肽底物的识别不仅涉及 C 端氨基酸，还可能涉及更长的序列和二级结构，这使得系统性地探究其特异性成为挑战。
2. **缺乏 PEARLs 的晶体结构**：在 AlphaFold3 出现之前，由于缺乏 PEARLs 的实验结构，对其催化机制和底物结合模式的理解一直受限。
3. **AlphaFold3 模型的局限性**：尽管 AlphaFold3 能够预测复合物结构，但其对非天然突变、tRNA 修饰以及氨酰-tRNA 结合模式的预测能力仍有限，需要实验验证和进一步的结构生物学研究。
4. **CFE 体系中氨酰-tRNA 的制备与纯化**：在 CFE 体系中，确保氨酰-tRNA 的高效制备和纯度，避免未加载的 tRNA 或其他杂质干扰，对于准确评估酶活性至关重要。

**三：研究方法**

- **无细胞表达 (Cell-free expression, CFE)**：使用 PURExpress® 体外蛋白质合成系统，快速表达 PEARL 酶 (BhaB^Trp) 及其肽底物 (BhaA-Ala) 及其变体，用于快速评估酶活性和底物特异性。
- **基质辅助激光解吸电离飞行时间质谱 (MALDI-TOF MS)**：用于监测 CFE 反应产物，通过检测肽底物质量的变化（如 Trp 连接后质量增加 186 Da）来评估酶的催化活性和转化率。
- **十二烷基硫酸钠聚丙烯酰胺凝胶电泳 (SDS-PAGE)**：用于分析 CFE 反应中蛋白质的表达和纯化情况，辅助确认酶和底物的存在。
- **肽底物变体构建**：通过基因工程手段，构建了一系列 BhaA-Ala 肽底物变体，包括 N 端截短、C 端区域的丙氨酸扫描、单点突变和序列延伸，以系统性地探究底物特异性。
- **AlphaFold3 结构预测**：利用最新发布的 AlphaFold3 程序，预测 BhaB^Trp 与 tRNA^Trp、肽底物 BhaA-Ala 和 ATP 的四元复合物结构，以获得原子水平的结合相互作用信息。
- **结构分析与可视化**：使用 ChimeraX 软件对 AlphaFold3 预测的结构进行可视化和分析，评估预测的局部距离差异测试 (pLDDT) 和预测对齐误差 (PAE) 值，以评估模型置信度。
- **定点突变 (Site-directed mutagenesis)**：基于 AlphaFold3 预测的关键相互作用残基，对 BhaB^Trp 酶进行定点突变（如 R208A, Y205A, R429A），并在 CFE 体系中评估突变酶的活性，以实验验证预测的结构功能关系。
- **蛋白质 C 端标记**：将鉴定的最小肽序列作为标签，通过基因融合技术连接到 eGFP, 溶菌酶 (lysozyme) 和麦芽糖结合蛋白 (MBP) 等模型蛋白的 C 端，并在 CFE 体系中测试 BhaB^Trp 催化 Trp 和 5-Br-Trp 连接到这些融合蛋白 C 端的效率。
- **蛋白质纯化**：使用镍-硝酸三乙酸 (Ni-NTA) 金属亲和层析纯化带有 His6 标签的融合蛋白，并通过烟草蚀刻病毒 (TEV) 蛋白酶切割去除 His6 标签，以便进行后续的质谱分析。

**四：实验设计及结果分析**

**研究部分一：无细胞表达体系的建立与 PEARL 酶 BhaB^Trp 活性的初步验证**

**实验目的与设计**

本部分旨在建立一个高效的无细胞表达 (CFE) 体系，用于快速评估 PEARL 酶 BhaB^Trp 的催化活性，并初步验证其将色氨酸 (Trp) 连接到肽底物 BhaA-Ala C 端的潜力。

按逻辑顺序，分点阐述为达到此目的而设计的关键实验步骤：

1. **CFE 体系构建**：使用 PURExpress® 体外蛋白质合成系统，分别表达带有 His6 标签的肽底物 BhaA-Ala 和 PEARL 酶 BhaB^Trp，或同时共表达两者。
2. **活性监测**：通过基质辅助激光解吸电离飞行时间质谱 (MALDI-TOF MS) 监测反应产物的质量变化，并辅以 SDS-PAGE 凝胶电泳确认蛋白质表达。

**实验结果与分析**

作者成功建立了 CFE 体系。当单独表达 His6-BhaA-Ala 时，MALDI-TOF MS 检测到对应其理论质量的峰（m/z = 6734.7）。当 His6-BhaA-Ala 与 BhaB^Trp 共表达时，观察到一个新的峰，其质量比 BhaA-Ala 增加了 186 Da，这与连接一个 Trp 残基（Trp 的分子量为 204 Da，连接时脱去一分子水 18 Da，净增 186 Da）的质量变化一致。SDS-PAGE 结果也显示了相应的蛋白质条带。这些结果明确证实了 BhaB^Trp 在 CFE 体系中能够有效地将 Trp 连接到肽底物 BhaA-Ala 的 C 端。

**Figure 1A (PEARL 催化机制示意图)**：清晰地描绘了 PEARL 酶催化肽延伸的通用机制。首先，PEARL 利用 ATP 磷酸化肽底物的 C 端羧基，形成活化的酰基磷酸中间体。随后，活化的羧基与氨酰-tRNA 的氨基发生缩合反应，形成新的酰胺键，并释放 tRNA。这为理解后续实验提供了基础。

**Figure 1B (BhaA 序列及级联反应示意图)**：展示了本研究中使用的肽底物 BhaA 的氨基酸序列，以及 BhaB^Ala 和 BhaB^Trp 酶的级联反应。BhaB^Ala 首先将 Ala 连接到 BhaA 的 C 端形成 BhaA-Ala，然后 BhaB^Trp 将 Trp 连接到 BhaA-Ala 的 C 端形成 BhaA-Ala-Trp。这个图谱为后续的底物特异性研究提供了背景。

**研究部分二：BhaB^Trp 对肽底物 C 端序列的识别特异性分析**

**实验目的与设计**

本部分旨在系统性地探究 BhaB^Trp 酶对肽底物 BhaA-Ala C 端序列的识别要求，以确定哪些残基或结构特征对其活性至关重要。

按逻辑顺序，分点阐述为达到此目的而设计的关键实验步骤：

1. **N 端截短实验**：设计 BhaA-Ala 的截短变体，评估底物长度对酶活性的影响。
2. **C 端区域丙氨酸扫描**：对 BhaA-Ala C 端区域的关键残基进行丙氨酸 (Ala) 替换，观察酶活性的变化。
3. **单点突变实验**：对 C 端特定残基进行 Leu 或 Phe 替换，评估其对酶活性的影响。
4. **C 端末端残基替换**：将 BhaA-Ala 的 C 端 Ala40 替换为多种不同性质的氨基酸（包括极性、非极性、大体积、小体积、芳香族等），评估酶对 C 端末端残基的耐受性。
5. **序列延伸实验**：在 BhaA-Ala 序列中插入 Ala 模块，评估底物长度增加对酶活性的影响。

**实验结果与分析**

实验结果显示 BhaB^Trp 对肽底物 C 端序列的识别具有显著的耐受性。

1. **N 端截短**：将 BhaA-Ala 截短至 18-mer (Ile23-Ala40) 仍能被 BhaB^Trp 完全修饰。进一步截短至 12-mer (Glu29-Ala40) 也能被修饰，但效率较低。这表明 N 端至 Glu28 的残基对于酶-底物相互作用并非关键，但 Ile23 至 Glu28 区域对最佳加工很重要。
2. **丙氨酸扫描**：将 Phe25-Glu28 区域替换为 Ala 残基（FDLE 变为 AAAA）后，BhaB^Trp 仍能完全转化该突变体。然而，完全删除这些残基的突变体仅被部分加工。这表明酶对肽底物长度敏感，可能需要一定的二级结构。
3. **PLALA 序列的重要性**：将 C 端 PLALA 序列改为 PAAAA 后，BhaB^Trp 无法加工该肽变体。将 LALA 基序前的序列 (Lys33-Pro36) 替换为 Ala 也导致活性丧失。删除 EDKVA 序列同样废除了活性。这些结果表明，BhaB^Trp 对 Pro36 之前的变化具有耐受性，但超出该点的替换会破坏活性。
4. **C 端末端残基的广泛耐受性**：将 Ala40 替换为多种氨基酸（包括 Ser, Lys, Glu, Leu, Phe, Trp, Tyr, Pro, Gly）后，BhaB^Trp 仍能连接 Trp。其中，A40P 和 A40W 变体仅被部分加工。这种对 C 端末端残基的低特异性表明，C 端侧链的大小、极性或电荷并非关键识别因素，再次暗示二级结构和/或底物长度可能更重要。
5. **序列延伸**：在 Ala35 之前插入 Ala 模块（BhaA-Ala extend-1, -2, -3）的延伸底物，BhaB^Trp 仍能完全修饰。

**研究部分三：AlphaFold3 预测 BhaB^Trp 四元复合物结构与 ATP 结合模式**

**实验目的与设计**

本部分旨在利用 AlphaFold3 预测 BhaB^Trp 与其底物肽 BhaA-Ala、tRNA^Trp 和辅因子 ATP 形成的四元复合物结构，以从三维角度理解酶的催化机制和底物结合模式。

按逻辑顺序，分点阐述为达到此目的而设计的关键实验步骤：

1. **AlphaFold3 建模**：使用 AlphaFold3 程序预测 BhaB^Trp、tRNA^Trp、BhaA-Ala 和 ATP 的四元复合物结构。
2. **结构置信度评估**：通过分析预测局部距离差异测试 (pLDDT) 和预测对齐误差 (PAE) 值，评估模型预测的置信度。
3. **关键相互作用分析**：识别模型中 BhaB^Trp 与 ATP、tRNA^Trp 和 BhaA-Ala 之间的关键相互作用残基。

**实验结果与分析**

AlphaFold3 模型成功预测了 BhaB^Trp 四元复合物的结构，且关键相互作用区域的置信度较高。

1. **ATP 结合模式**：模型显示 ATP 结合在 PEARLs 典型的 ATP-GRASP 结构域中。Glu813 通过氢键与核糖的 2' 和 3' 羟基以及一个 Mg2+ 离子相互作用。Asp558 与另一个 Mg2+ 离子相互作用，并与 Arg815 协同，Arg815 进而与 ATP 的 γ-磷酸氧和 BhaA-Ala 的 C 端羧基相互作用，这与之前 TglB 的突变研究结果一致。Arg715 和 Phe755 侧链靠近 ATP 的腺嘌呤基团，Arg715 可能通过阳离子-π 相互作用。
2. **tRNA 结合模式**：tRNA^Trp 的 3'-CCA 末端被预测直接插入 BhaB^Trp 的活性位点。Arg208 被预测与 C75 和 A76 的磷酸酯键以及 2' 羟基相互作用，Tyr205 与 C75 核糖氧和 C74 磷酸酯氧相互作用，Gln201 与 C74 氢键，Arg197 与 G73 相互作用。这些残基在 PEARLs 中高度保守，且突变实验证实它们对酶活性至关重要。
3. **肽底物结合模式**：BhaA-Ala 肽底物的 C 端被预测位于 tRNA^Trp 3' 末端和 ATP γ-磷酸的附近，处于催化活化状态。模型显示 BhaA-Ala 的 C 端螺旋结构（Leu27-Val34）与 BhaB^Trp 之间存在多个相互作用，但 CFE 实验表明这些相互作用并非活性所必需。

**研究部分四：AlphaFold3 预测 BhaB^Trp 与 tRNA^Trp 和肽底物的相互作用及突变验证**

**实验目的与设计**

本部分旨在深入分析 AlphaFold3 模型中 BhaB^Trp 与 tRNA^Trp 和 BhaA-Ala 肽底物的具体相互作用位点，并进行突变实验验证，以确认预测的结构功能关系。

按逻辑顺序，分点阐述为达到此目的而设计的关键实验步骤：

1. **tRNA 相互作用残基分析**：基于 AlphaFold3 模型，识别 BhaB^Trp 中与 tRNA^Trp 3'-CCA 末端序列（C75, A76）以及判别碱基 G73 相互作用的关键残基。
2. **tRNA 相互作用残基突变验证**：设计 BhaB^Trp 的点突变体 R208A, Y205A, R429A，并在 CFE 体系中评估其活性，以实验验证这些残基对 tRNA 结合的重要性。
3. **肽底物相互作用残基分析**：基于 AlphaFold3 模型，识别 BhaB^Trp 中与 BhaA-Ala 肽底物 C 端螺旋结构（Leu27-Val34）相互作用的关键残基。
4. **肽底物相互作用残基突变验证**：设计 BhaB^Trp 的点突变体 S619D 和 S503R，并在 CFE 体系中评估其活性。

**实验结果与分析**

AlphaFold3 模型预测了 BhaB^Trp 与 tRNA^Trp 3'-CCA 末端之间存在多个关键相互作用。Arg208 被预测与 C75 和 A76 的磷酸酯键以及 2' 羟基相互作用，Tyr205 与 C75 核糖氧和 C74 磷酸酯氧相互作用，Gln201 与 C74 氢键，Arg197 与 G73 相互作用。这些预测得到了突变实验的有力支持：R208A, Y205A, R429A 突变体均显示酶活性显著降低，表明这些残基对于 tRNA 结合和催化至关重要。

然而，对于肽底物 BhaA-Ala 与 BhaB^Trp 的相互作用，模型预测 Glu31 与 Ser619 氢键，Asp32/Glu28 与 Lys620 相互作用，Lys33 与 Ser503/Thr681 相互作用。但 CFE 实验和突变体 S619D, S503R 的结果表明，这些预测的相互作用并非酶活性所必需。这可能意味着酶对肽底物的识别更多依赖于整体的二级结构或长度，而非特定的点对点相互作用。AlphaFold3 在预测非天然突变和解释选择性方面仍存在局限性。

**研究部分五：最小底物序列的鉴定与蛋白质 C 端标记应用**

**实验目的与设计**

本部分旨在基于前述的底物特异性研究结果，鉴定 BhaB^Trp 催化 Trp 连接所需的最小肽序列，并将其作为标签应用于模型蛋白质的 C 端标记，以展示 PEARLs 在生物技术中的应用潜力。

按逻辑顺序，分点阐述为达到此目的而设计的关键实验步骤：

1. **最小序列确定**：结合 CFE 实验数据和 AlphaFold3 模型对肽底物二级结构的预测，确定一个能够被 BhaB^Trp 有效修饰的最小 14-mer 肽序列 (LEEIEDKVAPLALA)。
2. **蛋白质标签构建**：将该 14-mer 序列作为 C 端标签，通过基因融合技术连接到增强型绿色荧光蛋白 (eGFP)、溶菌酶 (lysozyme) 和麦芽糖结合蛋白 (MBP) 等模型蛋白质的 C 端。
3. **C 端标记实验**：在 CFE 体系中，共表达融合蛋白质和 BhaB^Trp，并尝试连接 Trp 和非规范氨基酸 5-溴色氨酸 (5-Br-Trp)。
4. **产物分析**：通过 Ni-NTA 纯化、TEV 蛋白酶切割和 MALDI-TOF MS 分析，确认蛋白质 C 端是否成功连接了 Trp 或 5-Br-Trp。

**实验结果与分析**

基于 CFE 实验和 AlphaFold3 模型，作者确定了 BhaA-Ala 的 C 端 14-mer 序列 (LEEIEDKVAPLALA) 能够形成螺旋结构，并被 BhaB^Trp 有效修饰。将该 14-mer 序列作为标签，成功地将其融合到 eGFP, lysozyme 和 MBP 等模型蛋白质的 C 端。在 CFE 体系中，当这些融合蛋白质与 BhaB^Trp 共表达时，MALDI-TOF MS 分析证实了 Trp 能够成功连接到这些蛋白质的 C 端。此外，非规范氨基酸 5-Br-Trp 也能被连接，尽管初始效率较低，但在优化 TrpRS 纯化步骤后有所改善。这些结果证明了 PEARLs 具有作为通用工具，实现蛋白质 C 端特异性标记的潜力，为生物分子研究和工程提供了新的策略。

- **Figure 5 (蛋白质 C 端标记策略示意图)**：清晰地展示了将 Trp 或 5-Br-Trp 连接到蛋白质 C 端的代表性策略，以 eGFP 为例。图中显示了 N-His6-eGFP 融合蛋白，其 C 端连接了 TEV 酶切位点和 14-mer 标签 (LEEIEDKVAPLALA)。在 BhaB^Trp 作用下，Trp 被连接到标签的 C 端。TEV 酶切后，可以分离出带有 Trp 标记的 eGFP。该图直观地展示了本研究的最终应用目标和实现路径。

**五：总体结论**

1. **底物特异性解析**：本研究首次系统性地阐明了 PEARL 酶 BhaB^Trp 对肽底物 C 端序列的识别机制，发现其对 C 端氨基酸具有广泛的耐受性，但对肽底物的长度和二级结构敏感。
2. **最小序列鉴定**：成功鉴定了 BhaB^Trp 催化 Trp 连接所需的最小 14-mer 肽序列 (LEEIEDKVAPLALA)。
3. **结构机制洞察**：利用 AlphaFold3 预测了 BhaB^Trp 四元复合物结构，揭示了 ATP 和 tRNA 结合的关键相互作用残基，并通过定点突变实验验证了 tRNA 识别的关键位点。
4. **蛋白质标记应用**：将鉴定的最小序列作为标签，成功实现了 eGFP, 溶菌酶和 MBP 等模型蛋白质的 C 端 Trp 和 5-Br-Trp 标记，展示了 PEARLs 在生物分子工程和蛋白质标记领域的巨大应用潜力。

**贡献与影响**：本研究不仅深化了我们对 PEARL 酶催化机制和底物识别的理解，还为利用这些酶进行蛋白质工程和非规范氨基酸掺入提供了宝贵的指导。鉴定的最小肽序列可作为通用的 C 端标记标签，为制备具有特定功能（如生物正交偶联、荧光标记）的“设计分子”开辟了新途径，对生物化学、分子生物学和生物技术领域具有重要意义。

**六：论文评价**

**优点与创新**

1. **CFE 体系的高效应用**：充分利用 CFE 体系快速、高通量地评估酶的底物特异性，克服了传统体内突变和异源表达的效率瓶颈，是该研究得以系统性开展的关键。
2. **AlphaFold3 的前瞻性应用**：在 PEARL 酶缺乏晶体结构的情况下，率先利用 AlphaFold3 预测四元复合物结构，为理解其催化机制和底物结合提供了宝贵的原子水平洞察，并成功指导了突变实验。
3. **底物识别的全面性分析**：通过截短、丙氨酸扫描、单点突变和 C 端末端残基替换等多种策略，对 BhaB^Trp 的肽底物特异性进行了迄今为止最全面的评估。
4. **从基础研究到应用转化**：不仅深入探究了酶的底物识别机制，还成功将研究成果转化为实际应用——开发了 PEARL 介导的蛋白质 C 端标记技术，具有明确的生物技术价值。

**未来研究方向**

1. **PEARL 酶的结构解析**：尽管 AlphaFold3 提供了宝贵的预测结构，但通过 X 射线晶体学或冷冻电镜 (Cryo-EM) 获得 PEARL 酶及其复合物的实验结构，将提供最高分辨率的证据，进一步精确理解其催化机制和底物识别。
2. **氨酰-tRNA 识别的机制**：本研究主要关注肽底物识别。未来应深入探究 PEARL 酶如何特异性识别氨酰-tRNA 上的氨基酸部分，以及 tRNA 骨架上的哪些特征对识别至关重要。这可能需要结合 Flexizyme 技术来制备带有不同氨基酸和 tRNA 序列的氨酰-tRNA。
3. **工程改造 PEARL 酶以连接更多 ncAAs**：基于本研究对底物特异性的理解，可以尝试对 BhaB^Trp 或其他 PEARL 酶进行理性设计或定向进化，以拓宽其对 ncAAs 的连接范围，从而实现更广泛的蛋白质功能化。

**七：关键问题及回答**

**问题一：无细胞表达 (CFE) 体系在本研究中发挥了怎样的关键作用？它相比传统体内表达有何优势？**

**回答**：CFE 体系在本研究中发挥了至关重要的作用，是实现高效、系统性底物特异性研究的关键。

1. **快速评估与高通量筛选**：CFE 体系允许研究人员在数小时内完成蛋白质表达和酶活性评估，这比传统的体内表达（需要转化、培养、诱导、裂解等多个步骤）快得多。这种速度使得作者能够快速制备和测试数百种肽底物变体，从而系统性地探究 BhaB^Trp 的底物特异性（如 Figure 2B 所示）。
2. **避免毒性与难表达问题**：CFE 体系可以在没有活细胞的限制下进行，因此可以表达对宿主细胞有毒性或难以表达的蛋白质。这为研究那些在体内可能干扰细胞生理的 PEARL 酶或其底物提供了便利。
3. **易于控制反应条件**：在 CFE 体系中，研究人员可以精确控制反应的组分和条件，例如底物浓度、辅因子浓度等，这有助于更准确地评估酶的动力学参数和底物特异性。
4. **简化纯化步骤**：CFE 产物通常比体内表达产物更纯净，减少了下游纯化工作的复杂性，尤其是在需要进行质谱分析时，可避免细胞内杂质的干扰。

**问题二：AlphaFold3 模型在本研究中提供了哪些关键的结构洞察？其预测的置信度是如何评估的？**

**回答**：AlphaFold3 模型在本研究中提供了前所未有的原子水平结构洞察，极大地推动了对 PEARL 酶催化机制的理解。

1. **四元复合物结构**：首次预测了 BhaB^Trp 与其底物肽 BhaA-Ala、tRNA^Trp 和辅因子 ATP 形成的四元复合物结构（Figure 3A），揭示了这些组分在活性位点中的空间排布。
2. **ATP 结合模式**：模型详细展示了 ATP 如何结合在 PEARL 酶的 ATP-GRASP 结构域中，并识别了 Glu813, Asp558, Arg815, Arg715, Phe755 等关键残基与 ATP 核糖、磷酸和腺嘌呤基团的相互作用（Figure 3B）。这些预测与之前 TglB 的突变研究结果一致，增强了可信度。
3. **tRNA 结合位点**：模型预测 tRNA^Trp 的 3'-CCA 末端直接插入酶的活性位点，并识别了 Arg208, Tyr205, Gln201, Arg197 等残基与 tRNA 骨架和碱基的相互作用（Figure 4A）。这些预测得到了后续定点突变实验的有力验证。
4. **肽底物结合位点**：模型展示了 BhaA-Ala 肽底物的 C 端螺旋结构如何与酶相互作用，并预测了其 C 端 PLALA 基序弯曲进入活性位点凹槽的模式（Figure 4C）。

**置信度评估**：作者通过分析 AlphaFold3 预测的两个关键参数来评估模型的置信度：

1. **预测局部距离差异测试 (pLDDT)**：这是一个残基级别的置信度指标，值越高（>90）表示预测结构高度可靠，可用于侧链相互作用分析；值在 70-90 之间表示主链预测良好；值低于 70 则不可靠。Figure 3C 通过 pLDDT 着色展示了复合物中不同区域的置信度。
2. **预测对齐误差 (PAE)**：这是一个残基对之间的置信度指标，表示预测的两个残基相对位置的准确性。PAE 值越低（<5 Å）表示两个残基的相对位置预测越准确。作者通过分析 PAE 图谱（SI Appendix, Fig. S11）来评估复合物界面相互作用的置信度。

通过结合这两个指标，作者能够有选择地信任模型预测的关键相互作用区域，并指导后续的实验验证。

**问题三：BhaB^Trp 对肽底物 C 端氨基酸的广泛耐受性有何生物学和生物技术意义？**

**回答**：BhaB^Trp 对肽底物 C 端氨基酸的广泛耐受性（如 Figure 2B, panel iv 所示，Ala40 可被多种氨基酸替换且酶活性保持）具有重要的生物学和生物技术意义：

1. **生物学意义**：这种广泛的耐受性表明，在自然界中，PEARL 酶可能不需要严格识别肽底物 C 端氨基酸的侧链，而是更依赖于肽底物的整体长度、二级结构或 C 端基序的特定构象来发挥作用。这可能赋予 PEARL 酶在不同生物合成途径中更大的灵活性，使其能够适应不同的肽底物，从而产生结构多样的天然产物。
2. **生物技术意义**：
- **蛋白质 C 端标记的通用性**：由于 BhaB^Trp 对 C 端氨基酸的性质不挑剔，这意味着研究人员可以将任何目标氨基酸（包括非规范氨基酸，如 5-Br-Trp）连接到带有特定标签序列的蛋白质 C 端。这使得 PEARL 酶成为一种通用的蛋白质 C 端标记工具，无需针对每种氨基酸或蛋白质进行复杂的酶工程改造。
- **“设计分子”的制备**：这种特性为制备“设计分子”提供了巨大潜力。通过将具有特定功能的非规范氨基酸（如荧光基团、生物正交偶联基团）连接到蛋白质 C 端，可以方便地对蛋白质进行功能化，用于成像、药物偶联、蛋白质相互作用研究等多种生物分子应用。
- **简化工程改造**：如果酶对 C 端氨基酸具有高度特异性，那么每次更换要连接的氨基酸时，都需要对酶进行复杂的工程改造。而 BhaB^Trp 的广泛耐受性大大简化了这一过程，加速了 PEARL 酶在生物技术中的应用开发。
