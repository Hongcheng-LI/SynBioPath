
**公众号推送流程**

\1. **文章生成**

1. 按照下列 AI 提示词，生成 AI 解读后的文章内容

[AI 辅助文献阅读](..\..\AI 辅助文献阅读.md)

2. 在知识库中对应的期刊目录下新建一个空白文档，将生成的文章粘贴至该文档中，并将文献中的图片粘贴到文章对应位置，并完成排版工作。

\2. **富文本转 Markdown 格式**

1. 复制飞书文档中的所有内容，复制到下面网址中，将富文本格式转化 markdown 格式，复制生成的 markdown 格式文档。

```<p>富文本转Markdown在线免费工具：</p><p>https://seo.yiguotech.com/seo-tools/rich-text-to-markdown-converter</p>```

2. 将上一步复制的 Markdown 格式文档内容粘贴至 markdown 编辑器中，检查并确认格式无误。

```<p>微信公众号 markdown 编辑器 1（强烈推荐）</p><p>https://md.doocs.org/</p><p></p><p>微信公众号 markdown 编辑器 2</p><p>https://www.md2wechat.cn/</p>```

\3. **公众号发表**

1. 复制微信公众号 markdown 编辑器中的排版后内容，粘贴至微信公众号中。
2. 添加文章的标题，作者，封面图、文章简介、原文链接和原创标识。
3. 选中全文，格式选择”两端对齐“。
4. 检查一下文章排版格式，确认无误后即可发表。
