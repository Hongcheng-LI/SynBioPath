
**准备蛋白和小分子结构文件**

\1. **蛋白**

1.1 **蛋白下载**

` `[如何从PDB数据库选择蛋白结构](如何从PDB数据库选择蛋白结构.md)

1.2 **将蛋白导入 AutoDock vina**

1. 导入蛋白结构。点击**菜单窗口**中的**File** -> **read molecular**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.001.png)

2. 在弹出的窗口中选中 **hsg1.pdb** 文件，并选择打开。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

3. 打开后，界面如图所示。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)

1.3 **对蛋白结构进行预处理**

1.3.1 **除水**

1. 点击**菜单窗口**中的**Edit** -> **delete water**，即可完成除水操作。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

1.3.2 **加氢**

1. 点击**菜单窗口**中的**Edit** -> **Hydrogens** -> **add**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

2. 在弹出的窗口中选择 **all Hydrogens**，即可完成加氢操作。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

1.3.3 **将蛋白设为受体**

1. 点击**菜单窗口**中的**Grid** -> **Macromolecular** -> **choose**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)

2. 在弹出的窗口中选中蛋白结构文件 **hsg1.pdb**，再点击 **select molecular**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

3. 点击 ok，对蛋白进行初始化，即可将蛋白设为受体。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

4. 此时，会在默认工作目录中生成一个名为 **hsg1.pdbqt** 的 pdbqt 格式的文件，此时选择保存即可。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

5. 在左侧路径中，鼠标右键点击 **hsg1** 文件，点击 **delete**。在当前窗口中删除该文件。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

\2. **小分子**

2.1 **PubChem 下载化合物结构**

1. 点击链接，进入 PubChem 官网：<https://pubchem.ncbi.nlm.nih.gov/>

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.004.png)

2. 以 emodin 为例，搜索之后下载其 sdf 格式的 3D 结构。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.005.png)

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.006.png)

2.2 **Chemdraw 处理**

1. 用 chemdraw 打开下载好的 sdf 文件

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.007.png)

2. 将该结构复制到 Chemdraw3D 中

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.008.png)

3. 能量优化，点击 **Calculation** --> **MMF94** --> **Perform MMF94 minimization**，点击 **run**，执行能量最小化。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.009.png)

4. 保存为 PDB 格式

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.009.png)

2.3 **将小分子导入 AutoDock vina**

1. 点击**菜单窗口**中的**File** -> **read molecular**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.001.png)

2. 在弹出的窗口中选中 **emodin.pdb** 文件，并选择打开。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.010.png)

3. 打开后，界面如图所示。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.009.png)

2.4 **对小分子进行预处理**

2.4.1 **加氢**

1. 点击**菜单窗口**中的**Edit** -> **Hydrogens** -> **add**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.010.png)

2. 在弹出的窗口中选择 **all Hydrogens**，即可完成加氢操作。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.008.png)

2.4.2 **将小分子设为配体**

1. 点击**菜单窗口**中的**Ligand** -> **Input** -> **choose**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

2. 在弹出的窗口中选中小分子结构文件 **emodin.pdb**，再点击 **select molecule for AutoDock4**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)

3. 点击 ok，对小分子结构进行初始化，即可将小分子设为配体。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)

2.4.3 **检测扭转中心及扭转键**

1. 点击**菜单窗口**中的**Ligand** -> **Torsion tree** -> **choose root**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

2. 点击**菜单窗口**中的**Ligand** -> **Torsion tree** -> **choose torsion**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)

3. 此时，化合物变成了红绿两种颜色，红色的键代表不可旋转，绿色的键代表可以旋转。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

4. 在弹出的窗口中显示，共有 14 根可以旋转的键，随后点击 **done**，即可完成操作。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)

2.4.4 **导出为 pdbqt 文件**

1. 点击**菜单窗口**中的**Ligand** -> **Output** -> **Save as PDBQT**。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.001.png)

2. 此时，会在默认工作目录中生成一个名为 **ind.pdbqt** 的 pdbqt 格式的文件，此时选择保存即可。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.002.png)

3. 在左侧路径中，鼠标右键点击 **emodin** 文件，点击 **delete**。在当前窗口中删除该文件。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\准备蛋白和小分子结构文件.003.png)
