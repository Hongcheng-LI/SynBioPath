真菌萜类化合物的生物合成起源于简单的五碳前体：异戊烯基焦磷酸（IPP）和二甲基烯丙基焦磷酸（DMAPP）。在丝状真菌中，这些前体几乎完全通过甲羟戊酸（MVA）途径合成，这一过程发生在胞质溶胶和特定的细胞器（如过氧化物酶体）中。相较于原核生物（如大肠杆菌）或简单的单细胞真核生物（如酿酒酵母），丝状真菌在合成复杂萜类（尤其是倍半萜、二萜和三萜）方面具有天然的生理优势。

## 1. 倍半萜底盘细胞：灰葡萄孢菌（Botrytis cinerea）

灰葡萄孢菌（_Botrytis cinerea_）作为广谱寄生性植物病原真菌，具备高产脱落酸（Abscisic Acid, ABA）的优势，是倍半萜类化合物合成的理想潜在底盘细胞。

### 1.1 脱落酸的多维生物学功能与全球市场需求
脱落酸作为一种在自然界中广泛分布的倍半萜类化合物，最初被确定为调控植物休眠、脱落及应激反应的核心植物激素。随着研究的深入，学术界对其功能定位已从单一的“压力激素”演变为跨物种的“通用信号分子”。在高等植物中，脱落酸在控制种子成熟、抑制萌发、调节气孔开闭以及增强植物对干旱、盐渍、低温等非生物胁迫的耐受性方面发挥着不可替代的作用。此外，ABA在调控果实发育、促进着色及提高作物营养品质方面的应用，使其成为现代精准农业中需求量巨大的植物生长调节剂。

近年来，脱落酸在医学领域的潜力逐渐显现，进一步推高了其市场价值。研究表明，ABA能够调节哺乳动物体内的血糖稳态，改善因高血糖诱导的胰岛素抵抗，并表现出显著的抗炎活性。在免疫调节方面，ABA能够激活先天免疫细胞，诱导内源性细胞因子活性，甚至在抑制多种癌细胞增殖方面展现出初步的药理前景。鉴于其在农业抗逆增产和人类健康维护中的双重角色，如何实现天然、高纯度脱落酸的大规模廉价供应已成为生物工程领域的关键课题。

在生产工艺层面，虽然植物提取法曾是早期ABA的主要来源，但由于植物组织中ABA含量极低（如200公斤棉籽仅含9毫克），且提取工艺繁琐，早已无法满足市场需求。化学合成法虽能实现规模化，但产生的产物通常是具有生物活性的 (S)-(+)-ABA 与低活性的 (R)-(-)-ABA 的外消旋混合物，不仅生理效能大打折扣，且涉及复杂的催化步骤和潜在的环境污染风险。相比之下，利用微生物发酵法生产天然型 (S)-(+)-ABA 具有原料可再生、生产条件温和、环境友好以及产物构型单一（纯天然构型）等核心优势。

在众多的ABA产生菌中，灰葡萄孢菌（*Botrytis cinerea*）脱颖而出。自1982年首次发现该真菌具备合成ABA的能力以来，科研人员通过长达四十年的努力，已将其从一种臭名昭著的植物病原真菌转化为生产ABA的工业明星菌株 [1]。目前，灰葡萄孢菌已成为全球天然脱落酸工业化发酵的首选底盘，其产率和发酵稳定性远超圆弧青霉（*Penicillium cyclopium*）或某些链格孢属（*Alternaria*）真菌。

### 1.2 灰葡萄孢菌中脱落酸的直接生物合成途径

灰葡萄孢菌合成脱落酸的机制与高等植物存在显著差异，这种差异源于两者在进化过程中采取了不同的代谢流策略。植物通过高等类胡萝卜素（C40）的氧化裂解途径间接生成ABA，而灰葡萄孢菌则采用直接合成路径，即通过倍半萜前体法尼基焦磷酸（Farnesyl Diphosphate, FPP, C15）经过一系列氧化还原反应直接形成ABA。


#### 1.2.1 甲羟戊酸（MVA）途径与前体分配

脱落酸的合成始于乙酰辅酶A（Acetyl-CoA），通过甲羟戊酸（MVA）途径生成异戊烯基焦磷酸（IPP）和二甲基烯丙基焦磷酸（DMAPP）。在灰葡萄孢菌的初级代谢中，3-羟基-3-甲基戊二酰辅酶A还原酶（HMGR）被确认为是整个类萜合成路径中的关键限速步骤。

FPP作为ABA合成的直接前体，处于真菌萜类代谢的分叉点上。FPP不仅流向ABA的合成，还参与甾醇（如麦角甾醇）的合成以及其他倍半萜类植物毒素（如botrydial）的生成。因此，提高ABA产量的核心挑战之一在于如何精准调控FPP在这些竞争途径之间的分配，最大限度地引导碳流进入ABA合成支路。

#### 1.2.2 核心合成基因簇：*bcaba1-4*
灰葡萄孢菌ABA合成的关键酶由一个相对紧凑的基因簇编码，通常被称为 _bcaba_ 基因簇。该簇包含四个核心基因：_bcaba1_、_bcaba2_、_bcaba3_ 和 _bcaba4_。

| 基因名称     | 编码酶类型           | 生化功能              | 反应底物与产物                                   |
| -------- | --------------- | ----------------- | ----------------------------------------- |
| _bcaba1_ | 细胞色素P450单加氧酶    | C-1' 和 C-4' 位的羟基化 | α-离香椎烷 → 1',4'-二羟基-α-离香椎烷 [1, 9]          |
| _bcaba2_ | 细胞色素P450单加氧酶    | 进一步氧化及侧链修饰        | 1',4'-二羟基-α-离香椎烷 → 1',4'-反式-ABA-二醇 [3, 7] |
| _bcaba3_ | 潜在的倍半萜环化酶/合酶    | 催化FPP环化           | FPP → (2Z,4E)-α-离香椎烷 [7, 10]              |
| _bcaba4_ | 短链脱氢酶/还原酶 (SDR) | 最后一步脱氢反应          | 1',4'-反式-ABA-二醇 → 脱落酸 (ABA) [3, 9]        |
尽管 _bcaba3_ 位于该基因簇内且在很长一段时间内被视为环化酶，但后续的研究对其实际功能提出了质疑。部分异源表达实验显示，单独表达 _bcaba1-4_ 确实足以让某些宿主产生ABA，但 _bcaba3_ 的序列特征与典型的真菌倍半萜环化酶并不完全一致。

#### 1.2..3 关键环化酶 Bcstc5 (Bcaba5) 的发现

为了寻找催化FPP环化生成ABA骨架的关键酶，研究人员对ABA高产株 ATCC58025 进行了基因组测序。研究发现，除了 _bcaba1-4_ 簇之外，另一个位于不同染色体位置的基因 _bcstc5_（亦被称为 _bcaba5_）编码一种典型的倍半萜环化酶，其表达谱与ABA的积累高度同步。敲除 _bcstc5_ 后，灰葡萄孢菌完全丧失了合成ABA的能力，而补回该基因后产率恢复。

这一发现揭示了灰葡萄孢菌ABA合成路径的一个独特特征：起始环化酶基因与后续的修饰酶基因在空间上是分离的，而非像大多数真菌次级代谢产物（如黄曲霉毒素或青霉素）那样完全聚集成簇。这种分离可能意味着灰葡萄孢菌在进化过程中，通过招募位于不同位点的环化酶进入已有的氧化修饰通路，形成了产生ABA的能力。
