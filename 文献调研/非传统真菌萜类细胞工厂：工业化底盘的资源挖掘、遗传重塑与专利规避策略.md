
在合成生物学驱动的现代生物制造中，真菌萜类化合物因其在医药、农业、食品和化妆品领域的广泛应用而备受关注。萜类化合物作为天然产物中多样性最高的一类，其分子结构的复杂性（如高度氧化修饰、多环骨架及手性中心）对生物合成底盘提出了极高的要求 [1, 2]。长期以来，米曲霉（_Aspergillus oryzae_）NSAR1和构巢曲霉（_Aspergillus nidulans_）A1145等菌株由于其遗传背景清晰、分泌能力强而成为实验室研究的标配。然而，随着生物制造进入产业化深度竞争阶段，这些模式菌株背后的专利壁垒、知识产权（IP）限制以及在特定工业环境下的适应性瓶颈，促使研究人员转向开发具有自主知识产权的非传统真菌底盘 [1, 3, 4]。

萜类生物合成的代谢逻辑与真菌底盘的独特性

真菌萜类化合物的生物合成起源于简单的五碳前体：异戊烯基焦磷酸（IPP）和二甲基烯丙基焦磷酸（DMAPP）。在丝状真菌中，这些前体几乎完全通过甲羟戊酸（MVA）途径合成，这一过程发生在胞质溶胶和特定的细胞器（如过氧化物酶体）中 [1, 5, 6, 7]。相较于原核生物（如大肠杆菌）或简单的单细胞真核生物（如酿酒酵母），丝状真菌在合成复杂萜类（尤其是倍半萜、二萜和三萜）方面具有天然的生理优势。

这种优势首先体现在丰富的内质网（ER）和复杂的细胞内膜系统上。萜类合成后的衍生化过程，特别是细胞色素 P450 单加氧酶（CYPs）参与的羟基化、环氧化和氧化反应，极度依赖于内质网上的电子传递链。丝状真菌强大的内膜系统能够容纳高密度的膜蛋白表达，并提供必要的辅助因子（如 NADPH）平衡 [1, 8, 9]。其次，真菌基因组中普遍存在的生物合成基因簇（BGCs）为代谢流的精准调控提供了演化基础。许多非传统真菌在长期演化中形成了高效的前体供给流，使其在特定条件下能够积累极高滴度的次级代谢产物 [1, 10, 11]。

萜类前体模块的代谢通量分析

在构建细胞工厂时，前体供给的充足性是决定最终滴度的第一要素。MVA 途径涉及一系列酶促反应，从乙酰-CoA（Acetyl-CoA）出发，通过 HMG-CoA 还原酶（HMGR）等限速步骤。

|关键酶名称|缩写|代谢作用|调节价值|
|---|---|---|---|
|乙酰-CoA 乙酰转移酶|Aat1|缩合两分子乙酰-CoA|提供 MVA 途径起始底物 [12]|
|3-羟基-3-甲基戊二酰-CoA 合酶|HMGS|生成 HMG-CoA|途径早期关键节点|
|3-羟基-3-甲基戊二酰-CoA 还原酶|HMGR|HMG-CoA 还原为甲羟戊酸|整个途径的最主要限速步骤 [5]|
|异戊烯基焦磷酸异构酶|Idi1|IPP 与 DMAPP 互换|维持两种 C5​ 前体比例平衡 [12]|
|法尼基焦磷酸合酶|FPPS|形成 C15​ 链|倍半萜合成的直接前提 [9, 12]|
|香叶基香叶基焦磷酸合酶|GGPPS|形成 C20​ 链|二萜和胡萝卜素的前体 [13, 14]|

极高产倍半萜底盘：灰葡萄孢菌（Botrytis cinerea）

灰葡萄孢菌（_Botrytis cinerea_）是一种广寄生性植物病原真菌，因其对脱落酸（Abscisic Acid, ABA）极高的天然合成能力而成为倍半萜类化合物研究的焦点。ABA 是一种重要的倍半萜类植物激素，在真菌中通过“直接途径”合成，即由 C15​ 前体法尼基焦磷酸（FPP）经过一系列氧化和环化反应直接生成，而非植物中的类胡萝卜素降解途径 [15, 16]。

产率突破与高产菌株演化

_B. cinerea_ 的 ABA 生产已被证明具有极高的工业价值。从原始分离株 TBC-6 出发，通过多次物理和化学诱变（如 ARTP、紫外线、EMS），研究人员已筛选出具有极高生产力的突变株。

| 菌株名称       | ABA 产量 (g/L) | 关键特征         | 来源       |
| ---------- | ------------ | ------------ | -------- |
| TBC-6 (WT) | 极低           | 原始分离株，作为背景研究 | [15]     |
| TB-31      | 0.55         | 早期诱变筛选株      | [15, 17] |
| TB-3-H8    | 1.4          | 高产量突变体       | [15]     |
| TBC-A      | 2.0          | 已投入工业化应用的强力株 | [15]     |

这种产量的跨越式提升表明，_B. cinerea_ 的 MVA 途径能够承受极高的代谢载荷，并能高效地将碳源导向 FPP 前体。此外，TB-31 等菌株已被证明是研究倍半萜生物合成调控的理想平台。通过对 TB-31 进行转录组分析，研究者锁定了甲基转移酶基因 _Bclae1_，它是 ABA 合成的全局调节因子。敲除 _Bclae1_ 导致 ABA 产量下降 95%，同时伴随着整个 _bcaba1-4_ 基因簇转录水平的崩溃，这为后续通过调节基因（Regulatory genes）改造底盘提供了理论支撑 [17]。

遗传工具的革新：CRISPR/Cas9 与 RNP 系统

对于 _B. cinerea_ 这种多核且细胞壁坚硬的丝状真菌，传统的同源重组效率极低且容易出现回复突变。为了将其改造为万能底盘，研究人员建立了基于核糖核蛋白（RNP）复合物的 CRISPR/Cas9 编辑系统 [9]。该系统的优势在于：首先，Cas9 蛋白和 sgRNA 在体外预组装，避免了 Cas9 基因在体内持续表达带来的细胞毒性；其次，RNP 介导的编辑能够实现精准的多基因敲除和片段缺失 [9]。

在最新的底盘构建尝试中，研究人员删除了整个 ABA 合成基因簇（Δbcaba1234）。这一举措不仅消除了内源 ABA 对异源产物的潜在污染，更重要的是实现了“代谢流重排”。实验数据显示，在 ABA 合成路径被切断后，胞内的 FPP 库显著扩大，为异源表达倍半萜合酶（如广藿香醇合酶、青蒿二烯合酶）提供了极其充裕的前体支持 [9]。

工业化二萜中心：藤仓赤霉（Fusarium fujikuroi）

藤仓赤霉（_Fusarium fujikuroi_，曾名 _Gibberella fujikuroi_）在工业微生物领域拥有超过 100 年的应用历史，主要用于生产赤霉素（Gibberellins, GAs）。赤霉素是一类重要的四环二萜类植物生长调节剂 [18, 19, 20]。

二萜合成的天然优势

_F. fujikuroi_ 的基因组展现了极其丰富的萜类代谢潜力。它拥有至少 8 个萜类环化酶（TCs）和 1 个预苯基转移酶（PT）[14]。其赤霉素生物合成基因簇是真菌中研究最透彻的 BGC 之一，其核心步骤涉及双功能萜类合酶 ent−copalyl diphosphate/ent−kaurene synthase (CPS/KS)，该酶能高效地将 GGPP 转化为 _ent_-kaurene [14, 21]。

从底盘开发的角度看，_F. fujikuroi_ 的工业株系经过数十年的驯化，已具备在高密度发酵条件下持续分泌二萜化合物的能力。在 5-L 生物反应器中，通过 ARTP-EMS 联合诱变及洛伐他汀（Lovastatin）抗性筛选，其 GA3 滴度可达 2.48 g/L，而在固态发酵条件下甚至能达到 6.5 g/kg [18]。这种高滴度的实现主要归功于其优化的碳氮源利用效率和对高浓度萜类产物的自我耐受机制。

遗传工程的深度定制

为了提高 _F. fujikuroi_ 作为底盘的灵活性，研究人员利用 CRISPR/Cas9 系统对其进行了多位点改造。通过筛选高效的内源启动子，如 5S rRNA 和 U6 启动子，sgRNA 的表达效率得到了极大提升，基因编辑准确率达到 79.2% [20, 22]。

在具体的路径工程中，研究者已成功实现了产物谱的受控切换。通过精准控制赤霉素合成簇中的氧化步骤，能够将原本积累 GA3 的菌株改造为专门生产 GA4 和 GA7 混合物的细胞工厂，这些特定组分在水果保鲜和园艺领域具有更高的商业价值 [22]。此外，该菌株还具有生产二萜类色素（如神经孢黄素）的能力，进一步证明了其二萜骨架合成的多样性 [14, 20]。

接合菌门的产萜奇迹：三孢布拉氏霉（Blakeslea trispora）

三孢布拉氏霉（_Blakeslea trispora_）是目前工业生产 β-胡萝卜素（β-carotene）和番茄红素（lycopene）的绝对核心。作为一种接合菌，它在代谢调节和生长形态上与子囊菌存在显著差异 [23, 24, 25]。

性诱导与代谢放大机制

_B. trispora_ 最具工业吸引力的特性是其“性诱导”合成机制。当 (+) 型和 (-) 型菌株在发酵罐中接触时，会通过三孢酸（trisporic acids）信号分子激活类胡萝卜素的合成，其产量比单菌培养提高 5-15 倍 [13, 25]。这种天然的信号响应机制可以被视为一种受控的诱导型强启动子系统。

| 核心基因    | 酶功能                 | 突变体改良效果                             |
| ------- | ------------------- | ----------------------------------- |
| _carB_  | 八氢番茄红素脱氢酶           | 在高产株 F-744 中发现 S528R 突变，增强脱氢效率 [13] |
| _carRA_ | 双功能番茄红素环化酶/八氢番茄红素合酶 | P143S 突变显著提升了 GGPP 向番茄红素的转化率 [13]   |

现代分子育种与进化工程

虽然 _B. trispora_ 曾因缺乏高效转化系统而被称为“难以工程化的菌株”，但最新的进展正在改变这一局面。基于转录组学筛选出的强启动子库以及 RNA 干扰（RNAi）技术的成熟，使得精准调节类胡萝卜素流向成为可能 [23]。

更为显著的成果来自于自适应实验室进化（ALE）。通过长达 16 个月的连续传代，并在培养基中加入乙酰乙酰苯胺（Acetoacetanilide）作为选择压力，进化后的菌株 A278 在 β-胡萝卜素产量上提升了 45%，达到 54 mg/L。ALE 过程中，关键基因 _hmgR_、_carRA_ 和 _SR5AL_ 的表达显著上调，且细胞形态发生了利于产物积累的改变 [25, 26]。这表明，即使在遗传操作工具尚不完善的情况下，通过进化工程也能有效提升非传统底盘的性能。

担子菌门的亲缘底盘：玉米黑粉菌（Ustilago maydis）

玉米黑粉菌（_Ustilago maydis_）虽然是植物病原菌，但在生物工程界被视为一种具有高度潜力的单细胞真菌底盘（酵母态生长）。作为担子菌门（Basidiomycota）的代表，它与生产复杂萜类药物的高等真菌（如灵芝、冬虫夏草）亲缘关系极近 [6, 12, 27, 28]。

与担子菌来源酶的优越相容性

许多来源于蘑菇（高等担子菌）的萜类合酶在子囊菌（如酵母、米曲霉）中表达时，常面临折叠错误、活性低或产物毒性等问题。_U. maydis_ 凭借其担子菌属性，展现了极佳的代谢相容性 [6, 27, 29]。

在实验测试中，研究人员在 _U. maydis_ 中异源表达了来源于 _Coprinopsis cinerea_ 的倍半萜合酶 Cop6。结果显示，其产物 α-cuprenene 的滴度比在同一系统中表达植物来源的 (+)-valencene 合酶高出 10 倍以上 [6, 12, 29]。这一发现为从高等真菌中挖掘和高效生产具有抗癌、抗炎活性的稀有萜类提供了关键底盘支持。

代谢工程工具箱与无标记技术

_U. maydis_ 已拥有一套成熟的遗传操作工具，包括 CRISPR/Cas9 和 FLP/FRT 系统。研究者已利用这些工具构建了“无标记”底盘株。通过删除产生衣康酸、黑粉菌脂和三酰甘油等副产物的内源基因簇，碳流被集中导向 MVA 途径 [30, 31]。此外，利用其独特的非典型蛋白分泌途径（Cts1），可以将异源表达的萜类合酶或修饰酶锚定在细胞表面或分泌到胞外，从而缓解胞内代谢负荷并优化反应动力学 [27, 32]。

药用真菌的特化底盘：灵芝与虫草

灵芝（_Ganoderma lucidum_）和蛹虫草（_Cordyceps militaris_）是天然的三萜和核苷酸类代谢物仓库。特别是灵芝产生的灵芝酸（Ganoderic Acids），是一类高度氧化的兰烷型四环三萜，其复杂的合成路径目前尚无法在模式生物中完全重构 [33, 34, 35]。

灵芝：高产突变与代谢组优势

由于灵芝三萜具有显著的抗肿瘤和免疫调节活性，提高其滴度是工业界的核心诉求。通过现代育种手段，如对 GL_V1 菌株进行诱变，获得的 GL_V2 菌株展现了显著的表型优势：

|指标|GL_V1 (常规株)|GL_V2 (改良株)|变化|
|---|---|---|---|
|总三萜含量 (mg/g)|5.89|8.24|+40% [36]|
|蛋白质含量 (mg/g)|155.00|278.91|+80% [36]|
|DPPH 清除率 (%)|31.12|39.25|+26% [36]|

代谢组分析显示，GL_V2 菌株不仅总量提升，还特异性地增加了 7 种灵芝酸和 2 种灵芝醇的水平 [36]。对于这种生长周期较长的菌株，通过原生质体育种和液体深层发酵（LSF）技术的优化，是目前实现三萜类化合物工业化生产的最可行途径 [34, 37]。

蛹虫草：遗传编辑的新宠

蛹虫草（_C. militaris_）是少数几种基因组已完全解析且易于人工栽培的药用真菌。其 CRISPR/Cas9 系统已实现常态化应用，能够精准编辑影响次级代谢、生长发育和逆境耐受的基因 [38, 39]。对于想要构建基于药用真菌背景的萜类底盘的研究者，蛹虫草提供了一个平衡了“天然合成潜力”与“遗传操作便捷性”的切入点。

底盘开发中的专利规避与 FTO 策略

在规避米曲霉 NSAR1 和构巢曲霉 A1145 的版权/专利问题时，必须进行系统的自由实施（Freedom to Operate, FTO）分析。FTO 分析的核心不是证明自己拥有专利，而是确认生产过程中所使用的菌株、启动子、标记基因和代谢路径不侵犯第三方的有效专利权 [40, 41, 42]。

菌株来源的合法化

使用来自实验室自主隔离、具有清晰历史记录的野生型株系（如 _B. cinerea_ TBC-6）作为起始点，是绕过专有底盘（Proprietary chassis）的最有效方法 [9]。许多由 Novozymes 或 DSM 等巨头开发的工业株系（如特定高分泌 _A. niger_ 或 _A. oryzae_ 支系）包含大量的专有基因组缺失和增强子，擅自使用这些遗传背景会带来严重的 IP 风险 [3, 43, 44]。

遗传元件的“去通用化”

为了确立技术主权，应尽量开发和使用特定菌株的内源元件。

|元件类型|规避建议|技术理由|
|---|---|---|
|启动子|使用内源强启动子（如 _B. cinerea_ 的 _Pef1a_）[8]|避免使用被大范围授权的 _Aspergillus_ 来源启动子 [45]|
|选择标记|采用 CRISPR 介导的 Marker-free 系统 [31, 46]|消除对特定抗性基因专利的依赖|
|核定位信号|筛选内源 H2B 来源的 NLS [22]|提高入核效率的同时确立专利新颖性|
|分泌信号|挖掘如 _U. maydis_ 的 Cts1 非典型分泌信号 [32]|建立独特的分泌底盘体系|

FTO 风险管理流程

在确定了 _B. cinerea_ 或 _F. fujikuroi_ 为目标底盘后，研发团队应在 NDA 或 BLA 申请前完成以下工作：首先，通过 AI 增强的专利检索工具（如 STN、CAS IP Finder）绘制专利地图 [47]；其次，对高风险专利进行权利要求映射（Claim Mapping），寻找设计规避（Design Around）的空间 [40, 41]；最后，如果发现某些关键路径（如通用的 MVA 强化策略）已被覆盖，可考虑通过自适应实验室进化（ALE）获取等效甚至更优的表型，因为 ALE 产生的自然变异通常不属于受保护的“重组遗传操作”范畴 [26, 48]。

综合评估与选择指南

根据本研究对非传统真菌底盘的详尽调查，以下为不同萜类生产需求的推荐路径：

1. **倍半萜类（如香料、抗疟药）**：首选 **灰葡萄孢菌（****B. cinerea****）**。其 ABA 生产背景证明了其极高的 FPP 代谢流量。通过 CRISPR/RNP 系统删除内源 ABA 簇，可获得一个纯净且高效的倍半萜生产平台 [9]。
2. **二萜类（如赤霉素、二萜类色素）**：首选 **藤仓赤霉（****F. fujikuroi****）**。其二萜代谢网络极其发达，且遗传编辑工具已高度优化（5S rRNA 驱动系统），适合进行复杂的路径重构 [14, 18, 22]。
3. **四萜及胡萝卜素类**：首选 **三孢布拉氏霉（****B. trispora****）**。利用其性诱导机制和 ALE 进化技术，可以实现极高的碳源向 C40​ 骨架的转化率 [21, 25, 26]。
4. **来自担子菌（蘑菇）的稀有萜类**：首选 **玉米黑粉菌（****U. maydis****）**。其跨门类的代谢兼容性是解决担子菌合酶在异源系统中失活的关键 [6, 12, 27]。
5. **高度氧化的三萜类（如药用三萜）**：考虑 **灵芝（****G. lucidum****）** 的深层发酵或 **蛹虫草（****C. militaris****）** 的定向改造，充分利用其内源的 CYPs 和修饰酶库 [34, 36]。

通过构建这些非传统真菌细胞工厂，不仅可以有效避开米曲霉、构巢曲霉等模式菌株的专利陷阱，还能利用这些菌株在千万年演化中形成的独特代谢优势，实现萜类化合物的高效、可持续和低成本生物制造。未来的研究重点将集中在这些底盘的全局调控因子挖掘、细胞器区室化工程以及在大规模连续发酵中的遗传稳定性优化上。通过“底盘-重构-区室化”三位一体的开发逻辑，非传统真菌必将在萜类生物制造的工业版图中占据主导地位 [1, 4]。
