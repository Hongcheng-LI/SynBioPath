# 1. 非核糖体多肽生物合成研究要点

2. **研究起始**：20 世纪 60 年代早期，核糖体密码破译后，研究人员开始探究芽孢杆菌中含 D 型氨基酸环肽的合成方式。
3. **早期假说**：Tatum 等人发现 tyrocidine 合成不受核糖体抑制剂金霉素影响，提出其合成不依赖核糖体蛋白的假说（1963 年）。
4. **机制研究**：Fritz Lipmann 和 Søren Laland 进一步阐明 NRPS（非核糖体多肽合成酶）的合成机制：通过 ATP 激活特定氨基酸、4'- 磷酸泛硫氨酸（Ppant）结合激活的氨基酸，确定多肽合成方向，开启相关研究。
5. **成熟合成过程**：非核糖体肽由非核糖体肽合成酶合成，先利用合成砌块产生稀有官能团，再经修饰酶（酰化、烷基化、糖基化、氧化、加氧等修饰）加工，最终形成结构稳定、具生物活性的多肽骨架。

# 2. 非核糖体肽合成酶
非核糖体多肽类天然产物的核心骨架由非核糖体多肽合成酶（Nonribosomal peptide synthetase, NRPS）合成。NRPS 由一个或多个模块构成，每个模块一般包含 3 个基本结构功能域：
- 腺苷化结构域（Adenylation domain，A）
- 硫酯化结构域（Thilation domain，T）
- 缩合结构域（Condensation domain，C）

此外，模块还可能包含修饰结构域，如甲基化结构域（Methylation，MT）、氧化结构域（Oxidation，Ox）、杂环化结构域（Heterocyclization，Cy）、异构化结构域（Epimerization，E）等。

NRPS 的 C 末端通常含有硫酯酶（Thioesterase，TE）或末端缩合（Terminal condensation，Cₜ）结构域，负责终产物的合成收尾。
