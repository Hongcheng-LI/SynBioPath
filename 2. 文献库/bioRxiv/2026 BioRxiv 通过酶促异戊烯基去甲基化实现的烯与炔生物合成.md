# 二、研究背景

烯与炔是天然产物中两类具有高反应性的官能团，在合成化学、药物化学与生物正交化学中具有广泛应用。叠氮-炔烃 1,3-偶极环加成（CuAAC 点击化学）以及近年来发展的烯烃参与的双膦-铜催化点击反应，均凸显了这些官能团在化学生物学中的核心地位。自然界中已知超过 **2000 种** 含炔天然产物和 **150 余种** 含烯天然产物，代表性分子包括珊瑚来源的 norditerpenoid ginamallene、真菌来源的环肽 pseudoxylallemycin B（含 **O-homoallenyl-L-Tyr**，即化合物 **1**）以及 sinuxylamide B（含 **O-but-2-ynyl-L-Tyr**，即化合物 **6**）。尽管烯与炔广泛存在，其生物合成机制却长期未被阐明。迄今为止，唯一报道的烯形成酶是藻类 violaxanthin de-epoxidase 类似酶 VDL1，通过 1,2-消除反应催化 violaxanthin 向 neoxanthin 的转化；炔形成酶则有少数几例报道，包括叁键形成的 diiron 蛋白与 P450 酶。上述数据表明，自然界中仍存在大量未被发现的烯与炔形成酶。

作者从 **CYP51（羊毛甾醇 14α-去甲基酶）** 催化的氧化去甲基化反应中获得灵感：CYP51 通过氧化性 C(sp³)-C 键断裂，将一个完整的甲基从 sp³ 杂化碳上切除，从而在 C14-C15 位引入双键（**图 1c**）。基于此，作者提出如下假说：对 **异戊烯基 (prenyl group)** 上 sp² 杂化碳的甲基进行类似的氧化去甲基化，可分别形成烯或炔（**图 1b**）。具体而言，四电子氧化 prenyl 的末端甲基生成醛或偕二醇中间体，随后发生氧化性 C-C 键均裂，产生乙烯基自由基中间体；该自由基后续的 **氢原子攫取 (HAT)** 位点决定了产物类型——从末端 sp³ 碳上攫氢生成 **末端烯**，从 sp² 乙烯基碳上攫氢生成 **内部炔**（**图 1d**）。该假说若被证实，将揭示一条此前未被报道的、由 P450 酶催化的烯与炔形成的新途径。

# 三、研究思路

本研究以 **P450 催化的氧化去甲基化** 为核心假说，从真菌天然产物 penipratynolene (**5**) 与 sinuxylamide B (**2**) 的生物合成途径入手，系统性地鉴定了负责烯与炔形成的 P450 酶，并通过密度泛函理论计算、异源表达、体内饲喂与体外微体酶活实验，结合氘代底物标记与定点突变，阐明了其催化机制。

首先，作者通过 DFT 计算验证了从 prenyl 底物经乙烯基自由基中间体形成烯与炔的热力学与动力学可行性。随后，以 *Penicillium polonicum* 中 penipratynolene (**5**) 的生物合成基因簇为对象，通过异源表达（在 *Aspergillus nidulans* ΔEMΔST 中）鉴定出 **PpnA**（ABBA 型异戊烯基转移酶，催化 L-酪氨酸的 O-异戊烯基化生成 **7**）与 **PpnB**（P450 酶，催化 **7** 转化为 O-homoallenyl-L-Tyr **1**，即末端烯产物）。同时，通过同源搜索与系统发育分析，从 *Nemania serpens* 的 **nse** 簇中鉴定出 PpnB 的同源酶 **NseB**，后者催化 **7** 转化为 O-but-2-ynyl-L-Tyr **6**（内部炔产物）。为进一步解析该转化，作者合成了一系列氘代底物（**4,4,4-[d₃]-7**、**5,5,5-[d₃]-7**、**2-[d₁]-7** 与 **1,1-[d₂]-7**），通过 LC-QTOF 分析产物中的氘保留情况，确定了去甲基化发生的区域选择性与第二 HAT 步骤的位点。此外，作者还鉴定了 **PpnD**（同一途径中的第二个 P450 酶），其催化末端烯 **10** 氧化异构化为 **α-羟基炔 14**，为自然界中罕见的“烯-炔异构酶”。最后，通过 AlphaFold3 预测结构比对与定点突变实验，初步探究了 PpnB 与 NseB 产物选择性差异的结构基础。

# 四、研究方法

- **密度泛函理论计算**：使用 Gaussian 16 软件，在 B3LYP-D3(BJ)/def2-TZVP/SMD(Et₂O)//B3LYP-D3(BJ)/def2-SVP/IEFPCM(Et₂O) 水平上计算反应能垒与中间体稳定性。过渡态通过内禀反应坐标验证。

- **基因组挖掘与系统发育分析**：以 PpnB 为 query 序列，在 NCBI nr 数据库中进行 BLASTp 搜索，获取 top 100 同源序列。使用 ClustalW 进行多序列比对，采用 IQ-TREE 网络服务器以最大似然法构建系统发育树（1000 次 ultrafast bootstrap 重复）。

- **异源表达**：将 **ppnA**、**ppnB**、**ppnC**、**ppnD**、**ppnE** 以及 **nseA**、**nseB**、**nseC** 等基因克隆至适合 *A. nidulans* 的表达载体，通过原生质体转化法导入 *A. nidulans* ΔEMΔST 宿主。转化株在 CD-ST 固体培养基（补充核黄素、尿嘧啶和/或吡哆醇）上于 28℃ 培养 3-5 天，菌体经丙酮提取后用于 LC-MS 分析。

- **化合物分离与结构鉴定**：采用 CombiFlash 反相 flash 色谱进行初分，半制备型 HPLC（COSMOSIL 5C18-AR-II 柱，4 mL/min）纯化目标化合物。NMR 谱在 Bruker AV500 谱仪上采集（¹H 500 MHz，¹³C 125 MHz）；高分辨质谱在 Agilent 6545 QTOF 上记录。

- **微体制备与体外酶活测定**：从 *A. nidulans* 转化株菌丝体中提取微体组分。反应体系含微体、100 μmol/L 底物（**7**、**17** 或 **18**）与 2 mmol/L NADPH，于 22℃ 孵育 16 小时后以甲醇淬灭，LC-QTOF 分析产物。

- **氘代底物合成与同位素标记实验**：化学合成了 **4,4,4-[d₃]-7**、**5,5,5-[d₃]-7**、**2-[d₁]-7** 与 **1,1-[d₂]-7**。将上述底物分别饲喂 *A. nidulans* 表达株或加入微体反应体系，通过 LC-QTOF 测定产物分子量变化，推断去甲基化发生的碳原子及第二 HAT 攫氢位点。

- **蛋白表达与纯化**：将 **ppnA** 与 **ppnC** 的 ORF 克隆至 pET28a 载体，在 *E. coli* BL21(DE3) 中于 16℃ 诱导表达（0.1 mmol/L IPTG），通过 Ni-NTA 亲和层析纯化 His6-标签融合蛋白。

- **定点突变**：通过重叠延伸 PCR 在 NseB 中引入 I126V/F379L 双突变，在 PpnB 中引入 V127I/L380F 双突变。突变体在 *A. nidulans* 中表达后，比较产物 **1** 与 **6** 的比例变化。

# 五、实验设计及结果分析

### (一) 理论验证：DFT 计算支持 prenyl 氧化去甲基化生成烯与炔的可行性

#### 实验目的与设计逻辑

为评估 prenyl 氧化去甲基化生成烯或炔的化学可行性，作者以 **O-prenyl phenol** 为模型底物，采用 DFT 计算分别考察了 Cpd I（高价铁-氧物种）起始于偕二醇的途径（**图 2a**）与 Cpd 0（Fe(III)-过氧物种）起始于醛的途径（**图 2b**）。重点计算了 C-C 键断裂步骤的能垒、乙烯基自由基中间体的稳定性以及第二 HAT 步骤的势能面特征。

#### 实验结果与机理解析

**图 2a** 展示了 Cpd I 依赖途径的势能面。偕二醇底物与 Cpd I 结合形成 **Int1** 后，Cpd I 从偕二醇的一个羟基上攫取氢原子，通过 **TS1（ΔG‡ = 11.7 kcal/mol）** 生成偕二醇自由基 **Int2** 与 Cpd II。从 **Int2** 发生 C-C 键均裂形成乙烯基自由基 **Int3** 并释放甲酸的步骤（**TS2，ΔG‡ = 8.1 kcal/mol**）为放能过程（ΔG° = **-11.0 kcal/mol**）。**图 2b** 展示了 Cpd 0 依赖途径：Cpd 0 的过氧氧对醛的亲核加成经 **TS3（ΔG‡ = 14.8 kcal/mol）** 生成 Criegee 中间体 **Int5**；O-O 键均裂经 **TS4（ΔG‡ = 15.4 kcal/mol）** 同样生成 **Int2** 与 Fe(III)-O• 自由基。值得注意的是，从乙烯基自由基 **Int3** 出发，无论攫取末端 sp³ 氢（生成烯）还是攫取 sp² 乙烯基氢（生成炔），在势能面上均表现为 **无垒过程 (barrierless)**。上述计算表明：乙烯基自由基中间体在热力学与动力学上均可达；第二 HAT 的区域选择性由底物在 P450 活性口袋中的结合构象决定，而非由 C-H 键解离能差异主导。速率决定步骤为 **TS2** 或 **TS4**（~15 kcal/mol），略高于 CYP51 反应中 Cpd I 依赖途径的能垒（~3.9 kcal/mol），但与 CYP51 的 Cpd 0 依赖途径（17.3 kcal/mol）相当。该计算为后续酶学实验提供了坚实的理论依据。

### (二) **ppn** 簇的鉴定与烯形成酶 PpnB 的功能解析

#### 实验目的与设计逻辑

基于理论计算，作者推测编码异戊烯基转移酶与 P450 酶的基因簇应为烯/炔形成的遗传基础。以 *Penicillium polonicum* 中 penipratynolene (**5**) 的生物合成途径为切入点，作者采用“PT-P450”基因对作为筛选标准，从基因组中鉴定出 **ppn** 簇（**图 3a**）。通过异源表达、体内饲喂与体外酶活实验，作者旨在确定：PpnA 的天然底物与催化特异性；PpnB 是否催化 **7** 向 **1**（末端烯）的转化；以及整个途径中其他酶（PpnC、PpnD）的功能。

#### 实验结果与机理解析

**图 3b** 显示，重组 PpnA 在体外仅能异戊烯基化 **L-酪氨酸** 生成 **O-prenyl-L-Tyr (7)**，而对 4-羟基苯甲酸、对香豆酸等其他底物无活性。同位素饲喂实验（**扩展数据图 2、3**）证实 (¹³C₉,¹⁵N)-L-Tyr 可掺入 **8**，而 4-HBA-d₄ 则不能，进一步确认 **7** 为 PpnA 的天然产物。

**图 3c** 显示，共表达 **ppnA** 与 **ppnB** 的 *A. nidulans* 转化株（补充 L-Tyr 与对香豆酸）中检测到分子量为 233 的化合物 **1**。纯化后的 **1** 的 ¹³C NMR 谱中显示 **δc 209 ppm** 的 sp 杂化烯碳特征信号，确证其为 **O-homoallenyl-L-Tyr**。不补充对香豆酸时，同一转化株中检测到分子量 190 的化合物 **10**（**O-homoallenyl-4-羟基苯甲酸**），表明内源氨裂解酶可将 **1** 脱氨生成 **11**，后者进一步氧化为 **10**。**扩展数据图 4** 的饲喂实验证实：PpnB 特异性地以 **7** 为底物生成 **1**，而 **8** 与 **9** 均不被转化。

**图 3d** 显示，重组 PpnC 可将 **7** 转化为 **9**，将 **1** 转化为 **11**，但对 L-Tyr 与 L-Phe 无活性，表明 PpnC 是 O-烷基化 L-Tyr 的专一性氨裂解酶。**图 3e** 展示了将 **1** 掺入更复杂天然产物的能力：共表达 **ppnAB** 与 **pfaA**（编码 PKS-NRPS，天然产物为 **N-demethylmelearoide A (12)**）的转化株中，检测到 **homoallenyl 类似物 13**，NMR 确认烯官能团的存在。

**图 3f** 与 **图 3g** 展示了途径中的第二个 P450 酶 **PpnD** 的功能：将 **10** 饲喂表达 **ppnD** 的 *A. nidulans* 转化株，生成分子量与光旋值匹配 **desmethylpenipratynolene (14)** 的产物，NMR 确证为 **α-羟基炔**。共表达 **ppnD** 与 **ppnE**（甲基转移酶）并以 **10** 为底物时，生成终产物 **5**（penipratynolene）。此外，表达 **ppnAD** 的转化株中分离出 **15**（3-甲基丁-3-烯-2-醇醚），表明 PpnD 亦可催化 **8** 的末端甲基氧化生成末端烯。

### (三) **nse** 簇的发现与炔形成酶 NseB 的功能解析

#### 实验目的与设计逻辑

PpnB 专一性地生成末端烯（**1**），而 DFT 计算预测通过改变第二 HAT 攫氢位点（从乙烯基碳上攫氢）可生成内部炔（**6**）。为寻找催化该反应的 P450 同源酶，作者以 PpnB 为探针进行 BLASTp 搜索，构建系统发育树，从中筛选可能编码炔形成酶的 BGC。**nse** 簇（来自 *Nemania serpens*）因同时编码 PpnA 同源物（NseA）、PpnB 同源物（NseB，与 PpnB 序列一致性 **89%** ）及 N-酰基转移酶（NseC）而被选中。

#### 实验结果与机理解析

**图 4a** 的系统发育树显示，PpnB 同源物分为三个 clade：Clade I（含 PpnB）与 Clade II（含 NseB、Xte2B 等）进化上密切相关，Clade III 则较远。基因组上下文分析表明，Clade II 成员几乎均与 PpnA 同源物及推定酰基转移酶共编码于同一 BGC 中。

**图 4b** 显示，共表达 **nseAB** 的 *A. nidulans* 转化株中检测到 **三个产物**：主要产物为 **6**（O-but-2-ynyl-L-Tyr，内部炔），次要产物为 **1**（末端烯），以及 **16**（O-but-2-ynyl-4-羟基苯甲酸，**6** 的脱氨/氧化 catabolite）。**6** 的 NMR 表征确认了内部叁键的存在（**表 S5**，**图 S33-S34**）。**扩展数据图 4c** 的饲喂实验确认 **7** 为 NseB 的天然底物。共表达 **nseABC** 时，检测到与 sinuxylamide B (**2**) 完全匹配的产物（含 N-己酰基链），以及痕量的 N-戊酰基（sinuxylamide C）与 N-丁酰基（sinuxylamide D）同系物。此外，还检测到推定的 **1** 的 N-己酰化产物 **2'**，提示 NseC 可接受 **1** 与 **6** 两种底物。

该结果表明，NseB 与 PpnB 催化相同的初始 C(sp²)-去甲基化反应，但第二 HAT 攫氢位点不同：NseB 中 Cpd II（或 Fe(III)-O•）更倾向于从 **C2 乙烯基位置攫氢** 生成内部炔 **6**（为主产物），同时也可从末端甲基攫氢生成 **1**（为副产物）。PpnB 则几乎专一性地从末端甲基攫氢生成 **1**。

### (四) 同位素标记实验揭示去甲基化区域选择性与第二 HAT 机制

#### 实验目的与设计逻辑

为确定 PpnB 与 NseB 催化过程中 **哪一个末端甲基被切除** 以及 **第二 HAT 攫氢位点**，作者合成了四组氘代底物：**4,4,4-[d₃]-7**（C4 位 cis-甲基全氘代）、**5,5,5-[d₃]-7**（C5 位 trans-甲基全氘代）、**2-[d₁]-7**（C2 乙烯基位单氘代）与 **1,1-[d₂]-7**（C1 亚甲基双氘代）。将这些底物分别饲喂表达 PpnB 或 NseB 的 *A. nidulans* 转化株（或加入微体反应体系），通过 LC-QTOF 测定产物分子量变化。

#### 实验结果与机理解析

**图 5a** 显示：以 **4,4,4-[d₃]-7** 为底物时，PpnB 生成的 **1** 无质量增加；以 **5,5,5-[d₃]-7** 为底物时，**1** 呈现 **+2 Da** 质量偏移。**扩展数据图 7** 显示 NseB 的类似实验：以 **4,4,4-[d₃]-7** 为底物时 **6** 无质量变化；以 **5,5,5-[d₃]-7** 为底物时 **6** 呈现 **+3 Da** 质量偏移。上述结果 **明确排除了 cis-甲基 (C4) 被切除的可能性**，证明 **trans-甲基 (C5) 是氧化去甲基化发生的位点**。

质量偏移的幅度进一步证实了第二 HAT 的位点差异：PpnB 从 **C5 末端甲基** 攫取一个氢生成烯，因此产物中保留两个氘（+2 Da）；NseB 不从 C5 甲基上攫氢，三个氘全部保留（+3 Da）。**补充图 20** 显示：以 **2-[d₁]-7** 为底物时，NseB 生成的 **6** 无质量变化（氘丢失），而 PpnB 生成的 **1** 呈现 +1 Da 偏移（氘保留），确证 NseB 从 **C2 乙烯基位** 攫氢生成炔。

**图 5b** 显示，化学合成的 C4 羟基化中间体 **17** 可被 PpnB 与 NseB 转化为相应的产物（**1** 或 **6/1**），而 C5 羟基化中间体 **18** 则不被转化。**补充图 22** 在 PpnB 反应体系中检测到与 **17** 标准品保留时间一致的中间体峰。这些结果支持如下机制（**图 5c**）：PpnB/NseB 首先 **区域选择性地羟化 C4 甲基**，经进一步氧化生成 **C4-醛**（或 **C4-偕二醇**）；随后通过 Cpd I 或 Cpd 0 依赖的途径发生 C3-C4 键均裂，生成乙烯基自由基中间体；第二 HAT 的区域选择性（**PpnB 攫 C5-H vs NseB 攫 C2-H**）决定了产物是烯还是炔。

### (五) 产物选择性的结构基础：活性位点微环境的调控作用

#### 实验目的与设计逻辑

PpnB 与 NseB 具有高度的序列同源性（**89%**），但产物选择性截然不同。为探究这一差异的结构基础，作者采用 AlphaFold3 预测了 PpnB 与 NseB 的三维结构，并通过结构比对鉴定出底物结合口袋与血红素结合口袋中的差异残基。在此基础上，设计了 NseB（I126V/F379L）与 PpnB（V127I/L380F）的互换突变，以测试这些残基对产物比例的影响。

#### 实验结果与机理解析

AlphaFold3 预测结构显示，PpnB 与 NseB 的整体折叠高度相似，底物结合口袋与血红素结合口袋的残基几乎完全相同。唯一的显著差异在于血红素结合口袋中的两个残基：PpnB 的 **V127** 与 **L380**，对应 NseB 的 **I126** 与 **F379**。**补充图 25** 显示，将 NseB 中的 I126V/F379L 双突变引入后，产物中 **1（烯）与 6（炔）的比例增加约 2 倍**（烯产物相对增加）；而 PpnB 中的 V127I/L380F 双突变 **并未导致 6 的产生**。该结果表明：这些“第二壳层”或“第三壳层”残基通过微调活性口袋的构象动力学、空腔体积与底物结合方式，间接影响了血红素铁中心相对于 prenyl 底物的空间位置，从而改变了第二 HAT 的优势位点。NseB 中 Ile126 与 Phe379 的侧链体积较大，可能使血红素铁更偏向 C2 方向，促进 C2-H 攫取；PpnB 中体积较小的 Val127 与 Leu380 则将铁中心导向 C5 甲基。该发现与近年来 P450 酶工程中“远端残基调控反应选择性”的研究趋势一致。

### (六) 提出的催化机制与自然界中的普遍性意义

#### 实验目的与设计逻辑

综合上述全部实验与计算数据，作者提出了 PpnB 与 NseB 催化 prenyl 氧化去甲基化生成烯与炔的完整机制（**图 5c**），并进一步推测该策略可能在自然界中更为普遍，例如珊瑚中 ginamallene 的生物合成可能涉及类似的氧化去甲基化（**图 6**）。

#### 实验结果与机理解析

**图 5c** 总结了如下机制：底物 **7**（O-prenyl-L-Tyr）首先在 PpnB/NseB 作用下发生 **C4 甲基的区域选择性羟化**，经两步氧化生成 C4-醛（或偕二醇）中间体。随后，Cpd I（或 Cpd 0）触发 **C3-C4 键均裂**，释放甲酸并生成乙烯基自由基中间体。该自由基中间体的命运由第二 HAT 的位点决定：**PpnB 攫取 C5 末端甲基的氢原子**，生成末端烯 **1**；**NseB 攫取 C2 乙烯基氢原子**，生成内部炔 **6**。PpnB 与 NseB 的产物选择性差异由活性位点中第二壳层残基（V127/I126 与 L380/F379）通过调控血红素铁中心相对于底物的空间位置来实现。

**图 6** 提出了该策略在天然产物生物合成中的潜在普遍性：珊瑚来源的 norditerpenoid ginamallene（含末端烯）可能与 9-deacetoxy-14,15-deepoxyxeniculin 共存于同一生物体内，后者可通过一个 P450 催化的氧化去甲基化（切除一个碳）转化为前者。由于目前尚无 *Acalyciogorgia* 基因组数据，该假说有待未来验证。

# 六、总体结论

本研究首次报道了真菌细胞色素 P450 酶 **PpnB** 与 **NseB** 通过 **氧化性 C(sp²)-去甲基化** 将常见的五碳异戊烯基转化为四碳烯或炔的催化机制。这是 **自然界中首次发现从 sp² 杂化碳上直接切除甲基的酶**，分别实现了 **末端烯** 与 **内部炔** 的酶促合成，作者将其命名为“去甲基化烯化酶 (demethylative allenylase)”与“去甲基化炔化酶 (demethylative alkynase)”。该机制通过 DFT 计算、异源表达、体内饲喂、微体酶活、氘代底物标记与定点突变等多种手段得以系统阐明，核心发现包括：**C4 甲基的区域选择性羟化** → **C3-C4 键均裂** 生成乙烯基自由基 → **第二 HAT 攫氢位点** 决定产物类型（末端烯 vs 内部炔）。该研究还鉴定了第二个 P450 酶 **PpnD**，其催化 **末端烯到 α-羟基炔的氧化异构化**，代表了一类此前未被报道的“烯-炔异构酶”。**这是首次对 P450 酶催化烯与炔形成的完整机制进行系统性解析**，极大地拓展了 P450 酶的催化谱系。

本研究的科学意义在于：揭示了一条自然界合成烯与炔的 **全新的、普适性的生物合成策略**，即通过对 abundant 的异戊烯基单元进行氧化性 C-C 键断裂与氢原子攫取的区域选择性调控。由于异戊烯基化是真核生物中最普遍的翻译后修饰之一（包括蛋白质异戊烯基化与小分子异戊烯基化），该发现为 **在活细胞中原位生成含烯或炔的生物分子** 以用于生物正交化学与化学生物学研究提供了酶学工具基础。

# 七、论文评价

### 优点与创新

本研究的 **核心创新点** 在于发现了 P450 酶催化的 **C(sp²)-去甲基化** 形成烯与炔的机制。此前已知的 P450 催化 C-C 键断裂的范例（如 CYP51）均作用于 sp³ 杂化碳（甲基从饱和碳上切除），而本研究首次证明 **sp² 杂化碳上的甲基同样可被氧化性切除**，生成的乙烯基自由基通过区域选择性的第二 HAT 可转化为两种截然不同的官能团。该发现不仅拓展了 P450 酶的催化谱系，更 **为烯与炔的生物合成提供了统一的理论框架**。

在 **机理研究的深度与严谨性** 上，本文达到了很高的水准。从 DFT 计算的前瞻性预测，到异源表达体系的建立，再到 **四组氘代底物的精巧设计与同位素标记实验**，作者以层层递进的方式确定了去甲基化的区域选择性（C4 vs C5 甲基的排除）、中间体（C4-羟基化中间体 **17** 的检测与转化）以及第二 HAT 的攫氢位点（PpnB 攫 C5-H，NseB 攫 C2-H）。这种“计算 + 体内 + 体外 + 同位素”的多维度验证策略，为该机制提供了强有力的证据链。

在 **酶工程与结构生物学的前瞻性** 上，作者不仅鉴定出控制产物选择性的关键残基（V127/I126 与 L380/F379），还通过定点突变验证了这些第二壳层残基的功能影响——NseB 的 I126V/F379L 突变使烯/炔比例增加约 2 倍。尽管尚未达到“开关”式的选择性逆转，但该结果为未来通过定向进化或理性设计 **构建可控的烯/炔合成酶** 奠定了初步基础。

此外，该研究还展示了 **异源表达体系在真菌 P450 酶功能表征中的强大能力**。由于 PpnB 与 NseB 是膜锚定的真核 P450 酶，在 *E. coli* 中表达活性形式极为困难。作者巧妙地采用了 **微体 (microsome)** 体系——从 *A. nidulans* 转化株中提取含目标 P450 的膜组分，在 NADPH 存在下进行体外活性测定——成功绕过了蛋白纯化的障碍。这一方法学对于其他膜结合 P450 酶的功能研究具有借鉴意义。

### 未来研究方向

尽管本研究已取得重要突破，但仍有多项问题有待深入探究。**第一，PpnB 与 NseB 的高分辨率三维结构** 目前仍缺失（AlphaFold3 预测仅为计算模型）。通过 X 射线晶体学或冷冻电镜解析 PpnB 与 NseB 与底物 **7**（或抑制剂）的复合物结构，将能够直观地揭示 **血红素铁中心相对于 prenyl 基团的精确空间位置**，以及 V127/I126、L380/F379 等残基如何通过调控活性口袋构象来影响第二 HAT 的选择性。**第二，PpnB 与 NseB 的底物谱与催化效率** 有待系统评估。目前仅测试了 **7** 及其类似物（C4-羟基、C5-羟基），但对于其他 O-prenyl 酚类底物（如 O-prenyl-4-羟基苯甲酸、O-prenyl-对香豆酸等）以及非天然底物（如含不同取代基的芳香环）的催化活性尚不明确。这将是评估这些酶作为 **生物催化剂** 用于 click chemistry 中烯/炔官能团原位生成潜力的关键。**第三，PpnD 催化烯到炔的氧化异构化的详细机制** 尚有待深入。DFT 计算支持 HAT 起始的途径（**扩展数据图 5**），但实验证据主要来自底物饲喂与产物鉴定。未来可通过同位素标记（如 deuterated **10**）追踪氢原子的迁移，并结合 PpnD 的结构解析，区分 HAT 起始机制与 Cpd I 直接进攻烯烃的竞争途径。**第四，该策略在自然界中的普遍性** 有待进一步验证。作者在 **图 6** 中推测珊瑚 ginamallene 可能通过类似的氧化去甲基化形成。随着珊瑚基因组数据的积累（如近期报道的 *Xenia* 等属），在珊瑚中寻找 PpnB/NseB 的同源酶并验证其功能，将揭示该策略是否在动物界中也存在。此外，已知的 **2000 余种含炔天然产物** 中有多少是通过类似的 prenyl 去甲基化途径生成的？系统性基因组挖掘与途径重构将是回答该问题的关键。

# 八、关键问题及回答

**Q1**：本研究提出的 P450 催化 C(sp²)-去甲基化生成烯或炔的机制，与已知的 CYP51 催化 C(sp³)-去甲基化生成双键的机制有何异同？从化学原理上看，为什么 C(sp²)-去甲基化在热力学上是可行的？

**A**：CYP51 催化的羊毛甾醇 14α-去甲基化与本研究中的 prenyl 去甲基化，均涉及 **P450 催化的氧化性 C-C 键断裂并释放甲酸**，最终引入一个不饱和键。两者的关键差异在于被切除的甲基所连接的碳原子杂化状态：CYP51 的底物中 C14 为 sp³ 杂化（饱和碳），而本研究中的 prenyl 底物中 C3（乙烯基碳）为 sp² 杂化。从化学原理看，C(sp²)-去甲基化的热力学可行性由以下几个因素共同决定。第一，**甲酸的释放提供强大的热力学驱动力**：甲酸是稳定的闭壳层小分子，其生成焓极负（ΔH_f° = -378.6 kJ/mol）。DFT 计算表明，从偕二醇自由基 **Int2** 到乙烯基自由基 **Int3** 的 C-C 键断裂步骤的 ΔG° 为 **-11.0 kcal/mol**（放能过程）。第二，尽管乙烯基自由基本身的稳定性远低于 CYP51 反应中生成的叔碳自由基（C-H BDE 分别为 ~111 kcal/mol 与 ~96-98 kcal/mol），但 DFT 计算显示 **TS2（ΔG‡ = 8.1 kcal/mol）与 TS4（ΔG‡ = 15.4 kcal/mol）的能垒仍然可跨越**，表明该反应在酶促条件下是动力学可行的。第三，酶活性口袋可通过 **底物结合能** 与 **过渡态稳定化作用** 进一步降低能垒。值得注意的是，DFT 计算中 Cpd 0 依赖途径的速率决定步骤（TS4，~15.4 kcal/mol）与 CYP51 中报道的 Cpd 0 依赖途径能垒（~17.3 kcal/mol）相近，表明两种系统在能量上具有可比性。因此，尽管乙烯基自由基中间体能量较高，但甲酸释放的强驱动力与酶的催化作用共同补偿了这一不利因素，使 C(sp²)-去甲基化成为可能。

**Q2**：PpnB 与 NseB 的产物选择性（烯 vs 炔）由第二 HAT 攫氢位点决定。目前仅鉴定出 V127/I126 与 L380/F379 两个残基的差异，且突变仅能部分改变产物比例，而非实现完全的“开关”式转换。从酶的结构-功能关系角度分析，还有哪些因素可能调控这一选择性？

**A**：目前鉴定的 I126V/F379L 双突变使 NseB 的烯/炔比例增加约 2 倍，但并未实现选择性逆转（即 PpnB 突变后并未产生炔）。这一现象提示，**产物选择性可能由多个结构层次的协同效应决定**，而非仅由一两个残基的侧链体积差异所主导。其他可能的关键因素包括：**第一，底物结合通道的构象动力学**。P450 酶的活性口袋通常具有一定的柔性，底物结合后可诱导构象变化。PpnB 与 NseB 的底物结合通道中可能还存在其他“第二壳层”或“第三壳层”残基（如芳香族残基的堆积方式、极性残基的氢键网络），这些残基通过远程调控影响底物 prenyl 基团在活性口袋中的 **旋转自由度与优势构象**。例如，某芳香族残基的 π-π 堆积可能迫使 prenyl 的 C2-C3 双键采取特定取向，从而改变 C2-H 与 C5-H 相对于血红素铁中心的距离与角度。**第二，血红素辅基的埋藏深度与定位**。PpnB 与 NseB 中血红素结合口袋的形状与体积可能存在细微差异，影响 Cpd I/Cpd II 物种的氧原子（或 Fe(III)-O•）在三维空间中的“可及性”。即使铁中心的绝对位置相同，血红素平面的微小倾斜也可导致不同的 C-H 键接近矢量。**第三，水分子或活性位点配体的参与**。某些 P450 酶的结构中，活性口袋内的水分子可作为氢键供体/受体参与底物定位或过渡态稳定化。PpnB 与 NseB 中水分子分布的可能差异可能影响第二 HAT 的动力学偏好。**第四，蛋白质 dynamics 的长程变构效应**。近年来研究表明，P450 酶中远离活性中心的突变可通过改变蛋白质整体构象动力学来调控催化选择性。本研究中 I126 与 F379 位于血红素结合口袋而非底物结合口袋，其突变可能通过改变血红素周围二级结构的刚性或柔性，间接影响铁中心的反应性。综上，要全面理解产物选择性的结构基础，需要 **高分辨率晶体结构**（或 cryo-EM 结构）的解析，并结合 **分子动力学模拟** 与 **ensemble 分析**，以捕捉活性口袋中 prenyl 底物的优势构象分布及其与铁中心的空间关系。

**Q3**：本研究发现 PpnB 与 NseB 通过氧化去甲基化将异戊烯基转化为烯或炔。这一策略在合成生物学与化学生物学中具有哪些潜在应用价值？

**A**：该发现具有重要的 **合成生物学** 与 **化学生物学** 应用潜力。**第一，活细胞中炔基的原位生物合成**。叠氮-炔烃点击化学（CuAAC）是化学生物学中应用最广泛的生物正交反应之一，但目前炔基通常需要通过非天然氨基酸掺入或化学修饰引入。本研究中的 NseB 可在真菌宿主中直接将天然氨基酸 L-酪氨酸转化为含内部炔的非天然氨基酸 **6**（O-but-2-ynyl-L-Tyr）。通过将 **nseAB** 基因簇或 **nseB** 单独导入合适的细胞工厂（如 *Saccharomyces cerevisiae* 或 *E. coli*），可实现 **炔基氨基酸的从头生物合成**，进而通过 tRNA 合成酶/tRNA 对（如正交的吡咯赖氨酰-tRNA 合成酶系统）将其位点特异性地掺入目标蛋白，用于后续的 click 标记与成像。**第二，烯基的原位生成用于新型生物正交反应**。PpnB 催化生成的末端烯 **1** 可与近期报道的双膦-铜催化 phenoxy diazaborinine 形成反应（CuPDF）兼容，为活细胞中提供第二类生物正交手柄（与叠氮/炔烃正交）。**第三，蛋白质异戊烯基化的“功能化”编辑**。真核生物中大量蛋白质发生异戊烯基化修饰（如 Ras 超家族小 GTP 酶），该修饰对蛋白质的膜定位与信号转导至关重要。通过将 PpnB 或 NseB 与异戊烯基转移酶（如 FTase、GGTase）在细胞中共表达，理论上可将天然蛋白质上的异戊烯基原位转化为烯或炔基，从而实现 **对内源性异戊烯基化蛋白质的选择性标记与功能调控**。**第四，新型抗生素的生物合成**。本研究中 penipratynolene (**5**) 与 sinuxylamide B (**2**) 均为生物活性天然产物。通过在异源宿主中重构完整的 **ppn** 或 **nse** 途径，并对 PpnB/NseB 进行定向进化以改变其产物选择性或底物谱，有望产生 **非天然的烯/炔类似物库**，用于抗菌活性筛选。特别是含炔基的化合物，其叁键可作为“潜在的弹头”与靶蛋白中的亲核残基发生共价结合，为开发 **共价抑制剂** 提供了新的化学空间。总之，本研究揭示的异戊烯基去甲基化策略为 **在活细胞中按需生成生物正交官能团** 开辟了新途径，具有广泛的化学生物学与合成生物学应用前景。
