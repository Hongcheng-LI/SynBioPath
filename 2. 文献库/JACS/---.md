# 二：研究背景

植物来源的 labdane 相关二萜 (labdane-related diterpenoids, LRDs) 是一类结构多样且数量庞大的天然产物，占全部萜类化合物的 10% 以上，广泛参与植物激素调节、植保素防御以及食品添加剂和制药工业等多个领域。在生物合成层面，LRDs 源于共同的 precursors geranylgeranyl diphosphate (GGPP)，经 class II 二萜合酶 (copalyl diphosphate synthase, CPS) 催化生成各类 copalyl diphosphate (CPP) 及其羟基化类似物，再经 class I 二萜合酶 (kaurene synthase, KS 或 kaurene synthase-like, KSL) 进一步环化，形成 kaurene、abietane、atisane、pimarane 等多种骨架。随后，细胞色素 P450 酶 (CYPs) 对这些二萜烯烃进行氧化修饰，包括羟化及进一步氧化形成醛、酮、羧酸、内酯等官能团。

**Figure 1** 呈现了本研究关注的典型 LRD 骨架及其代表性活性分子。图中展示了 *Euphorbia fischeriana* 中已鉴定的四种二萜烯烃：由 EfTPS54 (ent-CPS) 与 EfTPS14 (ent-neoabietadiene synthase) 协作生成的 **ent-neoabietadiene (1)**；由 EfTPS54 与 EfTPS43 生成的 **ent-atiserene (2)**；由 EfTPS54 与 EfTPS1 生成的 **ent-kaurene (3)**；以及由 EfTPS54 与 EfTPS41 生成的 **ent-isopimara-7(8),15-diene (4)**。图中同时标出了 jolkinolides 类化合物的代表结构——jolkinolide B 和 17-hydroxy-jolkinolide B，其结构特征为一个与 C 环稠合的 α,β-不饱和 γ-内酯环，该内酯环被认为是其抗癌活性的关键药效团。此外，图中还指明了两种微量但活性极强的 jolkinolides——helioscopinolide A 和 helioscopinolide E，它们在 *E. fischeriana* 中的含量仅为约 3 mg/kg 植物干重。

尽管已有大量关于 LRD 生物合成的研究报道，但对某些特定骨架位点的氧化修饰以及特征性官能团（如 α,β-不饱和 γ-内酯）的形成机制仍不明确。植物基因组中通常编码数百个 CYP 基因，且 CYP 序列多样性高、底物选择性宽泛，加之植物生物合成基因在基因组中呈分散分布而非成簇存在，使得从植物中准确鉴定参与目标代谢物生物合成的 CYP 基因极为困难。传统的局部序列比对搜索（以已知酶为查询序列）和整合转录组-代谢组分析方法在处理 CYP 家族时准确性有限。近年来，机器学习方法在建立序列-功能关系和结构-功能关系方面展现出巨大潜力，可显著提高酶挖掘的准确性。基于此，本研究旨在构建一个基于序列相似性与底物化学空间距离的监督学习模型，用于从 *E. fischeriana* 的 CYP 库中高效预测能够氧化 LRD 烯烃的候选 CYP，进而阐明 jolkinolides 的生物合成途径，并建立其异源生物生产体系。

# 三：研究思路

本研究采用了“机器学习预测—异源体内筛选—体外酶学验证—代谢工程优化”的闭环策略。首先，作者从已报道文献中收集了 138 对已知 CYP-LRD 烯烃底物对（涉及 77 个 CYP 和 17 个底物），以每个 CYP 的氨基酸序列相似性百分比（基于成对序列比对）和底物的归一化欧氏距离（基于 21 个 MORFEUS 描述符经 PCA 降维后的 5 维化学空间）为两个输入特征，采用梯度提升决策树 (GBDT) 算法训练排序学习模型。该模型在 CYP-level 数据划分（38 个 CYP 训练集，39 个 CYP 测试集）和留一法交叉验证中均显著优于基线模型。随后，将 *E. fischeriana* 转录组中经 BLAST 初步筛选出的 80 个候选 CYP 与已知 CYP 合并映射，分别以四种内源二萜烯烃 (**1-4**) 为查询底物，预测得到 269 个 CYP-底物对，并按底物分别排序后合并去重，获得 55 个高置信度候选 CYP。根据对底物 **1** 的预测分数，将 80 个 CYP 划分为 A 组（26 个，对 **1** 有潜在活性）、B 组（29 个，机器学习方法识别但对 **1** 预测分数较低）和 C 组（25 个，未被机器学习方法选中）。通过在本氏烟草中与相应二萜合酶及 GGPP 供应增强体系共表达，逐组验证 CYP 的体内氧化活性，并对活性 CYP 进行产物分离、NMR 及 X 射线晶体学结构鉴定。进一步通过体外酵母微粒体反应确证关键 CYP（EfCYP82BU7 与 EfCYP71BF25）协同催化形成 α,β-不饱和 γ-内酯的机制，并通过代谢工程优化（截短 HMGR、CYP-CPR 融合、酶组合共表达）在 *N. benthamiana* 中实现了 helioscopinolides A 和 E 的高效生产（分别达到 331.0 和 102.2 mg/kg 植物干重）。最后，对所有活性 CYP 在四种 LRD 骨架上的氧化位点进行了系统图谱绘制。

# 四：研究方法

- **机器学习模型构建**：收集 138 个已报道的 CYP-LRD 底物对（77 个 CYP，17 个底物），使用 MORFEUS 工具计算每个底物的 21 个分子描述符，经主成分分析 (PCA) 降至 5 维化学空间并计算归一化欧氏距离；对 77 个 CYP 进行多序列比对 (Clustal Omega) 并计算成对序列相似性百分比。采用梯度提升决策树 (GBDT) 算法，以序列相似性和化学空间距离为双特征，使用 YetiRank 损失函数训练排序模型，通过网格搜索优化超参数（树数量 50~1000，树深度 1~12），以 nDCG@20 为优化指标。
- **异源表达体系**：以 *Agrobacterium tumefaciens* 介导的瞬时转化法在 *Nicotiana benthamiana* 叶片中共表达目标基因。为增强 GGPP 供应，共表达了 *Arabidopsis thaliana* 来源的 HDR (AtHDR)、HMGR (AtHMGR)、DXS (AtDXS) 和 GGPPS (AtGGPPS)，并使用截短 HMGR (tHMGR) 强化 MVA 途径。将 class II 与 class I 二萜合酶通过 GGGS 接头融合以提高代谢通量。CYP 与 CPR 之间也通过 GGGS 接头融合以提高氧化效率。
- **产物分析与结构鉴定**：采用气相色谱-质谱联用 (GC-MS) 检测低极性产物（二萜烯烃及单/双氧化产物），采用液相色谱-高分辨质谱 (LC-HRESIMS) 检测高极性产物（如 helioscopinolides）。通过柱层析分离纯化主要产物，利用 1D 和 2D NMR (COSY, NOESY, HSQC, HMBC) 确定平面结构与相对构型，通过单晶 X 射线衍射确定绝对构型（共获得 10 个化合物的晶体数据，CCDC 编号 2479433-2479434 及 2479436-2479442）。
- **体外酶活测定**：将目标 CYP 基因在酿酒酵母中表达，制备含 CYP 的微粒体组分。在体外反应体系中加入底物和 NADPH 再生系统，通过 GC-MS 检测产物生成，以空载体微粒体为阴性对照。
- **系统发育与序列分析**：对 *E. fischeriana* 转录组中 424 个 CYP 进行 BLAST 搜索（以已知 LRD 相关 CYP 为查询序列，阈值 40% 序列一致性），构建系统发育树并划分 CYP 家族。

# 五：实验设计及结果分析

### （一）机器学习模型的构建与候选 CYP 的筛选

#### 实验目的与设计逻辑

鉴于植物 CYP 家族成员众多、序列多样性高、底物谱宽泛，传统 BLAST 搜索难以精准锁定参与特定 LRD 骨架氧化修饰的 CYP。作者旨在开发一个基于监督学习的排序模型，该模型利用已知 CYP-底物对中的两个关键信息——蛋白质序列相似性（反映 CYP 进化关系）和底物化学空间距离（反映底物结构相似性）——来预测未表征 CYP 与给定 LRD 烯烃底物的兼容性，从而大幅提高候选基因的命中率。

#### 实验结果与机理解析

**Figure 2** 完整呈现了机器学习模型的构建流程与性能评估。**Figure 2A** 展示了输入数据的准备过程：从文献中收集 138 个已知 CYP-底物对，涉及 77 个 CYP 和 17 个 LRD 烯烃底物（如 ent-kaurene、ent-abietadiene 等）。每个底物首先通过 MORFEUS 工具计算 21 个分子描述符（涵盖拓扑、几何、电子性质等），随后进行 PCA 降维，选择解释方差最大的 5 个主成分（Table S4, Figure S2）构建 5 维化学空间。在此空间中，任意两个底物之间的归一化欧氏距离被计算为底物特征之一（值越小表示底物结构越相似）。对于 CYP，通过 Clustal Omega 进行多序列比对后，计算任意两个 CYP 之间的成对序列相似性百分比（即较短序列中相同残基数占比）。**Figure 2B** 说明了模型训练与超参数优化策略：采用 CYP-based 数据划分（38 个 CYP 为训练集，39 个 CYP 为测试集，允许训练集与测试集共享相同底物以防止过度乐观评估），使用梯度提升决策树 (GBDT) 算法，以序列相似性和归一化欧氏距离为两个输入特征，采用 YetiRank 损失函数进行排序学习。通过网格搜索优化树数量（50-1000）和树深度（1-12），以 nDCG@20 （Normalized Discounted Cumulative Gain at rank 20）为评价指标，选择最佳超参数组合（Table S5）。**Figure 2C** 对比了优化模型与基线模型（随机排序）在测试集上的多项性能指标：在 enrichment@k、nDCG@k、precision@k、recall@k 以及首个命中排名等指标上，优化模型在所有 k 值（2-50）范围内均显著优于基线模型。以 precision@10 为例，优化模型达到约 0.6，而基线模型不足 0.1。此外，作者还采用了留一法交叉验证 (LOOCV) 重复上述流程，结果同样显示优化模型在各指标上均大幅领先（Table S7），**证实该机器学习引导方法能够以高精度预测 CYP-底物兼容对**。

随后，将 *E. fischeriana* 中经 BLAST 初步筛选（以已知 LRD 相关 CYP 为查询序列，序列一致性 >40%）获得的 80 个候选 CYP 与 77 个已知 CYP 合并映射到 CYP 序列空间。分别以四种内源二萜烯烃 **ent-neoabietadiene (1)**、**ent-atiserene (2)**、**ent-kaurene (3)** 和 **ent-isopimara-7(8),15-diene (4)** 为查询底物，预测其与 157 个 CYP 的排序分数，去除重复后获得 269 个预测对（Table S8）。作者注意到，以 **1** 为查询底物时，预测分数整体低于以 **2-4** 为查询底物的分数，作者认为这可能归因于已知数据集中参与 **1** 氧化的 CYP 数量远少于参与 **2-4** 氧化的 CYP。为规避底物驱动偏差和数据泄漏，分别对四个底物的预测列表单独排序，各自选取前 20 名 CYP 后合并去重，得到 55 个高置信度候选。考虑到 jolkinolides（属于 abietane 型）是 *E. fischeriana* 中最具活性的 LRDs，且其前体为 **1**，因此将对 **1** 有潜在活性的 CYP 作为重点。在 **1** 的排序列表中，第 20 至第 26 位 CYP 具有相同的排序分数，故实际选取了 26 个候选。最终将 80 个 CYP 划分为三组：**A 组**（26 个，机器学习预测对 **1** 有活性）、**B 组**（29 个，机器学习方法识别但对 **1** 预测分数较低）和 **C 组**（25 个，未被机器学习方法选中）。

> “To further identify the more promising candidates from 80 CYPs, we herein developed a machine-learning-guided process, using both sequence similarity in protein space and normalized distance in chemical space as the input features.”

### （二）A 组 CYP 的体内功能表征与产物鉴定

#### 实验目的与设计逻辑

为验证机器学习预测结果的可靠性并鉴定参与 **1** 氧化的 CYP，作者将 A 组的 26 个候选 CYP 分别与 **1** 的生物合成模块（EfTPS54 + EfTPS14）以及 GGPP 供应增强酶（AtHDR, AtHMGR, AtDXS, AtGGPPS）在 *N. benthamiana* 中共表达。通过 GC-MS 检测氧化产物，并对活性 CYP 的产物进行分离、NMR 及 X 射线晶体学结构鉴定。

#### 实验结果与机理解析

**Figure 3A** 展示了 A 组中 6 个具有活性的 CYP 的 GC-MS 总离子流图。图 3A-i 为仅表达 **1** 生物合成模块的阴性对照，仅检测到底物 **1**（保留时间约 18 min）。图 3A-ii 为共表达 EfCYP71BF25，出现两个新峰 **5**（minor）和 **6**（major）；图 3A-iii 为 EfCYP71D936，同样生成 **5** 和 **6** 但产量较低；图 3A-iv 和 3A-v 分别为 EfCYP80C32 和 EfCYP80C33，生成 **5** 和 **6**，但其区域选择性发生反转（见下文）；图 3A-vi 为 EfCYP71D937，生成两个新峰 **9** 和 **10**；图 3A-vii 为 EfCYP76A211，同样生成 **9** 和 **10**。**Figure 3B** 展示了这些产物的化学结构：**5** (ent-neoabietadiene-17-al) 和 **6** (ent-neoabietadiene-16-al) 为 C-17 或 C-16 位的醛基衍生物（分子量 286 Da）；**9** (ent-neoabietadiene-3-one) 和 **10** (ent-3α-hydroxyneoabietadiene) 为 C-3 位的酮基或 α-羟基衍生物（分子量分别为 286 和 288 Da）。通过 NMR 及 **6** 的单晶 X 射线衍射（Figure S33）确证了上述结构。**Figure 3C** 揭示了 **5** 和 **6** 在正己烷中可自发氧化为相应的羧酸 **7** (ent-neoabietadiene-17-oic acid) 和 **8** (ent-neoabietadiene-16-oic acid)（分子量 302 Da），说明这些酸性产物可能是分离过程中的人工产物而非酶促直接产物。**Figure 3D** 以柱状图比较了 EfCYP71BF25 及其三个同工酶 (EfCYP71D936, EfCYP80C32, EfCYP80C33) 的催化效率（以 GC-MS 峰面积表示）。数据显示，**EfCYP71BF25 对 **6** (C-16 醛) 的生成效率显著高于对 **5** (C-17 醛)，而 EfCYP80C32 和 EfCYP80C33 则表现出相反的 C-17 偏好**。考虑到 jolkinolides 中 α,β-不饱和 γ-内酯的形成需要 C-16 位的氧化（生成醛或羧酸），EfCYP71BF25 因其优异的 C-16 氧化活性而被确定为后续研究的核心酶。

对 EfCYP76A211 和 EfCYP71D937 产物的 NMR 解析（Figures S41-S51）确认 **9** 和 **10** 分别为 C-3 位酮和 3α-羟基衍生物。值得注意的是，这两个酶功能相同，但 EfCYP76A211 的效率更高（图 3A 中峰面积对比）。此外，作者还测试了 A 组 CYP 对底物 **2**、**3** 和 **4** 的活性（Figures S10, S11）。结果显示，EfCYP71BF25 和 EfCYP71D937 能够氧化 **2-4** 的 C-3 位生成相应的 3α-羟基产物（与已知 C-3 氧化酶 EfCYP71D450 的产物一致），而 EfCYP71BF25 对 **1** 与对 **2-4** 表现出不同的区域选择性（在 **1** 上主要氧化 C-16，在 **2-4** 上则氧化 C-3）。此外，还鉴定出两个仅作用于 **2** 的新酶：EfCYP71AN125（催化 **2** 生成 ent-14β-hydroxyatiserene (14)、ent-atiserene-14-one (15) 和 ent-3α-hydroxyatiserene-14-one (16)）和 EfCYP76A210（催化 **2** 生成 ent-17-hydroxyatiserene (17)）。通过 NMR 及 ROESY 中 H-14 与 H3-20 的相关信号，确定了 **14** 中 C-14 羟基的 β 取向。

### （三）B 组与 C 组 CYP 的活性筛选及底物混杂性分析

#### 实验目的与设计逻辑

为全面评估机器学习方法的覆盖面，作者对 B 组（29 个，机器学习识别但未进入 **1** 的前 26 名）和 C 组（25 个，BLAST 获得但未被机器学习选中）的 CYP 也进行了系统的活性筛选，分别以 **1-4** 为底物。同时，作者希望发现更多具有新颖区域选择性的氧化酶，丰富 LRD 氧化的酶学工具库。

#### 实验结果与机理解析

**Figure 4A** 展示了 B 组中唯一具有活性的 CYP——EfCYP82J33 的 GC-MS 分析结果。图 4A-i 为 **2** 生物合成模块的对照；图 4A-ii 为共表达 EfCYP82J33 后，检测到 **2** 的多个氧化产物 **18-20**；图 4A-iii 为以 **3** 为底物，检测到产物 **21-23**；图 4A-iv 为以 **4** 为底物，检测到产物 **24**。**Figure 4B** 展示了所有产物的结构：**18** (ent-2α-hydroxyatiserene)、**19** (ent-atiserene-2-one)、**20** (ent-1α-hydroxyatiserene-2-one)、**21** (ent-2α-hydroxykaurene)、**22** (ent-kaurene-2-one)、**23** (ent-1α-hydroxykaurene-2-one) 和 **24** (ent-2α-hydroxyisopimara-7(8),15-diene)。通过 NMR 及 **18, 20, 21, 23** 的单晶 X 射线衍射（Figures S80, S91, S97, S108），**确证了 C-1 和 C-2 位的羟基构型（1α, 2α）以及 C-2 酮基的绝对构型**。值得注意的是，当以 **1** 为底物时，EfCYP82J33 并未氧化 C-1/C-2，而是催化 C-6 和 C-18 位的羟化，生成产物 **25** (ent-6α-hydroxyneoabietadiene) 和 **26** (ent-18-hydroxyneoabietadiene)（图 4A, 4B）。ROESY 中 H-6 与 H-5 的相关信号确定了 **25** 中 6-羟基的 α 取向。**该结果揭示了 EfCYP82J33 具有罕见的底物依赖型区域选择性切换能力**：在 atisane (2) 和 kaurane (3) 骨架上优先氧化 C-1/C-2，在 pimarane (4) 骨架上仅氧化 C-2，而在 abietane (1) 骨架上则氧化 C-6/C-18。

**Figure 4C** 展示了 C 组中唯一具有活性的 CYP——EfCYP82BU7 的 GC-MS 分析结果。以 **1** 为底物时，EfCYP82BU7 催化生成一个分子量为 288 Da 的产物 **27**。NMR 解析（Figures S126-S130）确定 **27** 为 **ent-12β-hydroxyneoabietadiene**，即 C-12 位 β-羟基衍生物。**这是首次报道能够催化 abietane 型 LRD 骨架 C-12 位羟化的 CYP**，该位点正是 jolkinolides 中 α,β-不饱和 γ-内酯环形成所必需的另一个关键位置。

作者进一步统计了候选 CYP 的命中率：在机器学习方法选出的 55 个 CYP（A+B 组）中，共鉴定出 9 个活性 CYP（EfCYP71BF25, EfCYP71D936, EfCYP80C32, EfCYP80C33, EfCYP71D937, EfCYP76A211, EfCYP82J33, EfCYP71AN125, EfCYP76A210），其中 8 个对 **1** 具有活性；而在 C 组的 25 个 CYP 中仅鉴定出 1 个活性 CYP (EfCYP82BU7)。通过对每个候选 CYP 与已知 77 个 CYP 的最大序列相似性进行分析（Figure S4），发现在最大相似性 >65% 的 37 个候选 CYP 中仅有一个活性 CYP，**表明简单的序列相似性阈值无法有效预测功能，而机器学习模型通过同时整合序列和底物化学空间信息，显著提高了预测精度**。

> “These results suggest that this machine-learning-guided approach can significantly increase the hit rate. Out of 10 CYPs characterized in this study, 8 are active toward 1.”

### （四）Jolkinolides 生物合成途径的阐明：α,β-不饱和 γ-内酯环的形成机制

#### 实验目的与设计逻辑

鉴于 EfCYP82BU7 催化 **1** 的 C-12β-羟化生成 **27**，EfCYP71BF25 催化 **1** 的 C-16 氧化生成 **6**，两者分别提供了内酯环形成的羟基和醛基/羧基前体，作者假设这两个酶的协同作用能够生成具有 α,β-不饱和 γ-内酯环的 jolkinolides。为验证这一假设，作者在 *N. benthamiana* 中共表达 **1** 的生物合成模块、EfCYP82BU7 和 EfCYP71BF25，并检测产物。同时，通过体外酵母微粒体反应解析内酯环形成的化学机制（是羟醛缩合脱水途径还是半缩醛氧化途径）。

#### 实验结果与机理解析

**Figure 5A** 展示了共表达实验的色谱分析结果。上方两张 GC-MS 总离子流图显示：仅表达 EfCYP82BU7 或仅表达 EfCYP71BF25 时，分别检测到 **27** 或 **5/6**；而共表达两者时，**27**、**5** 和 **6** 的峰显著减弱，同时出现一个新峰 **28**（GC-MS 可检测，表明其挥发性较高）。下方 LC-HRESIMS 提取离子流图显示：当进一步加入 C-3 氧化酶 EfCYP76A211 后，检测到两个新峰 **29** 和 **30**，其质子化分子离子 ([M+H]+) 分别为 m/z 317.2121 和 315.1962，与 helioscopinolide A (C20H28O3, 计算值 317.2117) 和 helioscopinolide E (C20H26O3, 计算值 315.1960) 的标准品完全一致。通过 NMR 和 X 射线晶体学（Figures S132-S137）确认 **28** 为 **jolkinolide E**。**Figure 5B** 提出了两种可能的 α,β-不饱和 γ-内酯形成机制：途径 A（黄线）为 C-12 羟基与 C-16 羧基之间的脱水缩合；途径 B（绿线）为 C-12 羟基对 C-16 醛基发生亲核进攻生成半缩醛中间体，再经脱氢生成内酯。**Figure 5C** 通过体外酵母微粒体实验区分这两种机制。将含有 EfCYP82BU7 的微粒体与底物 **8**（C-16 羧酸）或 **6**（C-16 醛）分别孵育，结果表明 EfCYP82BU7 不能将 **8** 转化为 **28**，但能将 **6** 转化为 **28**；将含有 EfCYP71BF25 的微粒体与 **27**（C-12 羟基）孵育，也能生成 **28**。**这些结果明确证明：α,β-不饱和 γ-内酯环的形成源于 C-12 羟基对 C-16 醛基的亲核进攻（途径 B），而非脱水缩合途径。同时，C-12 与 C-16 的氧化顺序不具有严格的先后性**。此外，作者还发现共表达 EfCYP71BF25 与 EfCYP82BU7 时，仅生成源于 C-16 氧化与 C-12 氧化组合的产物 **28**，而未检测到 EfCYP71BF25 单独表达时产生的 C-17 醛产物 **5** 及其可能的 C-12 羟基衍生物（ent-12β-hydroxyneoabietadiene-17-al）。时间进程实验（Figure S14）显示，在共表达的 *N. benthamiana* 叶片中，**28** 在转染后第 1 天即可检测到，但中间体半缩醛始终未被检出，提示其迅速转化为内酯。体外时间进程实验（Figures S15, S16）同样未检测到 C-17 醛相关的内酯产物，**表明 EfCYP71BF25 与 EfCYP82BU7 之间可能存在蛋白质相互作用，从而“纠正”了 EfCYP71BF25 固有的混杂性，使其优先或仅催化 C-16 而非 C-17 的氧化**，这与 *E. fischeriana* 中从未分离到 C-17 系列 jolkinolides 的事实一致。

对于 C-3 氧化在生物合成时序中的位置，作者发现 EfCYP76A211 不能直接氧化 jolkinolide E (28)，但 EfCYP71BF25 + EfCYP82BU7 能够氧化 **9** (ent-neoabietadiene-3-one) 和 **10** (ent-3α-hydroxyneoabietadiene) 分别生成 **30** (helioscopinolide E) 和 **29** (helioscopinolide A)（Figure S17）。**这表明 C-3 氧化发生在内酯环形成之前**。

### （五）代谢工程优化与活性 CYP 的底物谱全景

#### 实验目的与设计逻辑

为实现微量高活性 jolkinolides 的规模化生产，作者对 *N. benthamiana* 中的代谢通路进行了多轮优化。同时，为系统评估所鉴定 CYP 的应用潜力，作者绘制了它们在四种 LRD 骨架上的氧化位点图谱，并探索了 CYP 组合的协同效应。

#### 实验结果与机理解析

作者首先通过共表达 AtHDR、AtHMGR、AtDXS 和 AtGGPPS 将 **1** 的产量从 64.9 mg/kg 提升至 1140.9 mg/kg（约 17.6 倍）（Figure S7）。进一步采用截短 HMGR (tHMGR) 强化 MVA 途径，并将 EfTPS54 与 EfTPS14 通过 GGGS 接头融合，使 **1** 的产量进一步提升至 2207.9 mg/kg。当在此优化底盘中共表达 EfCYP76A211、EfCYP82BU7 和 EfCYP71BF25 时，helioscopinolide A (29) 和 helioscopinolide E (30) 的产量分别达到 208.3 和 56.9 mg/kg。最后，将每个 CYP 与其还原伴侣 AtCPR 通过 GGGS 接头融合共表达，使 **29** 和 **30** 的产量进一步提升至 **331.0 mg/kg 和 102.2 mg/kg**，分别是 *E. fischeriana* 中天然含量的 **103.3 倍和 30.1 倍**。

**Scheme 1**（位于正文结果部分末尾及支撑信息中）以全景图形式总结了本研究鉴定的 10 个活性 CYP 在四种 LRD 骨架上的所有氧化位点。其中蓝色圆点标记的是此前从未报道过的氧化位点，包括：在 **1** 上的 C-6、C-12、C-16、C-17、C-18；在 **2** 上的 C-1、C-2、C-14、C-17；在 **3** 上的 C-1、C-2；在 **4** 上的 C-2。通过单个及组合表达，共生产了 31 个 LRDs，其中 22 个为新化合物（Scheme 1 中以星号标记）。特别值得注意的是，**EfCYP82J33 在所有四个底物上均表现出活性，且在不同骨架上切换区域选择性**，是迄今为止报道的最具底物混杂性的 LRD 氧化 CYP 之一。此外，作者还探索了 CYP 组合的协同效应（Figures S12, S13）：EfCYP71AN125 与 EfCYP82J33 共表达时生成 atisane 骨架的 C-2/C-14 双氧化产物；EfCYP71D450 (C-3 氧化酶) 与 EfCYP82J33 共表达时生成 **2-4** 的 2α,3α-二羟基衍生物，而单独表达 EfCYP82J33 时生成的 C-2 单氧化产物（**18-24**）在共表达体系中消失，提示两个 CYP 之间同样可能存在相互作用，改变了 EfCYP82J33 的区域选择性。

# 六：总体结论

本研究通过构建一个整合蛋白质序列相似性与底物化学空间距离的机器学习排序模型，从 *Euphorbia fischeriana* 的 80 个候选 CYP 中高效鉴定出 10 个参与 labdane 相关二萜氧化的活性 CYP，其中 8 个对 jolkinolides 前体 **ent-neoabietadiene (1)** 具有活性。**这 10 个 CYP 分属于 7 个亚家族（CYP71D、CYP76A、CYP82BU、CYP80C、CYP71BF、CYP82J、CYP71AN），其中 CYP82BU、CYP80C、CYP71BF、CYP82J 和 CYP71AN 五个亚家族是首次被报道具有 LRD 氧化能力**。**关键功能酶 EfCYP82BU7 催化 C-12β-羟化，EfCYP71BF25 催化 C-16 氧化（生成醛），两者协同通过半缩醛中间体形成 α,β-不饱和 γ-内酯环，首次揭示了 jolkinolides 特征药效团的酶促生成机制**。结合 C-3 氧化酶 EfCYP76A211 的三酶共表达，在代谢工程优化的 *Nicotiana benthamiana* 中实现了 helioscopinolides A 和 E 的高效生产，**产量分别达到 331.0 mg/kg 和 102.2 mg/kg 植物干重，是天然来源含量的 103.3 倍和 30.1 倍**。本研究还绘制了 10 个 CYP 在四种 LRD 骨架上的系统氧化图谱，**共生产 31 个 LRDs（其中 22 个为新化合物），首次实现了对 abietane 骨架 C-6、C-12、C-16、C-17、C-18 位点，atisane 骨架 C-1、C-2、C-14、C-17 位点，kaurane 骨架 C-1、C-2 位点以及 pimarane 骨架 C-2 位点的酶促氧化**。该工作不仅为植物 CYP 的精准挖掘提供了高性能的机器学习策略，也为 jolkinolides 及其他高价值 LRDs 的异源生物生产建立了丰富的酶学工具包。

# 七：论文评价

### 优点与创新

1. **创新的机器学习引导的 CYP 挖掘策略**：与传统的 BLAST 序列相似性搜索相比，本研究构建的监督学习模型同时纳入了蛋白质序列空间与底物化学空间两个维度的信息，将候选 CYP 的命中率从 BLAST 的 12.5%（80 个中 10 个活性）提升至机器学习预测集（A+B 组）的 16.4%（55 个中 9 个活性），而对 **1** 的命中率在 A 组中高达 30.8%（26 个中 8 个活性）。该方法为植物基因组中分散分布的超家族酶（如 CYP、2OG 加氧酶等）的功能挖掘提供了可推广的范式。
2. **首次揭示了 jolkinolides 中 α,β-不饱和 γ-内酯环的完整生物合成途径**：通过体内共表达与体外微粒体反应，明确证明了内酯环的形成源于 C-12 羟基对 C-16 醛基的亲核进攻（半缩醛途径），而非脱水缩合途径。同时发现了 EfCYP71BF25 与 EfCYP82BU7 之间可能的蛋白质相互作用对区域选择性的“纠正”作用。
3. **丰富的酶学多样性发现**：EfCYP82J33 展现出罕见的底物依赖型区域选择性切换能力（在 atisane/kaurane 上氧化 C-1/C-2，在 pimarane 上仅氧化 C-2，在 abietane 上氧化 C-6/C-18），为研究 CYP 结构-功能关系及定向进化提供了理想模型。
4. **高效的异源生产体系**：通过代谢工程多轮优化（增强 GGPP 供应、合酶融合、CYP-CPR 融合），在 *N. benthamiana* 中将两种微量活性 jolkinolides 的产量提高了两个数量级以上，为其后续药理学研究提供了物质基础。

### 未来研究方向

1. **CYP-CPR 相互作用与区域选择性调控的结构基础**：本研究发现 EfCYP71BF25 与 EfCYP82BU7 共表达时，EfCYP71BF25 的 C-17 氧化活性被抑制，推测存在酶-酶相互作用。通过解析 CYP-CPR 复合物及 CYP-CYP 复合物的三维结构，结合交联质谱和分子动力学模拟，可阐明这种相互作用的分子机制，并为人工设计 CYP 组合提供指导。
2. **EfCYP82J33 底物依赖型区域选择性切换的机制**：该酶在不同骨架上攻击不同碳原子（C-1、C-2、C-6、C-18），其结构基础尚不明确。通过获得 EfCYP82J33 与不同底物的共晶结构或进行同源建模-对接-突变验证，可揭示底物结合口袋的构象可塑性如何决定氧化位点。
3. **机器学习模型的扩展与优化**：当前模型仅使用序列相似性（一维）和底物化学空间（基于分子描述符）两个特征。未来可整合 CYP 结构模型（如 AlphaFold 预测结构中的底物结合通道体积、静电势等）以及转录组共表达数据，构建多模态深度学习模型，进一步提高预测准确性。此外，将训练集扩展至其他植物类型的 CYP（如萜类吲哚生物碱、黄酮类代谢相关 CYP）可提升模型的通用性。

# 八：关键问题及回答

**问题一：本研究中机器学习模型的两个输入特征（CYP 序列相似性百分比和底物化学空间中的归一化欧氏距离）分别捕捉了什么样的生物学信息？为什么将两者结合能够显著提高预测准确度？**

**解答**：CYP 序列相似性百分比反映了不同 CYP 之间的进化关系与结构保守性。在 CYP 超家族中，同一家族或亚家族的成员往往具有相似的底物结合口袋拓扑结构和催化残基排列，因此序列相似性高的 CYP 更可能识别结构类似的底物或催化相似类型的反应。然而，仅凭序列相似性无法区分那些序列相似但底物谱已发生漂移的 CYP，也无法预测同一 CYP 对不同底物的相对活性。底物化学空间中的归一化欧氏距离则基于 MORFEUS 描述符经 PCA 降维后计算，这些描述符涵盖了底物的分子量、拓扑极性表面积、氢键供体/受体数、可旋转键数、环系特征等 21 个理化参数。**两个底物在 5 维化学空间中的距离越近，说明它们在外形、电子分布和疏水性等方面越相似，越有可能被同一个或序列相似的 CYP 识别并发生类似的氧化反应**。将这两个特征结合训练排序模型，相当于让模型同时学习“酶-酶相似性”和“底物-底物相似性”对“酶-底物兼容性”的贡献。例如，一个未表征 CYP 与已知 CYP 序列相似性仅为 50%，但如果其目标底物与已知底物在化学空间中非常接近（小距离），模型仍可给出较高的预测分数。这种双特征融合策略有效弥补了单一特征的信息不足，这也是本模型在 enrichment@10 和 nDCG@20 等指标上显著优于基线模型（随机排序或仅基于序列相似性排序）的根本原因。

**问题二：EfCYP71BF25 与 EfCYP82BU7 共表达时，为何仅生成 C-16 氧化与 C-12 氧化组合的内酯产物 (28)，而不生成 C-17 氧化的潜在产物（如 ent-12β-hydroxyneoabietadiene-17-al）？请从酶学与代谢组学角度分析可能的原因。**

**解答**：作者在 *N. benthamiana* 共表达体系和酵母微粒体体外反应中均观察到这一现象，排除了宿主背景酶的干扰。可能的解释包括以下三个方面。第一，**蛋白质相互作用介导的区域选择性重定向**：EfCYP71BF25 与 EfCYP82BU7 可能形成瞬时的异源复合物（heterocomplex），这种复合物的构象变化改变了 EfCYP71BF25 底物结合口袋中 **1** 的结合模式，使其 C-16 位更靠近血红素铁氧活性物种 (Compound I)，而 C-17 位被“屏蔽”或不再处于催化距离内。类似的现象在植物黄酮代谢（如 CYP 与 CPR 及细胞色素 b5 的复合物）以及萜类代谢（如 AcaCYP71A1 与 AcaCYP701A1 共表达时 C-11 氧化活性被抑制）中已有报道。第二，**底物通道的竞争性占用**：若 EfCYP82BU7 优先结合并催化 **1** 生成 **27**，后者作为 EfCYP71BF25 的底物时可能具有更高的结合亲和力。实际上，体外实验（Figure 5C）已证明 EfCYP71BF25 能直接以 **27** 为底物生成 **28**，而不需要先催化游离 **1** 的 C-16 氧化。因此，当两个酶共表达时，代谢通量可能被引导至“EfCYP82BU7 先催化 C-12 氧化 → EfCYP71BF25 再催化 C-16 氧化”的途径，而 EfCYP71BF25 单独表达时对 **1** 的直接 C-17 氧化途径在通量上被抑制。第三，**代谢组学证据的一致性**：迄今为止，从 *E. fischeriana* 中分离的天然 jolkinolides 均为 C-16 氧化系列（如 jolkinolide B/E、helioscopinolides 等），从未分离到 C-17 系列内酯。这表明在植物体内，EfCYP71BF25 的 C-17 氧化活性可能始终被其他互作蛋白（包括 EfCYP82BU7）所“校正”，这一进化选择的结果是只生成具有更强生物活性的 C-16 内酯产物。

**问题三：EfCYP82J33 在四种 LRD 骨架上表现出截然不同的区域选择性（在 atisane/kaurane 上氧化 C-1/C-2，在 abietane 上氧化 C-6/C-18，在 pimarane 上仅氧化 C-2）。请从底物构象和酶活性口袋的柔性角度讨论这一现象的分子基础，并指出其在合成生物学中的潜在应用价值。**

**解答**：EfCYP82J33 的区域选择性切换是研究 CYP 底物混杂性与构象可塑性的理想模型。四种底物 **1 (ent-neoabietadiene)**、**2 (ent-atiserene)**、**3 (ent-kaurene)** 和 **4 (ent-isopimara-7(8),15-diene)** 虽然共享 labdane 骨架起源，但其环系结构和取代模式存在显著差异：**2** 和 **3** 具有更紧凑的笼状环系（B/C 环反式稠合，C 环未芳香化），**4** 为双环结构（C 环为十元环），**1** 则为具有 C 环双键和甲基取代的 abietane 骨架。推测 EfCYP82J33 的底物结合口袋具有较高的柔韧性，能够通过侧链重排或主链 loop 区的构象变化适应不同底物的形状。当结合 **2** 或 **3** 时，底物以某种取向进入，使得 C-1 和 C-2 上的 pro-R 或 pro-S 氢朝向血红素；当结合 **4** 时，由于十元环的构象灵活性，底物可能采用另一种结合模式，仅使 C-2 位处于催化距离内；当结合 **1** 时，abietane 骨架的 C 环双键和甲基可能迫使底物翻转，使得 C-6 和 C-18（位于 A 环的远端）朝向活性中心。**这种底物依赖的“重定向”机制暗示 EfCYP82J33 的活性口袋具有多个底物结合子口袋（sub-pockets），不同底物通过“诱导契合”选择不同的子口袋**。在合成生物学中，EfCYP82J33 的底物宽泛性和区域选择性切换能力使其成为一个极具价值的“混杂性催化剂”：可以通过蛋白质工程（定点突变、loop 区交换）将其改造为对特定骨架具有高区域选择性的专用催化剂；也可以利用其天然混杂性，在异源宿主中同时氧化多种 LRD 骨架，生成结构多样的氧化产物库（本研究中已用其生产了 7 个不同骨架的氧化产物），用于构建天然产物类化合物库以供药物筛选。此外，对其区域选择性分子基础的解析，可为设计“可按需切换”的智能生物催化剂提供理论指导。
