# 二、研究背景

细胞分裂素 (cytokinins, CKs) 是一类以腺嘌呤为核心、在 N6 位连接不同侧链的植物激素，参与细胞分裂、叶片衰老、营养信号转导以及根和芽的形态建成。根据 N6 取代基性质，CKs 主要分为异戊二烯型、芳香型和呋喃甲基型。其中，异戊烯腺嘌呤 (N6-(Δ2-isopentenyl)adenine, iP)、反式玉米素 (trans-zeatin, tZ) 和顺式玉米素 (cis-zeatin, cZ) 是植物中最具代表性的天然 CKs。

植物的 CK 生物合成及稳态调控已形成相对清晰的理论框架。腺苷磷酸异戊烯基转移酶 (adenosine phosphate-isopentenyltransferase, IPT) 首先生成异戊烯基化核苷酸；CYP735A 对其侧链进行羟基化；LONELY GUY 家族磷酸核糖水解酶 (phosphoribohydrolase, LOG) 随后切断核糖磷酸与嘌呤之间的 C—N 键，释放活性游离碱基。过量 CK 则主要通过 CKX 氧化裂解或糖基化进行清除、储存和再活化。

与植物相比，植物病原真菌中的 CK 化学长期缺乏系统研究。已有研究表明，部分病原真菌能够合成并分泌 CKs，从而重新分配寄主营养、改变细胞周期或调节感染组织发育。然而，真菌 CK 的合成酶组成、侧链修饰方式、失活反应以及这些衍生物是否仍能被植物受体识别，均未得到完整解析。尤其需要回答的是，真菌是否仅复制植物途径，还是已经演化出独立的 CK 化学调控体系。

本研究以真菌 IPT-LOG 融合蛋白为基因组挖掘入口，从 _Fusarium expansum_ CGMCC 3.0711 中锁定 fex 基因簇，并依次解析 FexA、FexB、FexC 和 FexD 的催化功能。作者的总体目标并非仅鉴定若干新化合物，而是建立“基因簇组织—酶催化反应—产物生物活性—植物信号响应”之间的完整对应关系，进而阐明 _Fusarium_ 如何同时完成 CK 的合成、活化与不可逆失活。

# 三、研究思路

研究首先从序列相似性网络 (sequence similarity network, SSN) 出发，对真核生物 LOG 家族进行系统分析，识别出一个与经典植物 LOG 分离的 IPT-LOG 融合蛋白群。随后，作者结合物种来源统计、系统发育分析及基因簇共线性比较，将 _Fusarium_ IPT-LOG 相关基因簇划分为 3 个进化分支，并选择分布较广、氧化还原酶配置丰富的 clade I fex 基因簇开展功能研究。

在基因功能解析阶段，作者采用构巢曲霉 (Aspergillus nidulans) 逐步异源重构 fex 途径。通过单基因和多基因组合表达，先确定 FexA 生成 iP，再解析 FexB 对 iP、tZ 和 cZ 的连续氧化功能。针对不稳定醛中间体，研究采用 2-nitrophenylhydrazine 衍生化捕获，从而将终产物结构与短寿命中间体连接起来，建立 FexB 催化 pyrrole-CK 形成的反应序列。

对于 FexC 和 FexD，作者进一步引入同位素示踪、辅因子依赖性分析、关键基序突变及异源氧化酶耦联实验。该部分不仅验证了产物，还区分出 FexC 催化后的 3 条竞争性反应分支：C—N 键裂解、C1′ 氧化和 FexD 介导的双键异构化。最后，研究通过 4 类植物生理实验和双组分信号通路基因表达分析，将化学结构变化与 CK 活性强弱对应起来，形成闭合的机制论证链。

# 四、研究方法

- **序列相似性网络分析 (SSN)**：以 InterPro IPR005269 家族中的 6057 条真核 LOG 蛋白序列为数据集，识别经典 LOG 与 IPT-LOG 融合蛋白之间的序列聚类关系。
    
- **系统发育与物种分布分析**：统计 IPT-LOG 融合蛋白的物种来源，并构建 _Fusarium_ 同源蛋白系统发育树，将相关蛋白及其 BGC 划分为 3 个进化分支。
    
- **基因簇共线性比较**：比较不同 _Fusarium_ 菌株中 IPT-LOG、CYP450、BBE-like oxidase、NmrA-like SDR、转录因子及转运相关基因的排列与保守程度。
    
- **真菌异源表达与途径重构**：以构巢曲霉为主要表达底盘，分别构建 AN-fexA、AN-fexAB、AN-fexACD 和 AN-fexABCD 等菌株，根据代谢谱变化确定基因功能。
    
- **重组蛋白表达及体外酶促反应**：在大肠杆菌中表达带有麦芽糖结合蛋白标签的 FexA、FexA 截短体和 FexD；对于难以获得可溶蛋白的 FexC，则使用构巢曲霉粗酶体系开展反应。
    
- **微粒体 CYP450 酶活分析**：制备含 FexB 的微粒体组分，分别以 iP、tZ、cZ 和 N6-methyl-iP 为底物，解析其羟基化、醛化及环化能力。
    
- **HPLC、LC-MS 与高分辨质谱分析**：监测异源表达菌株和体外反应的代谢物变化，并利用提取离子流图确认低丰度产物和衍生化产物。
    
- **化合物分离与 NMR 结构鉴定**：通过放大发酵或酵母生物转化制备目标产物，结合 HRMS 和一维、二维 NMR 确定化合物 **4–6**、**9**、**16–19** 等的结构。
    
- **醛基化学捕获**：使用 2-nitrophenylhydrazine 对不稳定的 C4′ 醛和 3-methyl-2-butenal 进行衍生化，分别获得 Z/E-**8**、Z/E-**12** 和 Z/E-**15**。
    
- **同位素示踪实验**：采用 H218O 判断 C1′-keto-iP 中羰基氧的来源，并利用 D2O 检验 FexC-FexD 双键异构化过程是否引入溶剂来源的氘。
    
- **植物生物活性评价**：通过烟草愈伤组织诱导、小麦叶片叶绿素保留、苋属子叶甜菜红素生成以及拟南芥主根伸长抑制，比较经典 CK 与真菌特异衍生物的活性。
    
- **实时荧光定量 PCR**：检测不同 CK 处理后拟南芥 AHK4、AHP2、ARR5 和 ARR15 的转录水平，评估衍生物对双组分信号通路的调节作用。
    

# 五、实验设计及结果分析

### (一) 从 IPT-LOG 融合蛋白锁定 _Fusarium_ CK 基因簇

#### 实验目的与设计逻辑

在解析具体酶功能之前，作者需要回答 fex 基因簇是否代表一个普遍存在于病原真菌中的 CK 代谢模块。为此，研究先以已知 LOG 蛋白家族为边界开展 SSN 分析，再将融合蛋白的物种分布、系统发育位置和邻近基因组成进行交叉验证。图 1 用于建立植物途径与真菌途径的比较框架，图 2 则从序列和基因组层面给出候选途径的发现依据。

#### 实验结果与机理解析

如图 1A 所示，作者首先标示了腺嘌呤骨架的原子编号及 N6 取代位置，并将 CKs 分为异戊二烯型、芳香型和 furfuryl CKs。iP、tZ、cZ 与 dihydrozeatin 共享异戊二烯侧链，但氧化状态、双键几何构型及还原程度不同；MeoT、BA 和 MemT 则连接芳香侧链。该结构分类说明，**CK 活性并不只取决于腺嘌呤核心，N6 侧链的构型与氧化状态同样构成关键活性决定因素。**

![image.png](https://synbiopath.online/20260724103930493.png)

图 1B 将植物异戊二烯型 CK 的合成和失活整合于同一反应图中。新生合成途径先由 IPT 将 dimethylallyl pyrophosphate 转移至 ATP、ADP 或 AMP，形成 iPRTP、iPRDP 或 iPRMP；CYP735A 可在 LOG 切割前羟基化核苷酸侧链，最终生成 tZ。cZ 则主要来自 tRNA 异戊烯基化、顺式羟基化和 tRNA 降解。图右侧显示 CKX 裂解为 adenine 属于不可逆降解，而 N-糖基化和 O-糖基化分别改变活性或形成可逆储存形式。

与植物模型相对照，图 1C 对全文发现进行了预先归纳。FexA 以 AMP 和 DMAPP 合成 iP；FexB 将其转化为 tZ、cZ 及 pyrrole-CK **6**；FexC 既能氧化裂解侧链，也可生成 C1′-keto-iP **16**；FexC 与 FexD 则共同形成 Δ1-iP **17** 和 Δ1-tZ **19**。**该图提出的核心区别是，_Fusarium_ 以吡咯环形成和双键异构化作为 CK 失活方式，而植物主要依赖侧链裂解与糖基化。**



在图 2A 的 SSN 中，植物、子囊菌、担子菌、酵母目及 _Cuscuta_ 来源 LOG 各自形成相对独立的网络区域。在经典真菌 LOG 网络之外，研究识别出一个规模较小但边界清晰的 IPT-LOG 融合蛋白簇。该簇并非普通 LOG 序列的随机延伸，而是形成独立的序列群，说明 IPT 与 LOG 的融合可能经历了特定的功能选择，并可作为真菌 CK 新生合成途径的基因组标志。

![image.png](https://synbiopath.online/20260724162547872.png)

图 2B 对融合蛋白的物种来源进行了量化。所检出的序列中，**108 条来自 _Fusarium_，10 条来自 _Claviceps_**，其余分别来自 _Dactylonectria_、_Clonostachys_、_Ilyonectria_、_Ustilaginoidea_、_Epichloë_ 和 _Rhexocercosporidium_ 等植物相关真菌。融合蛋白对 _Fusarium_ 的高度富集提示，该酶体系可能并非真菌普遍的基础代谢模块，而更可能与部分植物病原真菌的生态适应有关。

进一步的系统发育分析将 _Fusarium_ IPT-LOG 融合蛋白划分为 clade I、clade II 和 clade III，如图 2C 所示。3 个分支均包含多个物种来源序列，但分支内部聚类相对紧密，说明融合蛋白在属内发生了进一步分化。结合不同分支邻近 CYP450、糖基转移酶、乙酰转移酶和氧化酶的差异，作者推断这些基因簇可能生成不同氧化状态和生物活性的 CK 衍生物，而非仅重复合成 iP。

图 2D 对 clade I 基因簇进行了共线性比较。_F. expansum_ fex 基因簇包含核心融合酶 fexA、两个 CYP450 基因 fexB 和 fexE、BBE-like oxidase 基因 fexC 以及 NmrA-like SDR 基因 fexD，外围还分布转录因子、RTA1 和 permease。不同 _F. oxysporum_ 菌株中核心功能基因虽然存在方向倒置或位置变化，但总体组合得到保留。**保守的 FexA-FexB-FexC-FexD 模块为该途径具有选择性维持的生化功能提供了基因组证据。**

### (二) FexA 建立 iP 核心，FexB 将羟基化与 pyrrole-CK 形成耦联

#### 实验目的与设计逻辑

fex 基因簇中 IPT-LOG 融合酶位于途径核心，但融合形式并不能直接证明两个结构域均具有活性，也不能确定其实际核苷酸底物。作者因此先在不含相关基因簇的构巢曲霉中表达 fexA，再通过截短蛋白和体外反应解析两个结构域的分工。在此基础上，研究逐一加入 CYP450 基因，并利用底物替换及醛基捕获实验阐明 pyrrole-CK 的形成机制。

#### 实验结果与机理解析

图 3A 中，AN-WT 未产生目标 CK 峰，而 AN-fexA 出现 iP **1** 为主峰，并伴随 iPR **4** 和 C8-hydroxyl-iP **5**。向野生型构巢曲霉补加 iP 后同样可产生化合物 **5**，说明该产物主要来自底盘的过度氧化，而不是 FexA 的直接催化。AN-fexAE 的代谢谱与 AN-fexA 基本一致，表明在该条件下 FexE 未表现出可检测的修饰功能。

与之不同，AN-fexAB 中 iP 峰明显降低，并出现 tZ **2**、少量 cZ **3** 及一个新的低丰度产物。针对 m/z 200 [M+H]+ 的提取离子流图进一步确认，该产物仅在表达 fexB 的菌株中形成，并被鉴定为 pyrrole-CK **6**。**异源表达结果确立了 FexA 负责构建 iP，FexB 则同时承担 canonical CK 羟基化与非典型环化反应。**

图 3B 集中列出了机制研究涉及的化合物。iPR **4** 和 C8-hydroxyl-iP **5** 用于解释异源宿主中的旁路产物；化合物 **7** 是推定的 C4′-aldehyde 中间体；其经 2NP 捕获形成 Z/E-**8**。N6-methyl-iP **9** 及其羟基化产物 **10**、醛中间体 **11** 和衍生物 Z/E-**12** 则构成 N6-H 必要性实验。该结构组合使“连续氧化”和“后续成环”两个步骤能够被分别检验。

在图 3C 的体外反应中，全长 FexA 与 ATP、ADP 或 AMP 孵育时均能检测到 iP **1** 和 iPR **4**。结合截短结构域实验，AMP 体系中 iPRMP 积累效率最高，而 ATP 和 ADP 体系中的反应可由其自发水解为 AMP 解释。FexA-IPT 生成 iPRMP，FexA-LOG 随后专一切断 iPRMP 的 C1″—N9 键；iPR **4** 则来自 iPRMP 的非酶促脱磷酸，不能被 LOG 结构域重新利用。

因此，FexA 并不是直接以 adenosine 为受体，也不是由两个结构域分别释放游离中间体后再扩散反应。**其实际化学过程为：IPT 结构域以 AMP 为底物生成 iPRMP，LOG 结构域随即切除磷酸核糖基，一步输出游离 iP。**这一反应顺序与植物中先由独立 IPT 形成多磷酸核苷酸、再由独立 LOG 激活 CK 的组织方式不同。

图 3D 进一步解析 FexB 的底物和产物范围。以 iP **1** 为底物时，FexB 生成 tZ **2**、cZ **3** 和 pyrrole-CK **6**；以 tZ 或 cZ 为底物时均可继续生成化合物 **6**。这说明 tZ 和 cZ 既是终端 canonical CK，也可作为更深度氧化的中间体。FexB 因而不是单一位点羟化酶，而是可根据反应次数推进侧链羟基化和环化的多功能 CYP450。

将底物替换为 N6-methyl-iP **9** 后，图 3D 显示 FexB 仍能生成 N6-methyl-tZ **10**，说明 N6 甲基化并未阻断初始 C4′ 羟基化；但体系未形成相应吡咯环产物。**该结果将 FexB 的两类功能分开：侧链氧化不依赖游离 N6-H，而 C—N 环化必须保留该氢原子。**

由于游离醛中间体难以直接积累，作者在图 3E 中采用 2NP 进行捕获。FexB 与 tZ **2** 反应后出现 m/z 353 [M+H]+ 的 Z/E-**8**；与 N6-methyl-iP **9** 反应后出现 m/z 367 [M+H]+ 的 Z/E-**12**。两个体系均能形成 C4′-aldehyde，但只有保留 N6-H 的化合物能够继续闭环，由此排除了“甲基化阻断醛生成”的替代解释。

图 3F 据此提出连续催化模型。FexB 首先将 iP **1** 的 C4′ 甲基羟基化生成 tZ **2**，随后再次氧化形成 geminal diol，并经失水产生醛 **7**。FexB 活性位点进一步促进 N6-H 去质子化，使氮原子对醛侧链发生分子内亲核加成，形成新的 C4′—N6 键，最终脱水得到 pyrrole-CK **6**。**这一过程将 CYP450 连续氧化与非典型分子内 C—N 键构建耦联起来。**

### (三) FexC 的分支氧化与 FexD 介导的双键异构化

#### 实验目的与设计逻辑

FexC 与植物 CKX 同属 FAD 依赖型 BBE-like oxidase 超家族，序列上含有共价结合 FAD 所需的保守基序，因此最直接的假设是其负责 CK 侧链裂解。然而，fex 基因簇还编码缺少经典 YXXXK 催化基序的 NmrA-like SDR FexD。作者通过单酶、双酶、底物替换、基序突变和同位素示踪，考察 FexC 是否存在超出 CKX 的功能，以及 FexD 如何接入该氧化过程。

#### 实验结果与机理解析

图 4A 显示，将 iP **1** 与 FexC 粗酶孵育后，底物下降并生成 adenine **13**，同时出现少量 C1′-keto-iP **16**。加入 FexD、FAD 和 NADPH 后，体系新增 Δ1-iP **17**；而将 FexD 的 NADPH 结合基序 G11XXG14XXG17 中 Gly14 突变为 Ala 后，化合物 **17** 消失，裂解产物 **13** 和氧化产物 **16** 仍可形成。

这一结果表明，FexD-G14A 并未抑制 FexC 的氧化功能，而是选择性破坏了还原步骤。**FexC 可以独立完成底物脱氢、裂解和 C1′ 氧化；FexD 仅在 NADPH 结合能力完整时，截获 FexC 形成的氧化中间体并将其转化为双键异构化产物。**

图 4B 通过 2NP 捕获验证另一裂解片段。FexC 与 iP **1** 反应后，检测到 m/z 220 [M+H]+ 的 Z/E-**15**，其保留时间与 3-methyl-2-butenal **14** 的标准衍生物一致。adenine **13** 与醛 **14** 同时形成，说明 FexC 对 C1′—N6 键的作用并非单纯水解，而是先氧化为亚胺，再由水进攻导致碳氮键断裂。

图 4C 给出了化合物 **15–19** 的结构。C1′-keto-iP **16** 保留完整 N6 侧链，但将与嘌呤相邻的 C1′ 转化为羰基；Δ1-iP **17** 则将原有 Δ2 双键迁移至 C1′—C2′；化合物 **18** 是 Δ1-iP 的 C8 羟基化形式；Δ1-tZ **19** 同时具有双键迁移和末端羟基。结构对比显示，FexC-FexD 不是改变氧化态的简单还原，而是实现净双键位置异构化。

图 4D 对底物识别条件进行了限制。FexC 对 iPR **4**、pyrrole-CK **6** 和 N6-methyl-iP **9** 均无可检测转化，FexC-FexD 组合也不能将化合物 **9** 异构化。iPR 的核糖基造成空间或结合模式障碍，而化合物 **6** 已失去可供初始脱氢的侧链构型；N6 甲基化则直接阻断亚胺形成。**游离 N6-H 是 FexC 启动 C1′ 脱氢的必要结构条件。**

在图 4E 中，FexC 可分别消耗 tZ **2** 和 cZ **3**，并形成 adenine **13**，证明其 CKX-like 裂解功能不局限于 iP。FexC 与 FexD 联合处理 tZ 时则产生 Δ1-tZ **19**，说明侧链末端羟基不会妨碍双键迁移。论文补充实验进一步表明 cZ 也可进入该反应并形成化合物 **19**，提示顺反式差异可能在亚胺形成后被同一异构化终点覆盖。

图 4F 将体外结论带回异源重构体系。AN-fexACD 中产生 Δ1-iP **17** 和其底盘过度氧化产物 **18**，而同时表达 fexA、fexB、fexC 和 fexD 的 AN-fexABCD 进一步产生 Δ1-tZ **19**。由此可见，FexC-FexD 的反应并非体外高浓度条件下的偶然耦联，而能在真菌细胞内与 FexA、FexB 上游模块连续衔接。

图 4G 的 H218O 实验显示，C1′-keto-iP **16** 的分子离子相对普通水反应产物增加 2 Da，说明一个 18O 原子被引入羰基。**化合物 16 的 C1′ 氧并非直接来自分子氧或 FAD，而来自水分子。**相反，在 D2O 中生成的化合物 **17** 未出现氘掺入，支持关键还原当量主要由 NADPH 以氢负离子形式提供，而不是通过溶剂质子直接加成建立新键。

图 4H 将这些证据整合为分支反应模型。FexC 首先利用 FAD 从 iP **1** 的 C1′—N6 邻位体系移除氢，形成亚胺中间体。该中间体可被水进攻并裂解为 adenine **13** 和醛 **14**；也可在水加成后再次脱氢，保留 C—N 键并形成 C1′-keto-iP **16**；若 FexD 在亚胺水解前结合，则利用 NADPH 实施 1,4-还原，生成 Δ1-iP **17**。

**FexC 的初始脱氢是 3 条反应分支的共同入口，而中间体后续被水、FexC 再氧化或 FexD 还原所截获，决定最终发生降解、C1′ 氧化还是双键异构化。**FexD 甚至能够与玉米 ZmCKX 耦联生成化合物 **17**，说明其识别对象更可能是通用亚胺中间体，而不是 FexC 特有的蛋白相互作用界面。

### (四) 侧链微小改造重新定义 CK 的活性与失活状态

#### 实验目的与设计逻辑

化合物是否属于“失活产物”不能仅凭结构变化判断。作者因此将 fex 途径产物分为 iP-type 和 tZ-type 两组，分别以经典 iP **1** 和 tZ **2** 为参照，通过细胞增殖、叶片衰老、色素合成和根伸长 4 类独立指标评价其植物激素活性。图 5A–D 提供不同生理层面的证据，图 5E 则将活性结果重新映射至生物合成途径。

#### 实验结果与机理解析

图 5A 的烟草愈伤组织实验首先检验 CK 促进细胞分裂的能力。iP **1** 处理产生 **0.37 ± 0.03 g** 愈伤组织，而 Δ1-iP **17** 仅产生 **0.16 ± 0.01 g**，约为 iP 的 43%。C1′-keto-iP **16** 则产生 **1.04 ± 0.07 g**，约为 iP 的 2.8 倍。tZ **2** 产生 **0.44 ± 0.04 g**，而化合物 **19** 和 pyrrole-CK **6** 分别仅为 **0.20 ± 0.02 g** 和 **0.21 ± 0.02 g**。

**C1′ 氧化并未削弱 iP 活性，反而大幅增强细胞增殖作用；双键由 Δ2 迁移至 Δ1 或形成 N6-pyrrole 环，则明显降低愈伤组织诱导能力。**这说明受体识别对侧链几何形态高度敏感，而邻近嘌呤的羰基可能提高了受体结合、化合物稳定性或细胞内有效暴露。

图 5B 以离体小麦叶片暗处理后的叶绿素残留量评价延缓衰老作用。iP **1** 与 Δ1-iP **17** 分别保留 **18.88% ± 2.04%** 和 **17.18% ± 1.53%** 叶绿素，两者差异有限；C1′-keto-iP **16** 则达到 **39.45% ± 1.77%**，超过 iP 的 2 倍。tZ **2** 保留 **64.85% ± 4.03%**，而化合物 **19** 和 **6** 分别仅保留 **24.19% ± 3.08%** 与 **18.79% ± 1.71%**。

该实验说明不同生物测定对结构变化的敏感程度并不完全相同。Δ1-iP **17** 在叶绿素保留实验中与 iP 接近，但在愈伤组织实验中活性明显下降；C1′-keto-iP **16** 则在两个体系中均表现出增强作用。**化合物 16 的高活性具有跨生理指标一致性，而化合物 17 更接近弱效或组织依赖型 CK，而非绝对无活性分子。**

图 5C 检测 CK 对苋属子叶甜菜红素合成的促进作用。iP **1** 和 C1′-keto-iP **16** 均使甜菜红素水平达到溶剂对照的近 7 倍，二者总体接近；Δ1-iP **17** 约保留 iP 作用的 69%。在 tZ-type 组中，pyrrole-CK **6** 和 Δ1-tZ **19** 分别仅达到 tZ **2** 活性的约 37% 和 72%，进一步证明吡咯环化造成的活性损失强于末端带羟基的双键异构化。

图 5D 采用拟南芥主根伸长抑制作为另一方向的活性指标。溶剂对照根长接近 6 cm，iP **1** 将根长降至 **2.07 ± 0.07 cm**，C1′-keto-iP **16** 进一步降至 **1.85 ± 0.07 cm**；Δ1-iP **17** 处理后根长为 **4.45 ± 0.15 cm**。tZ **2** 使根长降至 **1.37 ± 0.04 cm**，而化合物 **19** 和 **6** 处理后的根长分别为 **2.78 ± 0.10 cm** 与 **5.22 ± 0.15 cm**。

由于 CK 活性越强，主根通常越短，图 5D 与愈伤组织实验呈方向相反但生物学含义一致的结果。**化合物 16 对根伸长的抑制强于 iP，化合物 6 几乎丧失 tZ 型根抑制作用，化合物 17 和 19 则表现为不同程度的活性衰减。**多种生物测定的一致趋势排除了某一单一组织实验造成的偶然判断。

图 5E 将结构、酶和活性标志合并为最终途径。FexA 生成活性 iP，FexB 羟基化形成活性 tZ 和少量 cZ；FexC 可通过氧化裂解生成 adenine **13**，也可将 iP 转为活性增强的 C1′-keto-iP **16**；FexC-FexD 将 iP 和 tZ 转为低活性的化合物 **17** 和 **19**；FexB 连续氧化及 C—N 成键形成低活性的 pyrrole-CK **6**。

**图 5E 所揭示的并非一条单向“合成后降解”途径，而是以 iP 和 tZ 为分支节点、同时生成高活性和低活性衍生物的化学调控网络。**其中，FexC 兼具降解酶、活性增强修饰酶和双酶级联起始酶 3 种角色，是该网络控制产物命运的中心节点。

### (五) 真菌 CK 衍生物通过植物双组分系统调节转录响应

#### 实验目的与设计逻辑

生理表型能够反映 CK 效应，却不能直接证明真菌衍生物是否通过经典 CK 信号通路发挥作用。作者因此检测拟南芥双组分信号系统中的受体、磷酸转移蛋白及早期响应调节因子。Figure 6 未设置字母子图，但由左侧信号模型和右侧 AHK4、AHP2、ARR5、ARR15 四组表达数据组成，各部分共同用于定位不同结构修饰影响的信号层级。

#### 实验结果与机理解析

图 6 左侧模式图概括了 CK 双组分信号转导。CK 与含组氨酸激酶结构域的 AHK 受体结合后，引发受体内部组氨酸和天冬氨酸残基间的磷酸传递；磷酸基团随后转移至 AHP 蛋白，并进入细胞核激活 ARR 类响应调节因子。该模型将化合物处理与后续 qPCR 读数连接起来：AHK4 反映受体相关响应，AHP2 反映磷酸中继，而 ARR5 和 ARR15 是较直接的早期转录输出。

AHK4 表达数据表明，iP **1** 和 tZ **2** 均提高转录水平；C1′-keto-iP **16** 仍能维持受体相关响应。与相应母体化合物相比，Δ1-iP **17** 和 Δ1-tZ **19** 的诱导能力下降，pyrrole-CK **6** 接近低响应水平。图中 tZ 与化合物 **19** 的比较为 P = 0.07，而 tZ 与化合物 **6** 的差异达到 P < 0.0001，说明 pyrrole 环化对受体相关通路的抑制更为明确。

AHP2 数据整体变化幅度小于其他基因。iP **1**、化合物 **17** 和 **16** 均接近基础水平，表明 iP-type 衍生物对该磷酸转移蛋白转录量的影响有限；tZ **2** 可提高 AHP2 表达，而化合物 **19** 和 **6** 均降低该响应。**Fungal CK 的主要差异并非必然发生在每一个磷酸中继组分，而更集中体现于受体响应和下游 ARR 转录输出。**

ARR5 表达对结构变化表现出更高分辨率。iP **1** 产生约 5 倍于对照的响应，C1′-keto-iP **16** 提高至约 7 倍，而 Δ1-iP **17** 接近基础水平。tZ **2** 诱导约 10 倍表达，化合物 **19** 仅保留约一半响应，pyrrole-CK **6** 几乎完全失去诱导作用。**ARR5 结果与愈伤组织及根伸长表型高度一致，支持化合物 16 为强效激动型 CK，而化合物 6 和 17 为弱响应形式。**

ARR15 的变化趋势与 ARR5 相似，但动态范围更大。iP **1** 诱导约 6 倍表达，C1′-keto-iP **16** 达到约 11 倍；tZ **2** 约为 14 倍，化合物 **19** 降至约 6 倍，而 Δ1-iP **17** 和 pyrrole-CK **6** 接近基础水平。ARR5 和 ARR15 的一致性说明，侧链修饰影响的是 CK 信号通路整体输出，而非仅对单一报告基因产生非特异性作用。

**Figure 6 证明 fex 途径产物仍主要通过植物经典双组分信号系统发挥作用，但不同侧链构型改变了信号幅度。C1′ 氧化增强下游 ARR 响应，双键异构化降低信号输出，pyrrole 环化则使受体至下游响应基本失活。不过，qPCR 仅反映转录响应，尚不能区分这些变化源于受体结合亲和力、化合物稳定性、跨膜运输还是细胞内代谢速率。

# 六、总体结论

本研究从 IPT-LOG 融合蛋白的基因组分布出发，完整解析了 _Fusarium_ fex 基因簇的 CK 化学。FexA 将 AMP 异戊烯基化并切除磷酸核糖基，一步生成 iP；FexB 将 iP 羟基化为 tZ 和 cZ，并通过连续 C4′ 氧化及分子内 C—N 成键形成 pyrrole-CK；FexC 以共同的亚胺中间体为起点，分别介导 CK 氧化裂解和 C1′-keto-iP 形成；FexD 则利用 NADPH 对该中间体实施 1,4-还原，完成双键位置异构化。

**该研究确立了 _Fusarium_ 区别于植物的两类 CK 失活机制：FexB 催化的 pyrrole 环形成，以及 FexC-FexD 协同催化的 Δ2→Δ1 双键异构化。与此同时，FexC 催化的 C1′ 氧化生成活性高于 iP 的 C1′-keto-iP，表明同一氧化酶可以在降解、失活和活性增强之间分配底物命运。**

从结构—活性关系看，N6 侧链并非只需保留疏水长度即可维持 CK 功能。双键位置、末端羟基、N6-H 和 C1′ 氧化状态共同决定植物受体通路输出。该工作因此将真菌病原体定位为非经典 CK 骨架和酶催化反应的来源，为后续发现植物生长调节剂及解析真菌—植物互作提供了化学基础。

# 七、论文评价

### 优点与创新

该研究的主要优势在于论证链完整。SSN 与系统发育分析不是作为附加性的生物信息学描述，而是直接确定了 IPT-LOG 融合蛋白及 fex 基因簇的研究对象；异源重构负责建立基因与代谢物的对应关系；体外酶促、化学捕获、同位素示踪和关键基序突变进一步解析瞬态中间体与反应机制；植物生理测定和 qPCR 则验证这些化学变化具有明确的功能后果。

在酶学层面，FexB 将常规 CYP450 羟基化扩展至连续氧化和分子内 C—N 键构建；FexC 从 CKX-like 降解酶扩展为能够生成 C1′-keto-iP 的分支氧化酶；FexD 则表现出罕见的 NADPH 依赖型 1,4-还原功能。特别是 FexD 能与植物 ZmCKX 耦联，说明该反应可能具有跨蛋白来源的可移植性，也为设计“氧化酶—还原酶”人工级联提供了依据。

论文对“失活”的判断并未停留在单一活性实验，而是同时考察细胞分裂、衰老、色素合成、根伸长和信号基因表达。多指标结果总体一致，使 pyrrole-CK **6**、Δ1-iP **17** 和 Δ1-tZ **19** 的低活性结论较为可靠，也使 C1′-keto-iP **16** 的增强活性具有较强的功能支撑。

### 未来研究方向

研究尚未在 _F. expansum_ 原生宿主中通过 fexA、fexB、fexC 或 fexD 敲除验证各产物的体内来源，也未测定这些化合物在感染过程中的实际浓度和时空变化。因此，后续应在原生菌株中进行基因删除、回补和靶向代谢组分析，并结合植物感染表型，确定 fex 途径是否直接影响毒力、宿主营养重分配或病斑形成。

FexB、FexC 和 FexD 的催化机制目前主要依据产物、中间体捕获及同位素实验推导，缺少酶—底物复合物结构。尤其是 FexC 如何在水解、二次氧化和 FexD 截获之间控制分支比例，FexB 如何定位 N6-H 并促进 C—N 成键，均需要晶体学、冷冻电镜或可靠的结构预测结合定点突变进一步解析。

植物活性实验显示化合物 **16** 活性增强，但尚不能确定其原因。后续可开展 AHK 受体结合实验、受体突变体分析、化合物摄取和代谢稳定性测定，以区分“受体亲和力提高”与“细胞内半衰期延长”两种机制。对于化合物 **17** 和 **19**，也应避免将低活性简单等同于完全失活，因为其在部分测定中仍保留一定效应。

此外，fexE 在异源重构中未显示明确功能。该基因在 clade I 基因簇中的保守存在提示其可能需要特定底物、诱导条件或蛋白伙伴。扩大底物谱、改变培养环境、分析感染阶段转录水平及测试更多途径中间体，可能有助于揭示 FexE 所对应的隐藏分支。

# 八、关键问题及回答

**Q1**：FexA 作为 IPT-LOG 融合酶，其生物化学优势是什么？为什么不能简单视为两个独立酶的物理拼接？

**A**：FexA 的 IPT 结构域以 AMP 为真实受体，先形成 iPRMP，LOG 结构域随后切断 C1″—N9 键生成游离 iP。两个步骤在同一多肽内连续发生，能够减少 iPRMP 向非酶促脱磷酸产物 iPR **4** 的流失，并使真菌直接输出可被 FexB 和 FexC 识别的游离 CK。植物 CYP735A 通常识别核苷酸型底物，而 FexB 只识别游离 iP，因此 IPT-LOG 融合不仅提高反应衔接效率，也与下游酶的底物偏好共同构成真菌特有的途径组织逻辑。

**Q2**：FexC 为什么能够同时催化 CK 裂解、C1′ 氧化及双键异构化，这 3 类反应是否由不同活性中心完成？

**A**：现有证据更支持三者共享同一个初始脱氢步骤，而非存在 3 个彼此独立的活性中心。FexC 首先利用 FAD 将 iP 的 C1′—N6 邻位体系氧化为亚胺。亚胺被水进攻并发生 C—N 断裂时形成 adenine 和醛；水加成后若再次被 FexC 氧化，则形成 C1′-keto-iP；若亚胺在水解前被 FexD 截获，FexD 利用 NADPH 进行 1,4-还原，则形成 Δ1 双键。产物分支主要取决于瞬态亚胺的竞争性去向，FexD 的作用相当于改变中间体的动力学分配。

**Q3**：论文将化合物 6、17 和 19 定义为 CK 失活形式，这一结论是否充分？

**A**：对于植物响应而言，该结论具有多项独立证据。3 种化合物在愈伤组织诱导、叶绿素保留、甜菜红素合成、根伸长抑制以及 ARR 基因表达中均弱于对应的 iP 或 tZ，尤其 pyrrole-CK **6** 在多数指标中接近低响应状态。不过，“失活”应理解为相对于经典植物 CK 的活性大幅降低，而不是在所有生物环境中完全没有功能。化合物 **17** 和 **19** 仍保留部分植物效应，其在真菌细胞中的作用也尚未测定。因此，更严谨的表述是二者属于 _Fusarium_ 产生的低活性或失活倾向型 CK，而其对真菌自身生理和病原互作的意义仍需原生宿主实验确定。
