
**第二章：酶如何催化反应**

\1. **青霉素的作用机制及耐药性**

1.1 **细菌细胞壁的成分**

细菌细胞壁的主要成分是肽聚糖，一般是由 N-乙酰葡糖胺与 N-乙酰胞壁酸重复交替链接构成糖骨架，短肽包含五个氨基酸，末端为两个 D-Ala，短肽连接在 N-乙酰胞壁酸上，短肽之间通过酰胺键交叉相连，这些酰胺键由 DD transpeptidase 催化生成。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.001.png)

1.2  **DD-transpeptidase 作用机制**

DD-transpeptidase 的 195 位的丝氨酸为活性催化残基，195 位丝氨酸的羟基失去一个氢之后带负电，进攻肽链末端两个 Ala 之间的酰胺键使其断裂，导致最末端的 D-Ala 游离出来，肽链上剩余的 D-Ala 则通过共价键与 DD-transpeptidase 相连。

此时，另一条糖链所连短肽中氨基酸的 NH2 失去一个氢带负电，进攻与 DD-transpeptidase 相连 D-Ala 的羰基使酯键断裂，导致 DD-transpeptidase 与糖肽链分离，最终，D-Ala 与另一条肽链末端的氨基酸通过酰胺键连接在一起。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.002.png)

1.3  **青霉素的作用机制**

青霉素的结构与细胞壁肽链末端的两个 D-Ala 较为相似，所以青霉素能够被 DD-transpeptidase 识别并结合生成 DD-transpeptidase-青霉素复合物，阻止 DD-transpeptidase 催化细胞壁中肽链的交联，阻碍细菌细胞壁的生成，从而丧失屏障作用。此时细胞外的水分会进入细胞内，使细菌的细胞肿胀、变形、破裂而死亡。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.003.png)

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.004.png)

1.4  **青霉素的耐药机制**

*β*-内酰胺酶能够水解青霉素的 *β*-内酰胺环，使 DD-transpeptidase 不能够与开环后的青霉素相结合，从而使 *β*-内酰胺类抗生素失活。

\2. **酶结构与功能（胰凝乳蛋白酶 Chymotrypsin）**

2.1 **胰凝乳蛋白酶结构**

胰凝乳蛋白酶 A 由三条多肽链组成，它们通过半胱氨酸残基之间的桥或二硫键彼此共价连接。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.006.png)

2.2 **胰凝乳蛋白酶功能**

胰凝乳蛋白酶负责水解肽键，主要攻击具有芳族侧基的氨基酸（例如酪氨酸，色氨酸和苯丙氨酸）的羧基部分。

2.3 **氧离子洞穴**

氧离子洞穴位于蛋白酶的活性中心，用于稳定带负电荷的去质子氧的过渡态的结构。

胰凝乳蛋白酶催化过程中，Ser-195 与 Gly-193 的-N-H-与底物形成的氧负离子以氢键结合形成的结构称为氧离子洞穴，因为这个结构结合过渡态比直接结合底物更稳定，所以降低了反应的活化能，从而实现了催化的作用。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.007.png)

2.4 **蛋白质结合口袋**

蛋白质结合口袋是指蛋白质表面或内部具有适合与配体结合的空腔，口袋周围的氨基酸残基决定了它的形状，位置，物化特性以及功能。

胰凝乳蛋白酶具有疏水性大口袋，结合部位含有 Ser 残基，可与底物侧链芳香基团氨基酸残基结合。

2.5 **催化三联体**

催化三联体通常指在水解酶和转移酶的活性位点中心同时作用的三个氨基酸残基（如蛋白酶、酰胺酶、酯酶、酰基转移酶、脂肪酶和 *β*-内酰胺酶）。用于共价催化的亲核残基一般是酸-碱-亲核三联体。残基会形成一个电荷中继网络，以极化和活化亲核试剂，来攻击底物形成共价中间体，然后中间体水解，再生出游离的酶。亲核试剂大多是丝氨酸或半胱氨酸，也有少量是苏氨酸。 

胰凝乳蛋白酶中的催化三联体由 His57 、Asp102、Ser195 等三个残基组成。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.008.png)

催化三联体的残基可能在其所在的氨基酸序列（一级结构）中离得很远，但最后它们将会折叠到一起。

2.6 **胰凝乳蛋白酶催化机制**

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.009.png)

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.010.png)

（1）底物与胰凝乳蛋白酶结合并结合形成典型的酶-底物复合物，形状是一个含有大量疏水残基的大环。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.011.png)

（2）Asp102 与 His57 以氢键（可能是低能障氢键）结合，将其咪唑氮的 pKa 从 7 增加至约 12。这使得 His57 变为一种强的广义碱，可将 Ser195 去质子化。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.007.png)

（3）Ser195 的 OH 上的 H 被 His57 的 N 夺走生成氧负离子，随后氧负离子进攻底物肽键的羰基， 并迫使羰基氧接受电子形成四面体氧负离子中间体。底物肽的氧负离子在氧离子洞穴内与 Gly193 和 Ser195 形成氢键，从而使得底物稳定。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.012.png)

（4）负电荷的不稳定性使得羰基氧负离子通过重排重新形成 C=O，同时底物肽键上的 N 夺取 His57 的 H 形成 -NH2，导致酰胺键断裂，释放第一个产物。此时，底物肽键处的羧基部分已转移至 Ser195 的羟基生成酯键。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.013.png)

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.014.png)

（5）水分子进入活性位点后，His57 的 N 夺取水分子的氢导致生成具有强亲核活性的氢氧根负离子。随后氢氧根负离子进攻与 Ser195 相连的羰基，再次生成四面体氧负离子中间体。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.015.png)

（6）羰基氧负离子再一次发生重排重新形成 C=O，同时 Ser195 的酯键断裂，形成第二个产物，此时产物仍与酶通过氢键连接。

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.016.png)

![...](..\..\..\1.%20天然产物生物合成\%201.%20课程笔记\%20天然产物生物合成概论\images\第二章：酶如何催化反应.017.png)

（7）第二个产物从活性位点中释放出来，酶再生，反应结束。

2.7 **催化三联体中 Asp102 的功能** 

1. 与 His57 之间形成氢键 
2. 使 His57 的 pKa 从 7 增加至约 12。这使得 His57 变为一种强的广义碱，可将 Ser195 去质子化。
3. 脱去 Ser195 的质子，使 Ser195 变为强的亲核试剂。
