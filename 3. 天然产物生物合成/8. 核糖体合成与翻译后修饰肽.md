
**8. 核糖体合成与翻译后修饰肽**

\1. **RiPPs——一种有成药前景的天然产物**

`      `核糖体合成和翻译后修饰肽（ribosomally synthesized and posttranslationally modified peptides，RiPPs）是一类由核糖体合成并经翻译后修饰而产生的多肽类天然产物。其生物合成过程中丰富的翻译后修饰（post-translational modifications，PTMs）赋予了RiPPs 独特的化学和拓扑结构，使其具有多种多样的生物活性，如抗菌、抗病毒、抗病毒等。然而，由于RiPPs 生物合成逻辑简单且生物合成基因簇相对较小，RiPP 前体肽通常很短且高度可变，它们的编码序列在公共数据库中通常没有注释，并且通常无法被序列同源性搜索工具识别（如BLAST）。因此在很长一段时间内 RiPPs 是被研究人员忽略的。进入21世纪以来，随着基因组测序数据的快速增长以及基因组学、生物信息学进步促进了各种天然产物的发现。ORF 描述与在线评估工具（Rapid ORF Description and Evaluation Online，RODEO），RiPPER 以及 RRE-Finder 等专门用于 RiPPs 产物挖掘的生物信息学工具的使用促进了 RiPP BGCs 的指数级增长。大量的基因组数据分析表明 RiPP 类的生物合成基因簇广泛存在于植物、真菌以及微生物的基因组中，具有极大地挖掘潜力。目前已有越来越多的新型活性 RiPP 不断地被发现和报道，人们对 RiPP 类产物的成药前景十分乐观。

` `尽管 RiPPs 的结构丰富多样，但它们都遵循着相似的生物合成逻辑。其 BGCs 一般至少包括一个或多个短的编码前体肽的结构基因（通常称为*“xxxA”*）以及一些编码翻译后修饰酶的基因。此外，在 RiPPs 的 BGCs 中通常还会编码一些诸如参与成熟、运输、自身抗性或调控相关的辅助基因。在生物合成过程中，前体肽基因 *xxxA* 首先通过转录和核糖体合成未修饰的前体肽 A（长度约为 20～110 个氨基酸残基）。前体肽通常是由 N 端前导肽（leader peptide，LP）和 C 端核心肽（core peptide，CP）组成。在某些情况下，C 端会存在跟随肽（follower peptide，FP）。对于一些真核生物多肽，前体的 N 端还存在信号序列或识别序列。前导肽对翻译后修饰酶的识别结合、终产物的输出具有重要作用，也会防止核心肽在翻译后修饰之前被蛋白水解酶裂解；核心肽则是翻译后修饰酶直接作用区域，经后修饰后形成最终产物。一些 RiPP，如 cyanobactins，dikaritins，orbitides，amatoxins，cyclotides 和 lycinums 的前体肽携带多个核心肽和多个识别序列，因此一个前体肽可以产生多个 RiPP 产物。后修饰酶在前体肽 A 的 CP 区域引入不同的 PTMs 后，称为 *mXxxA*。许多 RiPPs 还会被一些额外后修饰酶进一步修饰，从而在产物中引入“剪裁修饰”（tailoring modifications），从而对产物的成熟、活性或稳定性起到重要作用。常见的“剪裁修饰”包括脱水、羟基化、杂环化、大环化、甲基化、糖基化、酰化、卤化、异戊烯化和异构化等在内的多种化学修饰[109]。修饰完成后，前导肽随后被一系列转运体、肽酶或两者的复合体移除，并释放经修饰的核心肽。这些核心肽可以作为具有生物活性的成熟的 RiPP 类天然产物释放到细胞外以发挥生物作用。

```<p>RiPPs的全称是“核糖体合成与翻译后修饰肽”，这个名字有点拗口，但拆开来看就很容易理解了。它的合成过程主要分为三步：</p><p>**第一步：生产基础零件——前体肽的合成**</p><p>- **地点**：细胞内的核糖体（可以想象成一个3D打印机）。</p><p>- **过程**：细胞会根据基因（也就是设计图纸）上的信息，通过核糖体这个“打印机”，制造出一条由氨基酸串起来的“肽链”。这就像是乐高工厂先生产出最基础的长条形积木。</p><p>- **这个基础零件的特点**：这条肽链被称为“前体肽”，它由两部分组成：</p><p>- **核心肽**：这是我们最终想要的产品的“毛坯”部分。</p><p>- **前导肽**：这部分像是一个“工牌”或者“说明书”，它不构成最终产品，但作用非常关键，负责引导后续的“加工”步骤，并确保“毛坯”能被正确的“机床”识别。</p><p>**第二步：精加工——翻译后修饰**</p><p>- **地点**：细胞质内。</p><p>- **过程**：前体肽被制造出来后，并不能直接使用，它需要经过一系列的“精加工”，这个过程就叫做“翻译后修饰”。细胞内有各种各样的“修饰酶”（可以想象成各种功能的自动化加工机床），它们会识别前导肽这个“工牌”。</p><p>- **具体的加工内容**：这些“机床”会对核心肽这个“毛坯”进行各种化学改造，比如：</p><p>- 脱水和环化（像把直的积木掰弯并首尾相连）。</p><p>- 添加一些特殊的化学基团（像给积木上色或者增加一些特殊接口）。</p><p>经过这一步，核心肽部分的结构变得非常独特和复杂，但这还不是最终的产品。</p><p>**第三步：剪裁和出厂——肽链的切割与释放**</p><p>- **过程**：当所有的“精加工”都完成后，就需要把没用的部分“剪掉”。 这时会有特定的酶（就像一把剪刀）过来，识别并切掉前导肽这个“工牌”，以及可能存在的其他多余部分。</p><p>- **最终产品**：切掉前导肽后，剩下的经过复杂修饰的核心肽，就是最终成熟的、具有生物活性的RiPPs分子了。 这时，它就可以被运出细胞，去执行各种各样的生物学功能了，比如抗菌等。</p><p>**总结一下，RiPPs的合成过程就像是：**</p><p>1. **设计与打印**：根据基因图纸，在核糖体上打印出带有“工牌”（前导肽）的“毛坯零件”（核心肽）。</p><p>2. **精细加工**：在各种酶的作用下，对“毛坯零件”进行一系列复杂的化学改造。</p><p>3. **剪裁出厂**：剪掉“工牌”，释放出最终成型的、功能强大的产品。</p><p>这个过程高效且精准，使得生物体能够合成出结构复杂多样、功能强大的肽类物质。</p>```
