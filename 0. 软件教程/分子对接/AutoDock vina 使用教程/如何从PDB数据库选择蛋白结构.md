
**如何从PDB数据库选择蛋白结构**

某些蛋白在 PDB 数据库里面有好多结构，我们要从数据库里下载合适的结构。

\1. **PDB 数据库**

1. 点击链接，进入 PDB 官网：<https://www.rcsb.org/>

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.001.png)

2. 在搜索框中输入需要的**蛋白名字和对应基因 ID**，比如输入“Tryptophan-5-halogenase”，可以看到会出来很多结果。每个结果都会给出蛋白结构的解析时间，解析方法，分辨率，物种来源，蛋白名称等等。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.002.png)

3. **筛选原则**    
- **解析方法**：晶体 X 射线衍射 > NMR 解析
- **分辨率**：数值越大，蛋白结构质量越差，因此要选择分辨率小的结构。
- **物种来源**：如果没有明确的目标蛋白，尽可能选择与自己蛋白来源相近的物种
- **其他**：是否有小分子共晶结构，优先选择含有小分子共晶的蛋白结构，便于在分子对接时确定对接盒子的大小。

\2. **UniProt**

1. 点击链接，进入 UniProt 官网：<https://www.uniprot.org/>

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.003.png)

2. 在搜索框中输入需要的**蛋白名字和对应基因 ID**，比如输入“Tryptophan-5-halogenase”，可以看到会出来很多结果。每个结果都会给出蛋白结构对应的基因名称、物种来源，氨基酸序列长度，结构中是否给出活性位点，3D 结构的数量，与结构相关的文章等信息。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.004.png)

3. 此处选择第一个蛋白，该蛋白中给出了结合位点，并且包含 3 个 3D 结构。进入之后，可以看到关于这个蛋白的各种详细信息。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.004.png)

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.005.png)

结合位点

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.004.png)

3D 结构

4. 来到 3D 结构部分，可以看到有 3 个不同的 3D 结构，我们选择分辨率最低的结果（2WEU），在新的标签页中打开 RCSB-PDB，这样就直接跳转到 RCSB-PDB 数据库中该蛋白的详情页。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.004.png)

5. 2WEU 在RCSB-PDB 中的详细信息，图中的 **Full report** 是一个PDF 格式的质量报告，包含了这个结构质量的一些信息，包含序列信息、二级结构、键长键角等详细信息，有助于评估结构质量。

![...](..\..\..\0.%20软件教程\分子对接\AutoDock%20vina%20使用教程\images\如何从PDB数据库选择蛋白结构.006.png)

6. 确定目标蛋白结构之后，下载 PDB 格式文件即可
