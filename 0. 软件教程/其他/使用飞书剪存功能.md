
**使用飞书剪存功能**

```原文链接： [https://www.feishu.cn/hc/zh-CN/arti...](https://www.feishu.cn/hc/zh-CN/articles/606278856233-%E4%BD%BF%E7%94%A8%E9%A3%9E%E4%B9%A6%E5%89%AA%E5%AD%98%E5%8A%9F%E8%83%BD)```
**一、 功能简介** 

飞书剪存是一个可以将网页正文保存至飞书文档的浏览器扩展程序。 当你浏览到喜欢的网页时，可以使用飞书剪存，将网页内容保存至你的飞书文档中， 从此告别手动复制粘贴 **。** 同时，飞书剪存还可以智能识别并去除悬浮或嵌在网页中的广告，让你保存下来的内容无过多的干扰信息。  

**注** ：剪存效果可能受到具体剪存网页的格式、布局等影响，请以实际剪存后的内容为准。 

**二、操作流程** 

**安装飞书剪存** 

**Chrome 浏览器** 

方式 1： 你可以访问 Chrome 网上应用商店，找到 [飞书剪存 ](https://chromewebstore.google.com/detail/%E9%A3%9E%E4%B9%A6%E5%89%AA%E5%AD%98/mofcmpgnbnnlcdkfchnggdilcelpgegn?hl=zh-CN)，并将其添加至 Chrome 浏览器中。 

![...](..\..\0.%20软件教程\其他\images\使用飞书剪存功能.001.png)



方式 2：若无法访问 Chrome 网上应用商店，你也可以选择下载本地压缩包并安装。 

1. 点击 [下载飞书剪存安装包 ](https://internal-api.feishu.cn/clipapi/invoke/downloadFeishuClip)。 
2. 解压飞书剪存文件压缩包，得到 FeishuClip 文件夹。 
3. 打开 Chrome 浏览器，通过 **设置** > **更多工具** > **扩展程序** 进入扩展管理页面，并确认开发者模式处于打开状态。 
4. 点击 **加载已解压的扩展程序** ，选择 FeishuClip文件夹，或直接将该文件夹拖拽至扩展程序管理页面。 
5. 点击插件管理图标，固定剪存插件后，即成功安装至 Chrome 浏览器。 

![...](..\..\0.%20软件教程\其他\images\使用飞书剪存功能.002.png)

**使用飞书剪存** 

1. 在浏览器中打开扩展程序的管理页面，找到 **飞书剪存** ，将其固定在浏览器的工具栏。 
2. 右键点击工具栏中 的 **飞书剪存** 图标，点击 **登录** ，使用飞书账号完成验证并登录。 
3. 在想要保存的网页点击浏览器工具栏中的 **飞书剪存** 图标，当前网页正文将被一键复制并保存在处于登录状态的飞书文档中。 

![...](..\..\0.%20软件教程\其他\images\使用飞书剪存功能.011.png)



4. 剪存成功后，浏览器将自动打开该网页正文的剪存文档。若想关闭自动跳转，你可以右键点击 **飞书剪存** 图标，在 **剪存后自动打开云文档** 中选择 **关闭** 。 

![...](..\..\0.%20软件教程\其他\images\使用飞书剪存功能.012.png)



**三、常见问题** 

问：飞书剪存插件是否支持安装在 Safari 浏览器中？

答：不支持。 

问：我在桌面端的浏览器中安装了飞书剪存插件，还需要在移动端的相同浏览器中再安装一次吗？

答： 不支持在移动端的浏览器中安装飞书剪存插件。如需在移动端使用飞书剪存功能，请参考本文对于 移动端剪存 的介绍。 

问：飞书剪存功能支持仅剪存网页里选中的部分内容吗？

答：不支持。如果只需网页中的部分内容，可以在剪存后生成的飞书文档中手动编辑选择。
