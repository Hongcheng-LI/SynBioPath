
**Ubuntu 安装 Crest**

\1. **Crest 简介**

CREST（Conformer-Rotamer Ensemble Sampling Tool）是一款开源程序，主要用于高效、自动化地探索低能量分子化学空间。以下是 CREST 软件的主要功能和特点：

1. **构象和旋转异构体采样：**CREST 的核心功能是生成分子的低能量构象集合。它通过高效的采样算法，能够快速探索分子的构象空间，生成大量低能量的构象。这对于研究分子的动力学行为、构象稳定性以及与其他分子的相互作用非常重要。
2. **显式溶剂化研究：**CREST 支持显式溶剂化研究，可以模拟分子在溶剂环境中的行为。这对于研究分子在溶液中的结构和性质非常有帮助，尤其是在药物设计和材料科学中。
3. **绝对分子熵计算：**CREST 能够计算分子的绝对熵，这对于理解分子的热力学性质非常重要。通过计算分子在不同温度下的熵，可以更好地预测分子的行为。
4. **质子化和去质子化位点识别**：CREST 可以识别分子的质子化和去质子化位点，这对于研究分子的酸碱性质和电荷分布非常有帮助。这对于化学反应机理的研究和药物设计中的靶点识别非常重要。
5. **量子力学/分子力学（QM/MM）计算：**CREST 支持 QM/MM 计算，允许用户将量子力学方法与分子力学方法结合，用于研究复杂的生物分子和材料体系。这对于模拟生物大分子的反应机制和材料的电子性质非常有帮助。
6. **最小能量交叉点（MECP）计算：**CREST 还支持最小能量交叉点的计算，这对于研究分子的光化学反应和电子跃迁非常重要。
7. **高效的并行化计算：**CREST 的设计允许高效的单节点并行化，能够同时运行多个计算任务，显著提高计算效率。
8. **灵活的接口：**CREST 提供了与多种量子化学软件和力场的接口，用户可以轻松地将其与其他计算工具结合使用。
9. **用户友好的设计：**CREST 的设计要求最少的用户输入，提供了详细的文档和安装指南，使得用户可以轻松上手并进行计算。
10. **开源性：**CREST 是基于 LGPL v3 许可证的开源软件，用户可以自由使用、修改和分发。

\2. **Crest 安装**

1. 打开官网：<https://github.com/crest-lab/crest>；下载编译好的版本，此处为 **crest-gnu-12-ubuntu-latest.tar.xz**

![...](..\..\0.%20软件教程\化学软件（Linux）\images\Ubuntu%20安装%20Crest.001.png)

2. 将下载好的安装包 **crest-gnu-12-ubuntu-latest.tar.xz** 放在 **/home/lhc/** 目录下，打开终端，输入下列命令解压文件。

```C#
# 解压
tar -xf crest-gnu-12-ubuntu-latest.tar.xz```

3. 解压后的目录是 **/home/lhc/crest**，文件夹中包含三个文件，其中 **crest** 为可执行文件。

![...](..\..\0.%20软件教程\化学软件（Linux）\images\Ubuntu%20安装%20Crest.002.png)

4. 若要进行 ECD 计算，则需要将 crest 可执行文件添加到 **/home/lhc/xTB/xtb-dist/bin/** 目录中 。

![...](..\..\0.%20软件教程\化学软件（Linux）\images\Ubuntu%20安装%20Crest.003.png)

5. 打开 **/home/lhc/xTB/xtb-dist/bin/** 目录 中打开终端

```C#
# 增加可执行权限
chmod +x /home/lhc/xTB/xtb-dist/bin/crest```
